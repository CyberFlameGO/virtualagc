019254,000266:                                                                                                  #  INTEGRATED TOGETHER TO SAVE WORDS. COMPASS IS COMPLETELY RESTART
019255,000267:                                                                                                  #  PROOFED EXCEPT FOR THE FIRST 30 SECONDS OR SO. PERFORMANCE TESTS OF
019256,000268:                                                                                                  #  THE IRIGS IS RESTART PROOFED ENOUGH TO GIVE 75 PERCENT CONFIDENCE THAT
019257,000269:                                                                                                  #  IF A RESTART OCCURS THE DATA WILL STILL BE GOOD. GOOD PRACTICE TO RECYCL
019258,000270:                                                                                                  #  WHEN A RESTART OCCURS UNLESS IT HAPPENS NEAR THE END OF A TEST - THEN WAIT
019259,000271:                                                                                                  #  FOR THE DATA TO FLASH.
019260,000272: 
019261,000273:                                                                                                  #  A RESTART IN GYROCOMPASS DURING GYRO TORQUING CAUSES PULSES TO BE LOST
019262,000274:                                                                                                  #  THE PRELAUNCH ALIGNMENT TECHNIQUE IS BASICALLY THE SAME AS IN BLOCK 1

019265,000277:                                                                                                  #  OPTIMUM STATISTICAL FILTER. FOR A DESCRIPTION SEE E-1973. BOTH OF THESE
019266,000278:                                                                                                  #  ROUTINES USE STANDARD SYSTEM TEST LEADIN PROCEDURES. THE INITIALIZATION
019267,000279:                                                                                                  #  PROCEDURE FOR THE DRIFT TESTS IS IN THE JDC S. THE INITIALIZATION METHOD

019269,000281:                                                                                                  #  THE COMPASS ALIGNS TO Z DOWN, X DOWNRANGE, HAS THE CAPABILITY
019270,000282:                                                                                                  #  CHANGE AZIMUTH WHILE RUNNING, IS COMPENSATED FOR
019271,000283:                                                                                                  #  COMPONENT ERRORS, IS CAPABLE OF OPTICAL VERIFICATION (CSM ONLY).
