Source Code

These source-code files were obtained by digitally photographing an Artemis 72 (Apollo 15-17 Command Module) program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

043491,000124: 
043492,000125:                                                                                                  #  GIVEN THE OPERATING POINT NOT IN THE COAST REGION, THE DESIRED OGARATE
043493,000126:                                                                                                  #        IS AT THE POINT OF PENETRATION OF THE COAST REGION BY THE CONTROL
043494,000127:                                                                                                  #        PARABOLA WHICH PASSES THROUGH THE OPERATING POINT. FOR REGION 3
043495,000128:                                                                                                  #        DESIRED OGARATE IS SIMPLY +-MAXLIM. FOR REGIONS 1 OR 6 THE SOLUTION
043496,000129:                                                                                                  #        TO A QUADRATIC IS REQUIRED (THE PENETRATION IS ALONG THE STRAIGHT
043497,000130:                                                                                                  #        LINE OR MINLIM BOUNDRY SWITCH LINES). AN APPROXIMATION IS MADE
043498,000131:                                                                                                  #        INSTEAD. TAKE AN OPERATING POINT IN REGION 6' . PASS A TANGENT TO
043499,000132:                                                                                                  #        THE CONTROL PARABOLA THROUGH THE OPERATING POINT, AND FIND ITS
043500,000133:                                                                                                  #        INTERSECTION WITH THE STRAIGHT LINE SECTION OF THE SWITCH CURVE...
043501,000134:                                                                                                  #        THE INTERSECTION DEFINES DESIRED OGARATE. IF THE OPERATING POINT IS
043502,000135:                                                                                                  #        CLOSE TO THE SWITCH LINE, THE APPROXIMATION IS QUITE GOOD (INDEED
043503,000136:                                                                                                  #        THE APPROXIMATE AND QUADRATIC SOLUTIONS CONVERGE IN THE LIMIT AS
043504,000137:                                                                                                  #        THE SWITCH LINE IS APPROACHED). IF THE OPERATING POINT IS NOT CLOSE
043505,000138:                                                                                                  #        TO THE SWITCH LINE, THE APPROXIMATE SOLUTION GIVES VALID TREND
043506,000139:                                                                                                  #        INFORMATION (DIRECTION OF DESIRED OGARATE) AT LEAST. THE
043507,000140:                                                                                                  #        RE-EVALUATION OF DESIRED OGARATE IN SUBSEQUENT ROLL DAP PASSES (1/2
043508,000141:                                                                                                  #        SECOND INTERVALS) WILL BENEFIT FROM THE CONVERGENT NATURE OF THE
043509,000142:                                                                                                  #        APPROXIMATION.
043510,000143: 
043511,000144:                                                                                                  #  FOR LARGE OGAERROR THE TANGENT INTERSECTS +-MINLIM SWITCH BOUNDRY BEFORE
043512,000145:                                                                                                  #        INTERSECTING THE STRAIGHT LINE SWITCH. HOWEVER THE MINLIM IS
043513,000146:                                                                                                  #        IGNORED IN COMPUTING THE FIRING TIME, SO THAT THE EXTENSION (INTO
043514,000147:                                                                                                  #        THE COAST REGION) OF THE STRAIGHT LINE SWITCH IS WHAT IS FIRED TO.
043515,000148:                                                                                                  #        IF THE ROLL DAP FINDS ITSELF IN THE COAST REGION BEFORE REACHING
043516,000149:                                                                                                  #        THE DESIRED INTERSECTION (IE, IN THE REGION BETWEEN THE MINLIM
043517,000150:                                                                                                  #        AND THE STRAIGHT LINE SWITCH) IT WILL EXIBIT NORMAL COAST-REGION
043518,000151:                                                                                                  #        BEHAVIOR AND TURN OFF THE JETS. THE PURPOSE OF THIS FIRING POLICY
043519,000152:                                                                                                  #        IS TO MAINTAIN STATIC ROLL STABILITY IN THE EVENT OF A JET
043520,000153:                                                                                                  #        FAILED-ON.
043521,000154: 
043522,000155:                                                                                                  #  WHEN THE OPERATING POINT IS IN REGION 1 THE SAME APPROXIMATION IS
043523,000156:                                                                                                  #        MADE, BUT AT AN ARTIFICIALLY-CREATED OR DUMMY OPERATING POINT,
043524,000157:                                                                                                  #        DEFINED BY.. OGAERROR = INTERSECTION OF CONTROL PARABOLA  AND
043525,000158:                                                                                                  #        OGAERROR AXIS, OGARATE = +-LMCRATE WHERE SIGN IS OPPOSITE THAT OF
043526,000159:                                                                                                  #        REAL OPERATING POINT RATE.  WHEN THE OPERATING POINT HAS PASSED
043527,000160:                                                                                                  #        FROM REGION 1 TO REGION 6', THE DUMMY POINT IS NO LONGER REQUIRED,
043528,000161:                                                                                                  #        AND THE SOLUTION REVERTS TO THAT OF A REGULAR REGION 6' POINT.
043529,000162:

043532,000165:                                                                                                  #                                    2
