Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

059704,000020:                                                                                                  #  PROGRAM NAME -  SELF-CHECK                    LOG SECTION AGC BLOCK TWO SELF-CHECK
059705,000021:                                                                                                  #  MOD NO - 1                                    ASSEMBLY SUBROUTINE UTILITYM REV 25
059706,000022:                                                                                                  #  MOD BY - GAUNTT
059707,000023: 
059708,000024:                                                                                                  #  FUNCTIONAL DESCRIPTION
059709,000025: 
059710,000026:                                                                                                  #  PROGRAM HAS TWO MAIN PARTS. THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
059711,000027:                                                                                                  #  PART OF THE BACK-UP IDLE LOOP. THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
059712,000028:                                                                                                  #  STARTING VERB.
059713,000029:                                                                                                  #  THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
059714,000030:                                                                                                  #  THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK, ONE AT A TIME.
059715,000031:                                                                                                  #  IN ALL THERE ARE  7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK. MORE DETAIL DESCRIPTION MAY BE
059716,000032:                                                                                                  #  FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
059717,000033:                                                                                                  #  THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27). BELOW IS
059718,000034:                                                                                                  #  A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
059719,000035:                                                                                                  #  OCTAL, TO LOAD INTO SMODE.
059720,000036:                                                                                                  #        +-4             ERASABLE MEMORY
059721,000037:                                                                                                  #        +-5             FIXED MEMORY
059722,000038:                                                                                                  #        +-1,2,3,6,7,10  EVERYTHING IN OPTIONS 4 AND 5.
059723,000039:                                                                                                  #        -0              SAME AS +-10 UNTIL AN ERROR IS DETECTED.
059724,000040:                                                                                                  #        +0              NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
059725,000041: 
059726,000042:                                                                                                  #  WARNINGS
059727,000043: 
059728,000044:                                                                                                  #        USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
059729,000045:                                                                                                  #        SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
059730,000046: 
059731,000047:                                                                                                  #  CALLING SEQUENCE
059732,000048: 
059733,000049:                                                                                                  #        TO CALL SELF-CHECK KEY IN
059734,000050:                                                                                                  #                V 21 N 27 E     OPTION NUMBER E
059735,000051:                                                                                                  #        TO CALL SHOW-BANKSUM KEY IN
059736,000052:                                                                                                  #                V 91 E          DISPLAYS FIRST BANK
059737,000053:                                                                                                  #                V 33 E          PROCEED, DISPLAYS NEXT BANK
059738,000054: 
059739,000055: 
059740,000056:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
059741,000057: 
059742,000058:                                                                                                  #  SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED. IF SO + OPTION NUMBERS PUT
059743,000059:                                                                                                  #  COMPUTER INTO BACKUP IDLE LOOP, - OPTION NUMBERS RESTART THE OPTION.
059744,000060:                                                                                                  #  THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
059745,000061:                                                                                                  #  SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E). THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP
059746,000062: 
059747,000063: 
059748,000064:                                                                                                  #  OUTPUT

059750,000066:                                                                                                  #  SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
059751,000067:                                                                                                  #  TURNS ON THE ALARM LIGHT. THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E. FOR FURTHER
059752,000068:                                                                                                  #  INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
059753,000069:                                                                                                  #  IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
059754,000070:                                                                                                  #  INITIATED FRESH START (SLAP1).
059755,000071:                                                                                                  #  SHOW-BANKSUM STARTING WITH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
059756,000072:                                                                                                  #  THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
059757,000073: 
059758,000074: 
059759,000075:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
059760,000076: 
059761,000077:                                                                                                  #        ACCOMPLISHED BY FRESH START
059762,000078:                                                                                                  #                SMODE SET TO +0
059763,000079: 
059764,000080:                                                                                                  #  DEBRIS
059765,000081: 
059766,000082:                                                                                                  #        ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
059767,000083:                                                                                                  #        EXCEPTION IS A RESTART. RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
059768,000084:                                                                                                  #        E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
