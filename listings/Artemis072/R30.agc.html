Source Code

These source-code files were obtained by digitally photographing an Artemis 72 (Apollo 15-17 Command Module) program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

022735,000026:                                                                                                  #  MOD NO- 4             MOD  BY CROCKER                 DATE- 27 AUG 69         PROG MOD FOR TIME OPTION.
022736,000027:                                                                                                  #   NEW FUNCTIONAL DESCRIPTION-     CALLED BY VERB 82 ENTER.  PRIORITY 7
022737,000028:                                                                                                  #     USED THROUGHOUT.              CALCULATE AND DISPLAY ORBITAL PARAMETERS
022738,000029: 
022739,000030:                                                                                                  #  1. IF AVERAGE G IS OFF:
022740,000031:                                                                                                  #          FLASH DISPLAY V04N12. R2 INDICATES WHICH SHIP'S STATE VECTOR IS
022741,000032:                                                                                                  #           TO BE UPDATED. INITIAL CHOICE IS THIS SHIP (R2=1). ASTRONAUT
022742,000033:                                                                                                  #           CAN CHANGE TO OTHER SHIP BY V22EXE, WHERE X NOT EQ 1.
022743,000034:                                                                                                  #           FLASH DISPLAY TIME V06N16. INITIAL CHOICE = 0 = PRESENT TIME.
022744,000035:                                                                                                  #           ASTRONAUT CAN INSERT A TIME BY V25E, R1=HR R2=MIN R3=CSEC.
022745,000036:                                                                                                  #           CHOSEN STATE VECTOR UPDATED TO CHOSEN TIME BY THISPREC/OTHPREC
022746,000037:                                                                                                  #          CALLS SR30.1 (WHICH CALLS TFFCONMU + TFFRP/RA) TO CALCULATE
022747,000038:                                                                                                  #           RPER (PERIGEE RADIUS), RAPO (APOGEE RADIUS), HPER (PERIGEE
022748,000039:                                                                                                  #           HEIGHT ABOVE LAUNCH PAD OR LUNAR LANDING SITE), HAPO (APOGEE
022749,000040:                                                                                                  #           HEIGHT AS ABOVE), TPER (TIME TO PERIGEE), TFF (TIME TO
022750,000041:                                                                                                  #           INTERSECT 300 KFT ABOVE PAD OR 35KFT ABOVE LANDING SITE).
022751,000042:                                                                                                  #          FLASH MONITOR V16N44 (HAPO, HPER, TFF).TFF IS -59M59S IF IT WAS
022752,000043:                                                                                                  #           NOT COMPUTABLE, OTHERWISE IT INCREMENTS ONCE PER SECOND.
022753,000044:                                                                                                  #           IF P00 IS RUNNING, CALL DELRSPL SPLASH ERROR COMPUTATION.
022754,000045:                                                                                                  #           ASTRONAUT HAS OPTION TO MONITOR TPER BY KEYING IN N 32 E.
022755,000046:                                                                                                  #           DISPLAY IS IN HMS, IS NEGATIVE (AS WAS TFF), AND INCREMENTS
022756,000047:                                                                                                  #           ONCE PER SECOND ONLY IF TFF DISPLAY WAS -59M59S.
022757,000048: 
022758,000049:                                                                                                  #  2. IF AVERAGE G IS ON:
022759,000050:                                                                                                  #          CALLS SR30.1 APPROX EVERY TWO SECS.  STATE VECTOR IS ALWAYS
022760,000051:                                                                                                  #           FOR THIS VEHICLE. V82 DOES NOT DISTURB STATE VECTOR.  RESULTS
022761,000052:                                                                                                  #           OF SR30.1 ARE RAPO, RPER, HAPO, HPER, TPER, TFF.
022762,000053:                                                                                                  #          FLASH MONITOR V16N44 (HAPO, HPER, TFF).
022763,000054:                                                                                                  #           IF MODE IS P11, THEN CALL DELRSPL SO ASTRONAUT CAN MONITOR
022764,000055:                                                                                                  #           RESULTS BY N50E. SPLASH COMPUTATION DONE ONCE PER TWO SECS.
022765,000056:

022768,000059:                                                                                                  #            ABOVE REMARKS.
