Source Code

These source-code files derive from a printout of Luminary 116 (the Apollo 12 Lunar Module guidance computer program), from the personal library of original AGC developer Don Eyles, digitally photographed at archive.org, financially sponsored by Ron Burkey, and transcribed to source code by a team of volunteers. This colorized, syntax-highlighted form was created by assembling that transcribed source code. Note that the full page images are available on the Virtual AGC project page at archive.org, while reduced-size images are presented at the VirtualAGC project website. Report or fix any transcription errors at the Virtual AGC project code repository.

Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 116 OF AGC PROGRAM LUMINARY BY NASA 2021112-071
	19:09 AUG. 11,1969
Note that the date is the date of the printout, not the date of the program revision.

046935,000036:                                                                                                  #          PART OF A SECTION OF PROGRAMS,-WAITLIST, TASKOVER, T3RUPT, USED  TO CALL A PROGRAM, (CALLED A TASK),

046944,000045:                                                                                                  #                               C(LST1 +2) = -(T4-T3)+1
046945,000046:                                                                                                  #                                          .
046946,000047:                                                                                                  #                                          .

046948,000049:                                                                                                  #                               C(LST1 +7) = -(T9-T8)+1
046949,000050: 
046950,000051:                                                                                                  #                               C(LST2)    = 2CADR OF TASK1
046951,000052:                                                                                                  #                               C(LST2 +2) = 2CADR OF TASK2
046952,000053:                                                                                                  #                                          .
046953,000054:                                                                                                  #                                          .
046954,000055:                                                                                                  #                               C(LST2 +14)= 2CADR OF TASK8
046955,000056:                                                                                                  #                               C(LST2 +16)= 2CADR OF TASK9
046956,000057: 
046957,000058:                                                                                                  #  WARNINGS -
046958,000059:                                                                                                  #  --------
046959,000060:                                                                                                  #               1) 1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
046960,000061:                                                                                                  #               2) 9 TASKS MAXIMUM
046961,000062:                                                                                                  #               3) TASKS CALLED UNDER INTERRUPT INHIBITED
046962,000063:                                                                                                  #               4) TASKS END BY TC TASKOVER

046965,000066: 
046966,000067:                                                                                                  #               L-1    CA     DELTAT   (TIME IN CENTISECONDS TO TASK START)
046967,000068:                                                                                                  #               L      TC     WAITLIST

047006,000107:                                                                                                  #  DETAILED ANALYSIS OF TIMING-
047007,000108: 
047008,000109:                                                                                                  #          CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR)   IN EXACTLY DELTA T CENTISECONDS.
047009,000110:                                                                                                  #     THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
047010,000111: 
047011,000112:                                                                                                  #               LET TO = THE TIME OF THE TC WAITLIST
047012,000113:                                                                                                  #               LET TS = TO +147U + COUNTER INCREMENTS (SET UP TIME)
047013,000114:                                                                                                  #               LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
047014,000115:                                                                                                  #               LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
047015,000116:                                                                                                  #               LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
047016,000117:                                                                                                  #     (Z=0, USUALLY)
047017,000118:                                                                                                  #               LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
047018,000119:                                                                                                  #            THEN   DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
047019,000120:                                                                                                  #     *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
047020,000121: 
047021,000122: 
047022,000123:                                                                                                  #          IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
047023,000124:                                                                                                  #     INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
047024,000125:                                                                                                  #     OF OTHER TASKS INHIBITING THE INTERRUPT.
