Source Code

These source-code files derive from a printout of Luminary 116 (the Apollo 12 Lunar Module guidance computer program), from the personal library of original AGC developer Don Eyles, digitally photographed at archive.org, financially sponsored by Ron Burkey, and transcribed to source code by a team of volunteers. This colorized, syntax-highlighted form was created by assembling that transcribed source code. Note that the full page images are available on the Virtual AGC project page at archive.org, while reduced-size images are presented at the VirtualAGC project website. Report or fix any transcription errors at the Virtual AGC project code repository.

Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 116 OF AGC PROGRAM LUMINARY BY NASA 2021112-071
	19:09 AUG. 11,1969
Note that the date is the date of the printout, not the date of the program revision.

042312,000026:                                                                                                  #  PROGRAM NAME- DOWN TELEMETRY PROGRAM
042313,000027:                                                                                                  #  MOD NO.- 0      TO COMPLETELY REWRITE THE DOWN TELEMETRY PROGRAM AND DOWNLINK ERASABLE DUMP PROGRAM FOR THE
042314,000028:                                                                                                  #                  PURPOSE OF SAVING APPROXIMATELY 150 WORDS OF CORE STORAGE.
042315,000029:                                                                                                  #                  THIS CHANGE REQUIRES AN ENTIRELY NEW METHOD OF SPECIFYING DOWNLINK LISTS.REFER TO DOWNLINK
042316,000030:                                                                                                  #                  LISTS LOG SECTION FOR MORE DETAILS. HOWEVER THIS CHANGE WILL NOT AFFECT THE GROUND PROCESSING
042317,000031:                                                                                                  #                  OF DOWN TELEMETRY DATA.
042318,000032:                                                                                                  #  MOD BY-  KILROY, SMITH, DEWITT
042319,000033:                                                                                                  #  DATE-    02 OCT 67
042320,000034:                                                                                                  #  AUTHORS- KILROY, SMITH, DEWITT, DEWOLF, FAGIN
042321,000035:                                                                                                  #  LOG SECTION- DOWN-TELEMETRY PROGRAM
042322,000036:                                                                                                  #  FUNCTIONAL DESCRIPTION- THIS ROUTINE IS INITIATED BY TELEMETRY END
042323,000037:                                                                                                  #           PULSE FROM THE DOWNLINK TELEMETRY CONVERTER. THIS PULSE OCCURS
042324,000038:                                                                                                  #           AT 50 TIMES PER SEC (EVERY 20 MS) THEREFORE DODOWNTM IS
042325,000039:                                                                                                  #           EXECUTED AT THESE RATES.  THIS ROUTINE SELECTS THE APPROPRIATE
042326,000040:                                                                                                  #           AGC DATA TO BE TRANSMITTED DOWNLINK AND LOADS IT INTO OUTPUT
042327,000041:                                                                                                  #           CHANNELS 34 AND 35. THE INFORMATION IS THEN GATED OUT FROM THE
042328,000042:                                                                                                  #           LGC IN SERIAL FASHION.
042329,000043:                                                                                                  #           THIS PROGRAM IS CODED FOR A 2 SECOND DOWNLIST. SINCE DOWNRUPTS
042330,000044:                                                                                                  #           OCCUR EVERY 20 MS AND 2 AGC COMPUTER WORDS CAN BE PLACED IN
042331,000045:                                                                                                  #           CHANNELS 34 AND 35 DURING EACH DOWNRUPT THE PROGRAM IS CAPABLE
042332,000046:                                                                                                  #           OF SENDING 200 AGC WORDS EVERY 2 SECONDS.
042333,000047:                                                                                                  #  CALLING SEQUENCE- NONE
042334,000048:                                                                                                  #           PROGRAM IS ENTERED VIA TCF DODOWNTM WHICH IS EXECUTED AS A
042335,000049:                                                                                                  #           RESULT OF A DOWNRUPT. CONTROL IS RETURNED VIA TCF RESUME WHICH
042336,000050:                                                                                                  #           IN EFFECT IS A RESUME.
042337,000051:                                                                                                  #  SUBROUTINES CALLED- NONE
042338,000052:                                                                                                  #  NORMAL EXIT MODE- TCF RESUME
042339,000053:                                                                                                  #  ALARM OR ABORT EXIT MODE - NONE
042340,000054:                                                                                                  #  RESTART PROTECTION:
042341,000055:                                                                                                  #           ON A FRESH START AND RESTART THE 'STARTSUB' SUBROUTINE WILL INITIALIZE THE DOWNLIST POINTER (ACTUALLY
042342,000056:                                                                                                  #           DNTMGOTO) TO THE BEGINNING OF THE CURRENT DOWNLIST(I.E. CURRENT CONTENTS OF DNLSTADR). THIS HAS THE
042343,000057:                                                                                                  #           EFFECT OF IGNORING THE REMAINDER OF THE DOWNLIST WHICH THE DOWN-TELEMETRY PROGRAM WAS WORKING ON WHEN
042344,000058:                                                                                                  #           THE RESTART (OR FRESH START) OCCURRED AND RESUME DOWN TELEMETRY FROM THE BEGINNING OF THE CURRENT
042345,000059:                                                                                                  #           DOWNLIST.
042346,000060:                                                                                                  #           ALSO OF INTEREST IS THE FACT THAT ON A RESTART THE AGC WILL ZERO DOWNLINK CHANNELS 13, 34 AND 35.
042347,000061:                                                                                                  #  DOWNLINK LIST SELECTION:
042348,000062:                                                                                                  #           THE APPROPRIATE DOWNLINK LISTS ARE SELECTED BY THE FOLLOWING:
042349,000063:                                                                                                  #           1.  FRESH START
042350,000064:                                                                                                  #           2.  V37EXXE  WHERE XX = THE MAJOR MODE BEING SELECTED.
042351,000065:                                                                                                  #           3.  UPDATE PROGRAM (P27)
042352,000066:                                                                                                  #           4.      NON-V37 SELECTABLE TYPE PROGRAMS (E.G. AGS INITIALIZATION(SUNDANCE, LUMINARY) AND P61-P62
042353,000067:                                                                                                  #                   TRANSITION(COLOSSUS) ETC.).
042354,000068: 
042355,000069:                                                                                                  #  DOWNLINK LIST RULES AND LIMITATIONS:
042356,000070:                                                                                                  #           READ SECTION(S) WHICH FOLLOW 'DEBRIS' WRITEUP.
042357,000071: 
042358,000072:                                                                                                  #  OUTPUT-  EVERY 2 SECONDS 100 DOUBLE PRECISION WORDS (I.E. 200 LGC
042359,000073:                                                                                                  #           COMPUTER WORDS) ARE TRANSMITTED VIA DOWNLINK.
042360,000074: 
042361,000075:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED- NONE
042362,000076:                                                                                                  #           'DNTMGOTO' AND 'DNLSTADR' ARE INITIALIZED BY THE FRESH START PROGRAM.
042363,000077: 
042364,000078:                                                                                                  #  DEBRIS(ERASABLE LOCATIONS DESTROYED BY THIS PROGRAM)-
042365,000079:                                                                                                  #        LDATALST, DNTMBUFF TO DNTMBUFF +21D, TMINDEX, DNQ.

042374,000088:                                                                                                  #  NOTES REGARDING DOWNLINK LISTS ASSOCIATED WITH THIS PROGRAM:
042375,000089:                                                                                                  #  1. DOWNLISTS. - DOWNLISTS MUST BE COMPILED IN THE SAME BANK AS THE
042376,000090:                                                                                                  #     DOWN-TELEMETRY PROGRAM. THIS IS DONE FOR EASE OF CODING, FASTER
042377,000091:                                                                                                  #     EXECUTION.
042378,000092:                                                                                                  #  2. EACH DOWNLINK LIST CONSISTS OF A CONTROL LIST AND A NUMBER OF
042379,000093:                                                                                                  #     SUBLISTS.
042380,000094:                                                                                                  #  3. A SUBLIST REFERS TO A SNAPSHOT OR DATA COMMON TO THE SAME OR OTHER
042381,000095:                                                                                                  #     DOWNLINK LISTS.  ANY SUBLIST CONTAINING COMMON DATA NEEDS TO BE
042382,000096:                                                                                                  #     CODED ONLY ONCE FOR THE APPLICABLE DOWNLINK LISTS.
042383,000097:                                                                                                  #  4. SNAPSHOT SUBLISTS REFER SPECIFICALLY TO HOMOGENOUS DATA WHICH MUST BE
042384,000098:                                                                                                  #     SAVED IN A BUFFER DURING ONE DOWNRUPT.
042385,000099:                                                                                                  #  5. THE 1DNADR FOR THE 1ST WORD OF SNAPSHOT DATA IS FOUND AT THE END
042386,000100:                                                                                                  #     OF EACH SNAPSHOT SUBLIST, SINCE THE PROGRAM CODING SENDS THIS DP WORD
042387,000101:                                                                                                  #     IMMEDIATELY AFTER STORING THE OTHERS IN THE SNAPSHOT BUFFER.
042388,000102:                                                                                                  #  6. ALL LISTS ARE COMBINATIONS OF CODED ERASABLE ADDRESS CONSTANTS
042389,000103:                                                                                                  #     CREATED FOR THE DOWNLIST PROGRAM.
042390,000104:                                                                                                  #     A. 1DNADR              1-WORD DOWNLIST ADDRESS.
042391,000105:                                                                                                  #         SAME AS ECADR, BUT USED WHEN THE WORD ADDRESSED IS THE LEFT
042392,000106:                                                                                                  #         HALF OF A DOUBLE-PRECISION WORD FOR DOWN TELEMETRY.
042393,000107:                                                                                                  #     B. 2DNADR - 6DNADR     N-WORD DOWNLIST ADDRESS, N = 2 - 6.
042394,000108:                                                                                                  #         SAME AS 1DNADR, BUT WITH THE 4 UNUSED BITS OF THE ECADR FORMAT
042395,000109:                                                                                                  #         FILLED IN WITH 0001-0101.  USED TO POINT TO A LIST OF N DOUBLE-
042396,000110:                                                                                                  #         PRECISION WORDS, STORED CONSECUTIVELY, FOR DOWN TELEMETRY.
042397,000111:                                                                                                  #     C. DNCHAN              DOWNLIST CHANNEL ADDRESS.
042398,000112:                                                                                                  #         SAME AS 1DNADR, BUT WITH PREFIX BITS 0111.  USED TO POINT TO
042399,000113:                                                                                                  #         A PAIR OF CHANNELS FOR DOWN TELEMETRY.
042400,000114:                                                                                                  #     D. DNPTR               DOWN TELEMETRY SUBLIST POINTER.
042401,000115:                                                                                                  #         SAME AS CAF BUT TAGGED AS A CONSTANT. USED IN CONTROL LIST TO POINT TO A SUBLIST.
042402,000116:                                                                                                  #         CAUTION--- A DNPTR CANNOT BE USED IN A SUBLIST.
042403,000117:                                                                                                  #  7. THE WORD ORDER CODE IS SET TO ZERO AT THE BEGINNING OF EACH DOWNLIST (I.E. CONTROL LIST) AND WHEN
042404,000118:                                                                                                  #     A '1DNADR TIME2' IS DETECTED IN THE CONTROL LIST (ONLY).
042405,000119:                                                                                                  #  8. IN THE SNAPSHOT SUBLIST ONLY, THE DNADR'S CANNOT POINT TO THE FIRST WORD OF ANY EBANK.

042416,000130:                                                                                                  #     EQUIVALENT OF THE FOLLOWING ECADRS(I.E. IDNADRS): 377, 777, 1377, 1777, 2377, 27777, 3377, 3777.

042627,000341:                                                                                                  #  SUBROUTINE NAME- DNDUMP
042628,000342: 
042629,000343:                                                                                                  #  FUNCTIONAL DESCRIPTION - TO SEND (DUMP) ALL 8 BANKS OF ERASABLE STORAGE TWICE.  BANKS ARE SENT ONE AT A TIME
042630,000344:                                                                                                  #           EACH BANK IS PRECEEDED BY AN ID WORD, SYNCH BITS, ECADR AND TIME1 FOLLOWED BY THE 256D WORDS OF EACH
042631,000345:                                                                                                  #           EBANK. EBANKS ARE DUMPED IN ORDER (I.E. EBANK 0 FIRST, THEN EBANK1 ETC.)
042632,000346: 
042633,000347:                                                                                                  #  CALLING SEQUENCE- THE GROUND OR ASTRONAUT BY KEYING V74E CAN INITIALIZE THE DUMP.
042634,000348:                                                                                                  #           AFTER KEYING IN V74E THE CURRENT DOWNLIST WILL BE IMMEDIATELY TERMINATED AND THE DOWNLINK ERASABLE DUMP
042635,000349:                                                                                                  #           WILL BEGIN.
042636,000350:                                                                                                  #           ONCE INITIATED THE DOWNLINK ERASABLE DUMP CAN BE TERMINATED (AND INTERRUPTED DOWNLIST REINSTATED) ONLY
042637,000351:                                                                                                  #           BY THE FOLLOWING:
042638,000352:                                                                                                  #            1. A FRESH START
042639,000353:                                                                                                  #            2. COMPLETION OF BOTH COMPLETE DUMPS
042640,000354:                                                                                                  #            3. AND INVOLUNTARILY BY A RESTART.
042641,000355:                                                                                                  #  NORMAL EXIT MODE- TCF DNPHASE1
042642,000356:                                                                                                  #  ALARM OR ABORT MODE- NONE
042643,000357:                                                                                                  #  *SUBROUTINES CALLED- NONE.
042644,000358:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED -
042645,000359:                                                                                                  #           NONE

042647,000361:                                                                                                  #  TIMING -   TIME (IN SECS) = ((NO.DUMPS)*(NO.EBANKS)*(WDSPEREBANK + NO.IDWDS)) / NO.WDSPERSEC
