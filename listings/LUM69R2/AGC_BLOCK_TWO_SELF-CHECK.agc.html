Source Code

These source-code files are part of a reconstructed copy of Luminary 69/2, the flown, final release of the Apollo 10 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction is based on the source code of Luminary 69 — i.e., the initial, unflown version, "revision 0" — of which a contemporary listing was available. The code was then updated by incorporating the differences between Luminary 69 and Luminary 69/2, known from other contemporary documentation. The only such difference is the implementation in Luminary 69/2 of the "R-2 Lunar Potential Model", the source code for which was taken from program Luminary 99/1 (Apollo 11 LM), of which a contemporary listing was also available. Finally, the now-reconstructed Luminary 69/2 was then validated by assembling it to executable form and verifying that its memory banks had the correct checksums, known from still other contemporary documentation.

Note that page numbers in the reconstructed code match those in the original Luminary 69 program listing, or in the ORBITAL INTEGRATION log section, in Luminary 99/1. Page numbers would certainly differ somewhat in a contemporary Luminary 69/2 listing. Moreover, annotations that were not present in the contemporary source code have been added to the reconstructed code to justify each change in Luminary 69/2 relative to Luminary 69. Here's a guide to the Apollo documentation referenced in those annotations:
  • MIT/IL engineering drawing 2021152N, sheet 3. Listing of the memory-bank checksums for Luminary 69/2 (called "LUMINARY 69 REV. 2" in the document itself).
  • Luminary 99/1 (Apollo 11 LM final release) assembly listing.
  • Luminary 69 (Apollo 10 LM initial release) assembly listing.
  • LUMINARY Memo #75, "R-2 Lunar Potential Model Added to LUM".
  • LUMINARY Memo #78, "LUMINARY Revisions 93-96".
Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

055126,000027:                                                                                                  #        PROGRAM HAS TWO MAIN PARTS. THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
055127,000028:                                                                                                  #  PART OF THE BACK-UP IDLE LOOP. THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
055128,000029:                                                                                                  #  STARTING VERB.
055129,000030:                                                                                                  #        THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
055130,000031:                                                                                                  #        THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK, ONE AT A TIME.
055131,000032:                                                                                                  #        IN ALL THERE ARE 7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK. MORE DETAIL DESCRIPTION MAY BE
055132,000033:                                                                                                  #  FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
055133,000034: 
055134,000035:                                                                                                  #        THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27). BELOW IS
055135,000036:                                                                                                  #   A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
055136,000037:                                                                                                  #  OCTAL, TO LOAD INTO SMODE.
055137,000038:                                                                                                  #  +-4   ERASABLE MEMORY
055138,000039:                                                                                                  #  +-5   FIXED MEMORY
055139,000040:                                                                                                  #  +-1,2,3,6,7,10   EVERYTHING IN OPTIONS 4 AND 5.
055140,000041:                                                                                                  #  -0    SAME AS +-10 UNTIL AN ERROR IS DETECTED.
055141,000042:                                                                                                  #  +0    NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
055142,000043: 
055143,000044:                                                                                                  #  WARNINGS
055144,000045:                                                                                                  #        USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
055145,000046:                                                                                                  #        SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
055146,000047: 
055147,000048:                                                                                                  #  CALLING SEQUENCE
055148,000049:                                                                                                  #        TO CALL SELF-CHECK KEY IN
055149,000050:                                                                                                  #                V 21 N 27 E     OPTION NUMBER E
055150,000051:                                                                                                  #        TO CALL SHOW-BANKSUM KEY IN
055151,000052:                                                                                                  #                V 91 E          DISPLAYS FIRST BANK
055152,000053:                                                                                                  #                V 33 E          PROCEED, DISPLAYS NEXT BANK
055153,000054: 
055154,000055:                                                                                                  #  EXIT MODES, NORMAL AND ALARM

055156,000057:                                                                                                  #  COMPUTER INTO BACKUP IDLE LOOP, - OPTION NUMBERS RESTART THE OPTION.
055157,000058:                                                                                                  #        THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
055158,000059:                                                                                                  #        SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E).  THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP
