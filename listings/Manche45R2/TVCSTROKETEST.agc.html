Source Code

These source-code files are part of a reconstructed copy of Comanche 45/2 (AKA Manche45 revision 2), the final, flown mission release of the Apollo 10 Command Module (CM) Apollo Guidance Computer (AGC) software. Comanche 45/2 was preceded by Comanche 44 and Comanche 45. The reconstruction is believed to be exactly accurate, but in the absence of an actual Apollo-era listing of Comanche 45/2, some explanation of the reconstruction process is needed to insure confidence in that accuracy.

The starting point was the source code of Comanche 45 (itself previously accurately reconstructed). Comanche 45 source code was then modified by incorporating changes known from contemporary documentation to have been made between revisions 45 and 45/2. The now-reconstructed Comanche 45/2 source code was verified by assembling it and checking that the assembled code had the expected known-correct memory-bank checksums. Note that page numbers in the reconstructed code tend to match those in the Comanche 55, the most-similar revision of Comanche for which we have access to an actual non-reconstructed hardcopy; the source code would likely have different page numbers in a contemporary Comanche 45 listing.

Annotations that were not present in the contemporary source code have been added to the reconstructed code to justify each change relative to Comanche 44. Here's a guide to the Apollo documentation referenced in those annotations:
  • MIT/IL engineering drawing 2021153D, sheet 3. Listing of the memory-bank checksums for Comanche 45.
  • Reconstructed Comanche 45 (Apollo 10 CM second release) assembly listing.
  • Reconstructed Comanche 51 (Apollo 11 CM initial release) assembly listing.
  • Programmed Guidance Equations for Colossus 2, p. 91.
Comments expected to have been present in the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC Project's GitHub repository.

042719,000020:                                                                                                  #  LOG SECTION...TVCSTROKETEST                   SUBROUTINE...DAPCSM
042720,000021:                                                                                                  #  MODIFIED BY SCHLUNDT                          21 OCTOBER 1968
042721,000022: 
042722,000023:                                                                                                  #  FUNCTIONAL DESCRIPTION....
042723,000024:                                                                                                  #        STROKE TEST PACKAGE GENERATES A WAVEFORM DESIGNED TO EXCITE BENDING
042724,000025:                                                                                                  #        STRKTSTI (STROKE TEST INITIALIZATION) IS CALLED AS A JOB BY VB68.
042725,000026:                                                                                                  #                IT INITIALIZES ALL ERASABLES REQD FOR A STROKE TEST, AND
042726,000027:                                                                                                  #                THEN TESTS FOR CSM/LM (BIT 13 OF DAPDATR1). IF CSM/LM,
042727,000028:                                                                                                  #                IN EITHER HIGH OR LOW-BANDWIDTH MODE, THE TEST IS STARTED
042728,000029:                                                                                                  #                IMMEDIATELY.  IF NOT CSM/LM, PROGRAM EXITS WITH NO ACTION.
042729,000030:                                                                                                  #        HACK (STROKE TEST) GENERATES THE WAVEFORM BY DUMPING PULSE BURSTS
042730,000031:                                                                                                  #                OF PROPER SIGN AND IN PROPER SEQUENCE DIRECTLY INTO
042731,000032:                                                                                                  #                TVCPITCH, WORKING IN CONJUNCITON WITH BOTH PITCH AND YAW
042732,000033:                                                                                                  #                TVC DAPS, WITH INTERMEDIATE WAITLIST CALLS.  NOTE, HOWEVER
042733,000034:                                                                                                  #                THAT THE STROKE TEST IS PERFORMED ONLY IN THE PITCH AXIS.
042734,000035:                                                                                                  #                AN EXAMPLE WAVEFORM IS GIVEN BELOW, TO DEMONSTRATE STROKE-
042735,000036:                                                                                                  #                TEST PARAMETER SELECTION
042736,000037:                                                                                                  #        RESTARTS CAUSE TEST TO BE TERMINATED.  ANOTHER V68 REQD IF TEST
042737,000038:                                                                                                  #                IS TO BE RE-RUN.
042738,000039:                                                                                                  #        PULSE BURST SIZE IS PAD-LOADED (ESTROKER) SO THAT AMPLITUDE OF
042739,000040:                                                                                                  #                WAVEFORM CAN BE CHANGED.  THERE ARE TEN PULSE BURSTS IN
042740,000041:                                                                                                  #                THE HALF-AMPLITUDE OF THE FIRST FREQUENCY SET IN THE
042741,000042:                                                                                                  #                STANDARD WAVEFORM.  AMPLITUDE IS 10(ESTROKER)(1/42.15),
042742,000043:                                                                                                  #                NOMINALLY 50/42.15 = 1.185 DEG
042743,000044: 
042744,000045:                                                                                                  #  CALLING SEQUENCE....
042745,000046:                                                                                                  #        EXTENDED VERB 68 SETS UP STRKTSTI JOB
042746,000047:                                                                                                  #        PITCH AND YAW TVCDAPS, FINDING STROKER NON-ZERO, DO A ..TC HACK..
042747,000048:                                                                                                  #        AN INTERNALLY-GENERATED WAITLIST CALL ENTERS AT ..HACKWLST..
042748,000049: 
042749,000050:                                                                                                  #  NORMAL EXIT MODES....
042750,000051:                                                                                                  #        TC BUNKER (..Q.. IF ENTRY FROM DAP, ..TCTSKOVR.. IF FROM WAITLIST) LIST
042751,000052: 
042752,000053:                                                                                                  #  SUBROUTINES CALLED....
042753,000054:                                                                                                  #        WAITLIST
042754,000055: 
042755,000056:                                                                                                  #  ALARM OR ABORT EXIT MODES....
042756,000057:                                                                                                  #        NONE
042757,000058: 
042758,000059:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED....
042759,000060:                                                                                                  #        ESTROKER (PAD-LOAD)
042760,000061:                                                                                                  #        STROKER, CADDY, REVS, CARD, N
042761,000062: 
042762,000063:                                                                                                  #  OUTPUT....
042763,000064:                                                                                                  #        STRKTSTI...INITIALIZATION FOR STROKE TEST
042764,000065:                                                                                                  #        HACK, HACKWLST...PULSE BURSTS INTO TVCPITCH VIA ..ADS..
042765,000066:                                                                                                  #                          RESETS STROKER = +0 WHEN TEST COMPLETED

042769,000070:                                                                                                  #        BUNKER
