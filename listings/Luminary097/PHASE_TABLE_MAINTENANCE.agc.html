Source Code

These source-code files are part of a reconstructed copy of Luminary 97, the original release of the Apollo 11 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Luminary 99 revision 1 previously transcribed from a digitized copy of that program. The code was then updated by undoing changes described in original Luminary memos 83 and 85, using asterisks indicating changed lines in the listing as a guide. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021152D. Note that page numbers in the reconstructed code match those on the Luminary 099 revision 001 printout, although the added code would likely have changed page numbers for a real Luminary 97 listing.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

054520,000074:                                                                                                  #  PHASCHNG IS THE MAIN WAY OF MAKING PHASE CHANGES FOR RESTARTS.  THERE ARE THREE FORMS OF PHASCHNG, KNOWN AS TYPE
054521,000075:                                                                                                  #  A, TYPE B, AND TYPE C.  THEY ARE ALL CALLED AS FOLLOWS, WHERE OCT XXXXX CONTAINS THE PHASE INFORMATION,
054522,000076:                                                                                                  #                TC      PHASCHNG
054523,000077:                                                                                                  #                OCT     XXXXX
054524,000078:                                                                                                  #  TYPE A IS CONCERNED WITH FIXED PHASE CHANGES, THAT IS, PHASE INFORMATION THAT IS STORED PERMANENTLY.  THESE
054525,000079:                                                                                                  #  OPTIONS ARE, WHERE G STANDS FOR A GROUP AND .X FOR THE PHASE,
054526,000080:                                                                                                  #        G.0             INACTIVE, WILL NOT PERMIT A GROUP G RESTART
054527,000081:                                                                                                  #        G.1             WILL CAUSE THE LAST DISPLAY TO BE REACTIVATED, USED MAINLY IN MANNED FLIGHTS
054528,000082:                                                                                                  #        G.EVEN          A DOUBLE TABLE RESTART, CAN CAUSE ANY COMBINATION OF TWO JOBS, TASKS, AND/OR
054529,000083:                                                                                                  #                        LONGCALL TO BE RESTARTED.
054530,000084:                                                                                                  #        G.ODD NOT .1    A SINGLE TABLE RESTART, CAN CAUSE EITHER A JOB, TASK, OR LONGCALL RESTART
054531,000085: 
054532,000086:                                                                                                  #  THIS INFORMATION IS PUT INTO THE OCTAL WORD AFTER TC PHASCHNG AS FOLLOWS
054533,000087:                                                                                                  #        TL0 00P PPP PPP GGG                                               ,
054534,000088:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G:S STAND FOR THE GROUP, OCTAL 1-7, THE P:S FOR THE PHASE,
054535,000089:                                                                                                  #  OCTAL 0 - 127.  0:S MUST BE 0.  IF ONE WISHES TO HAVE THE TBASE OF GROUP G TO BE SET AT THIS TIME,
054536,000090:                                                                                                  #  T IS SET TO 1, OTHERWISE IT IS SET TO 0.  SIMIARLY IF ONE WISHES TO SET LONGBASE, THEN L IS SET TO 1, OTHERWISE
054537,000091:                                                                                                  #  IT IS SET TO 0.  SOME EXAMPLES,
054538,000092:                                                                                                  #                TC      PHASCHNG        THIS WILL CAUSE GROUP 3 TO BE SET TO 0,
054539,000093:                                                                                                  #                OCT     00003           MAKING GROUP 3 INACTIVE
054540,000094: 
054541,000095:                                                                                                  #                TC      PHASCHNG        IF A RESTART OCCURS THIS WOULD CAUSE
054542,000096:                                                                                                  #                OCT     00012           GROUP 2 TO RESTART THE LAST DISPLAY
054543,000097: 
054544,000098:                                                                                                  #                TC      PHASCHNG        THIS SETS THE TBASE OF GROUP 4 AND IN
054545,000099:                                                                                                  #                OCT     40064           CASE OF A RESTART WOULD START UP THE TWO
054546,000100:                                                                                                  #                                        THINGS LOCATED IN THE DOUBLE 4.6 RESTART
054547,000101:                                                                                                  #                                        LOCATION
054548,000102: 
054549,000103:                                                                                                  #                TC      PHASCHNG        THIS SETS LONGBASE AND UPON A RESTART
054550,000104:                                                                                                  #                OCT     20135           CAUSES 5.13 TO BE RESTARTED (SINCE
054551,000105:                                                                                                  #                                        LONGBASE WAS SET THIS SINGLE ENTRY
054552,000106:                                                                                                  #                                        SHOULD BE A LONGCALL)
054553,000107: 
054554,000108:                                                                                                  #                TC      PHASCHNG        SINCE BOTH TBASE4 AND LONGBASE ARE SET,

054558,000112:                                                                                                  #  TYPE C PHASCHNG CONTAINS THE VARIABLE TYPE OF PHASCHNG INFORMATION.  INSTEAD OF THE INFORMATION BEING IN A
054559,000113:                                                                                                  #  PERMANENT FORM, ONE STORES THE DESIRED RESTART INFORMATION IN A VARIABKE LOCATION.  THE BITS ARE AS FOLLOWS,
054560,000114:                                                                                                  #        TL0 1AD XXX CJW GGG
