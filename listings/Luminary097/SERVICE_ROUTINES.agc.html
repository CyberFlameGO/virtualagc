Source Code

These source-code files are part of a reconstructed copy of Luminary 97, the original release of the Apollo 11 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Luminary 99 revision 1 previously transcribed from a digitized copy of that program. The code was then updated by undoing changes described in original Luminary memos 83 and 85, using asterisks indicating changed lines in the listing as a guide. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021152D. Note that page numbers in the reconstructed code match those on the Luminary 099 revision 001 printout, although the added code would likely have changed page numbers for a real Luminary 97 listing.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

057869,000047:                                                                                                  #  UPFLAG AND DOWNFLAG ARE ENTIRELY GENERAL FLAG SETTING AND CLEARING SUBROUTINES.  USING THEM, WHETHER OR
057870,000048:                                                                                                  #  NOT IN INTERRUPT, ONE MAY SET OR CLEAR ANY SINGLE, NAMED BIT IN ANY ERASABLE REGISTER, SUBJECT OF COURSE TO
057871,000049:                                                                                                  #  EBANK SETTING.  A "NAMED" BIT, AS THE WORD IS USED HERE, IS ANY BIT WITH A NAME FORMALLY ASSIGNED BY THE YUL
057872,000050:                                                                                                  #  ASSEMBLER.
057873,000051: 
057874,000052:                                                                                                  #  AT PRESENT THE ONLY NAMED BITS ARE THOSE IN THE FLAGWORDS.  ASSEMBLER CHANGES WILL MAKE IT POSSIBLE TO
057875,000053:                                                                                                  #  NAME ANY BIT IN ERASABLE MEMORY.
057876,000054: 
057877,000055:                                                                                                  #  CALLING SEQUENCES ARE AS FOLLOWS :-
057878,000056:                                                                                                  #                TC      UPFLAG                  TC      DOWNFLAG
057879,000057:                                                                                                  #                ADRES   NAME OF FLAG            ADRES   NAME OF FLAG

057985,000163:                                                                                                  #  GENTRAN, A BLOCK TRANSFER ROUTINE.
057986,000164:                                                                                                  #  WRITTEN BY D. EYLES
057987,000165:                                                                                                  #  MOD 1 BY KERNAN                               UTILITYM REV 17 11/18/67
057988,000166:                                                                                                  #  MOD 2 BY SCHULENBERG (REMOVE RELINT) SKIPPER REV 4 2/28/68
057989,000167: 
057990,000168:                                                                                                  #  THIS ROUTINE IS USEFULL FOR TRANSFERING N CONSECUTIVE ERASABLE OR FIXED QUANTITIES TO SOME OTHER N
057991,000169:                                                                                                  #  CONSECUTIVE ERASABLE LOCATIONS.  IF BOTH BLOCKS OF DATA ARE IN SWITCHABLE EBANKS, THEY MUST BE IN THE SAME ONE.
057992,000170: 
057993,000171:                                                                                                  #  GENTRAN IS CALLABLE IN A JOB AS WELL AS A RUPT.  THE CALLING SEQUENCE IS:
057994,000172:                                                                                                  #        I       CA      N-1             # OF QUANTITIES MINUS ONE.
057995,000173:                                                                                                  #        I +1    TC      GENTRAN         IN FIXED-FIXED.
057996,000174:                                                                                                  #        I +2    ADRES   L               STARTING ADRES OF DATA TO BE MOVED.
057997,000175:                                                                                                  #        I +3    ADRES   M               STARTING ADRES OF DUPLICATION BLOCK.
057998,000176:                                                                                                  #        I +4                            RETURNS HERE.
057999,000177: 
058000,000178:                                                                                                  #  GENTRAN TAKES 25 MCT'S (300 MICROSECONDS) PER ITEM + 5 MCT'S (60 MICS) FOR ENTERING AND EXITING.
058001,000179:                                                                                                  #  A, L AND ITEMP1 ARE NOT PRESERVED.
