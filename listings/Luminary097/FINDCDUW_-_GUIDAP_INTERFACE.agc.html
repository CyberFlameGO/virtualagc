Source Code

These source-code files are part of a reconstructed copy of Luminary 97, the original release of the Apollo 11 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Luminary 99 revision 1 previously transcribed from a digitized copy of that program. The code was then updated by undoing changes described in original Luminary memos 83 and 85, using asterisks indicating changed lines in the listing as a guide. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021152D. Note that page numbers in the reconstructed code match those on the Luminary 099 revision 001 printout, although the added code would likely have changed page numbers for a real Luminary 97 listing.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

039024,000023:                                                                                                  #  PROGRAM NAME:  FINDCDUW
039025,000024:                                                                                                  #  MOD NUMBER:  1        68 07 15
039026,000025:                                                                                                  #  MOD AUTHOR:  KLUMPP
039027,000026: 
039028,000027:                                                                                                  #  OBJECTS OF MOD:       1.      TO SUPPLY COMMANDED GIMBAL ANGLES FOR NOUN 22.
039029,000028:                                                                                                  #                        2.      TO MAINTAIN CORRECT AND CURRENT THRUST
039030,000029:                                                                                                  #                                DIRECTION DATA IN ALL MODES.  THIS IS DONE BY
039031,000030:                                                                                                  #                                FETCHING FOR THE THRUST DIRECTION FILTER THE
039032,000031:                                                                                                  #                                CDUD'S IN PNGCS-AUTO, THE CDU'S IN ALL OTHER
039033,000032:                                                                                                  #                                MODES.
039034,000033:                                                                                                  #                        3.      TO SUBSTITUTE A STOPRATE FOR THE NORMAL
039035,000034:                                                                                                  #                                AUTOPILOT COMMANDS WHENEVER
039036,000035:                                                                                                  #                                1) NOT IN PNGCS-AUTO, OR
039037,000036:                                                                                                  #                                2) ENGINE IS OFF.
039038,000037: 
039039,000038:                                                                                                  #  FUNCTIONAL DESCRIPTION:
039040,000039: 
039041,000040:                                                                                                  #  FINDCDUW PROVIDES THE INTERFACES BETWEEN THE VARIOUS POWERED FLITE GUIDANCE PROGRAMS
039042,000041:                                                                                                  #  AND THE DIGITAL AUTOPILOT.  THE INPUTS TO FINDCDUW ARE THE THRUST COMMAND VECTOR
039043,000042:                                                                                                  #  AND THE WINDOW COMMAND VECTOR, AND THE OUTPUTS ARE THE GIMBAL ANGLE
039044,000043:                                                                                                  #  INCREMENTS, THE COMMANDED ATTITUDE ANGLE RATES, AND THE COMMANDED
039045,000044:                                                                                                  #  ATTITUDE LAG ANGLES (WHICH ACCOUNT FOR THE ANGLES BY WHICH THE BODY WILL
039046,000045:                                                                                                  #  LAG BEHIND A RAMP COMMAND IN ATTITUDE ANGLE DUE TO THE FINITE ANGULAR
039047,000046:                                                                                                  #  ACCELERATIONS AVAILABLE).
039048,000047: 
039049,000048:                                                                                                  #  FINDCDUW ALINES THE ESTIMATED THRUST VECTOR FROM THE THRUST DIRECTION
039050,000049:                                                                                                  #  FILTER WITH THE THRUST COMMAND VECTOR, AND, WHEN XOVINHIB SET,
039051,000050:                                                                                                  #  ALINES THE +Z HALF OF THE LM ZX PLANE WITH THE WINDOW COMMAND VECTOR.

039067,000066:                                                                                                  #                        3.      IF ENGINE OFF, DO STOPRATE AND RETURN WITHOUT
039068,000067:                                                                                                  #                                ISSUING AUTOPILOT CMDS.
