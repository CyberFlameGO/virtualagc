Source Code

These source-code files are part of a reconstructed copy of Luminary 97, the original release of the Apollo 11 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Luminary 99 revision 1 previously transcribed from a digitized copy of that program. The code was then updated by undoing changes described in original Luminary memos 83 and 85, using asterisks indicating changed lines in the listing as a guide. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021152D. Note that page numbers in the reconstructed code match those on the Luminary 099 revision 001 printout, although the added code would likely have changed page numbers for a real Luminary 97 listing.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

059678,000324:                                                                                                  #  FDAI ATTITUDE ERROR DISPLAY SUBROUTINE
059679,000325: 
059680,000326:                                                                                                  #  PROGRAM DESCRIPTION:          D. KEENE        5/24/67
059681,000327:                                                                                                  #  MOD 1 BY CRAIG WORK, 12 DEC 67
059682,000328:                                                                                                  #  MOD 2 BY CRAIG WORK, 6 APRIL 68 CONVERTS ATTITUDE ERROR DISPLAY SCALING FROM 16 7/8 DEG. TO 42 3/16 DEGREES.
059683,000329: 
059684,000330:                                                                                                  #  THIS SUBROUTINE IS USED TO DISPLAY ATTITUDE ERRORS ON THE FDAI VIA THE DIGITAL TO ANALOG CONVERTERS (DACS)
059685,000331:                                                                                                  #  IN THE CDUS.  CARE IS TAKEN TO METER OUT THE APPROPRIATE NUMBER OF PULSES TO THE IMU ERROR COUNTERS AND PREVENT
059686,000332:                                                                                                  #  OVERFLOW, TO CONTROL THE RELAY SEQUENCING, AND TO AVOID INTERFERENCE WITH THE COARSE ALIGN LOOP WHICH ALSO USES
059687,000333:                                                                                                  #  THE DACS.
059688,000334: 
059689,000335:                                                                                                  #  CALLING SEQUENCE:
059690,000336:                                                                                                  #        DURING THE INITIALIZATION SECTION OF THE USER'S PROGRAM, BIT3 OF RCSFLAGS SHOULD BE SET TO INITIATE THE
059691,000337:                                                                                                  #        TURN-ON SEQUENCE WITHIN THE NEEDLES PROGRAM:
059692,000338:                                                                                                  #                CS      RCSFLAGS        IN EBANK6
059693,000339:                                                                                                  #                MASK    BIT3
059694,000340:                                                                                                  #                ADS     RCSFLAGS
059695,000341:                                                                                                  #        THEREAFTER, THE ATTITUDE ERRORS GENERATED BY THE USER SHOULD BE TRANSFERED TO THE FOLLOWING LOCATIONS IN EBANK6:
059696,000342:                                                                                                  #                AK      SCALED 180 DEGREES      NOTE:   THESE LOCATIONS ARE SUBJECT
059697,000343:                                                                                                  #                AK1     SCALED 180 DEGREES              TO CHANGE
059698,000344:                                                                                                  #                AK2     SCALED 180 DEGREES
059699,000345:                                                                                                  #        FULL SCALED DEFLECTION OF THE NEEDLES CORRESPONDS TO 5 1/16 DEGREES, WHILE 384 BITS IN THE IMU ERROR COUNTER
059700,000346:                                                                                                  #        CORRESPONDS TO 42 3/16 DEGREES.  (DAC MAXIMUM CAPACITY IS 384 BITS.)  46 BITS EFFECTIVELY PIN THE NEEDLES.
059701,000347: 
059702,000348:                                                                                                  #        A CALL TO NEEDLER WILL THEN UPDATE THE DISPLAY:
059703,000349:                                                                                                  #                INHINT
059704,000350:                                                                                                  #                TC      IBNKCALL        NOTE:  EBANK SHOULD BE SET TO E6
059705,000351:                                                                                                  #                CADR    NEEDLER
059706,000352:                                                                                                  #                RELINT
059707,000353:                                                                                                  #        THIS PROCESS SHOULD BE REPEATED EACH TIME THE ERRORS ARE UPDATED.  AT LEAST 3 PASSES THRU THE PROGRAM ARE
059708,000354:                                                                                                  #        REQUIRED BEFORE ANYTHING IS ACTUALLY DISPLAYED ON THE ERROR METERS.
059709,000355:                                                                                                  #        NOTE:  EACH CALL TO NEEDLER MUST BE SEPARATED BY AT LEAST 50 MS TO ASSURE PROPER RELAY SEQUENCING.
