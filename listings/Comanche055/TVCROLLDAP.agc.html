Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

044068,000041:                                                                                                  #  FUNCTIONAL DESCRIPTION....
044069,000042: 
044070,000043:                                                                                                  #       *AN ADAPTATION OF THE LEM P-AXIS CONTROLLER
044071,000044:                                                                                                  #       *MAINTAIN OGA WITHIN 5 DEG DEADBND OF OGAD, WHERE OGAD = OGA AS SEEN
044072,000045:                                                                                                  #        BY IGNITION (P40)
044073,000046:                                                                                                  #       *MAINTAIN OGA RATE LESS THAN 0.1 DEG/SEC LIMIT CYCLE RATE
044074,000047:                                                                                                  #       *SWITCHING LOGIC IN PHASE PLANE.... SEE GSOP CHAPTER 3
044075,000048:                                                                                                  #       *USES T6 CLOCK TO TIME JET FIRINGS
044076,000049:                                                                                                  #       *MAXIMUM JET FIRING TIME = 2.56 SECONDS, LIMITED TO 2.5 IF GREATER
044077,000050:                                                                                                  #       *MINIMUM JET FIRING TIME = 15 MS
044078,000051:                                                                                                  #       *JET PAIRS FIRE ALTERNATELY
044079,000052:                                                                                                  #       *AT LEAST 1/2 SECOND DELAY BEFORE A NEW JET PAIR IS FIRED
044080,000053:                                                                                                  #       *JET FIRINGS MAY NOT BE EXTENDED, ONLY SHORTENED, WHEN RE-EVALUATION
044081,000054:                                                                                                  #        OF A JET FIRING TIME IS MADE ON A LATER PASS
044082,000055: 
044083,000056:                                                                                                  #  CALLING SEQUENCE....
044084,000057: 
044085,000058:                                                                                                  #       *ROLLDAP CALL VIA WAITLIST, IN PARTICULAR BY TVCEXEC (EVERY 1/2 SEC)
044086,000059:                                                                                                  #        WITH A 3CS DELAY TO ALLOW FREE TIME FOR OTHER RUPTS (DWNRPT, ETC.)
044087,000060: 
044088,000061:                                                                                                  #  NORMAL EXIT MODES.... ENDOFJOB
044089,000062: 
044090,000063:                                                                                                  #  ALARM OR ABORT EXIT MODES.... NONE
044091,000064: 
044092,000065:                                                                                                  #  SUBROUTINES CALLED.....NONE
044093,000066: 
044094,000067:                                                                                                  #  OTHER INTERFACES....
044095,000068: 
044096,000069:                                                                                                  #       *TVCEXEC SETS UP ROLLDAP TASK EVERY 1/2 SECOND AND UPDATES 1/CONACC
044097,000070:                                                                                                  #        EVERY 10 SECONDS (VIA MASSPROP AND S40.15)
044098,000071:                                                                                                  #       *RESTARTS SUSPEND ROLL DAP COMPUTATIONS UNTIL THE NEXT 1/2 SEC

044125,000098:                                                                                                  #                                  *
044126,000099:                                                                                                  #   * * * * * * * * * * *          *
044127,000100:                                                                                                  #                                  *     (REGION 1, SEE TEXT BELOW)

044133,000106:                                                                                                  #                    *             *                (FIRE NEG ROLL JETS)

044138,000111:                                                                                                  #  ************************************************************************
044139,000112:                                                                                                  #                                  *         *                (-AK, OGAERR)
044140,000113:                                                                                                  #                         *        *         *      (REGION 6-PRIME)
044141,000114:                                                                                                  #                                  *         *      (SEE TEXT BELOW)
044142,000115:                                                                                                  #                           *      *           *
044143,000116:                                                                                                  #                                  *             *     ...STRAIGHT LINE
044144,000117:                                                                                                  #     (FIRE POS ROLL JETS)     *   *               *  .
044145,000118:                                                                                                  #                                  *   (COAST)       *
044146,000119:                                                                                                  #                                  *                   * * * * * * * * * *
044147,000120:                                                                                                  #                                  *                         -MINLIM
044148,000121:                                                                                                  #                                  *    *
044149,000122:                                                                                                  #                                  *
044150,000123:                                                                                                  #                                  *          * * * * * * * * * * * * * * *
044151,000124:                                                                                                  #                                  *                        -MAXLIM
044152,000125:                                                                                                  #                                  *
044153,000126:                                                                                                  #                                  *
044154,000127: 
044155,000128:                                                                                                  #  SWITCHING PARABOLAS ARE CONTROL PARABOLAS, THUS REQUIRING KNOWLEDGE OF
044156,000129:                                                                                                  #        CONTROL ACCELERATION CONACC, OR ITS RECIPROCAL, 1/CONACC, THE TVC
044157,000130:                                                                                                  #        ROLL DAP GAIN  (SEE TVCEXECUTIVE VARIABLE GAIN PACKAGE).  JET
044158,000131:                                                                                                  #        FIRING TIME IS SIMPLY THAT REQUIRED TO ACHIEVE THE DESIRED OGARATE,
044159,000132:                                                                                                  #        SUBJECT TO THE LIMITATIONS DISCUSSED UNDER FUNCTIONAL DESCRIPTION,
044160,000133:                                                                                                  #        ABOVE.
044161,000134: 
044162,000135:                                                                                                  #  THE THREE CONTROL REGIONS  (+, -, AND ZERO TORQUE) ARE COMPRIZED OF
044163,000136:                                                                                                  #        TWELVE SUBSET REGIONS  ( 1...6, AND THE CORRESPONDING 1-PRIME...
044164,000137:                                                                                                  #        6-PRIME )  SEE SECTION 3 OF THE GSOP  (SUNDISK OR COLOSSUS)

044166,000139: 
044167,000140:                                                                                                  #  GIVEN THE OPERATING POINT NOT IN THE COAST REGION, THE DESIRED OGARATE
044168,000141:                                                                                                  #        IS AT THE POINT OF PENETRATION OF THE COAST REGION BY THE CONTROL
044169,000142:                                                                                                  #        PARABOLA WHICH PASSES THROUGH THE OPERATING POINT.  FOR REGION 3
044170,000143:                                                                                                  #        DESIRED OGARATE IS SIMPLY +-MAXLIM.  FOR REGIONS 1 OR 6 THE SOLUTION
044171,000144:                                                                                                  #        TO A QUADRATIC IS REQUIRED (THE PENETRATION IS ALONG THE STRAIGHT
044172,000145:                                                                                                  #        LINE OR MINLIM BOUNDRY SWITCH LINES).  AN APPROXIMATION IS MADE
044173,000146:                                                                                                  #        INSTEAD. TAKE AN OPERATING POINT IN REGION 6'.  PASS A TANGENT TO
044174,000147:                                                                                                  #        THE CONTROL PARABOLA THROUGH THE OPERATING POINT, AND FIND ITS
044175,000148:                                                                                                  #        INTERSECTION WITH THE STRAIGHT LINE SECTION OF THE SWITCH CURVE...
044176,000149:                                                                                                  #        THE INTERSECTION DEFINES DESIRED OGARATE.  IF THE OPERATING POINT IS
044177,000150:                                                                                                  #        CLOSE TO THE SWITCH LINE, THE APPROXIMATION IS QUITE GOOD (INDEED
044178,000151:                                                                                                  #        THE APPROXIMATE AND QUADRATIC SOLUTIONS CONVERGE IN THE LIMIT AS
044179,000152:                                                                                                  #        THE SWITCH LINE IS APPROACHED).  IF THE OPERATING POINT IS NOT CLOSE
044180,000153:                                                                                                  #        TO THE SWITCH LINE, THE APPROXIMATE SOLUTION GIVES VALID TREND
044181,000154:                                                                                                  #        INFORMATION (DIRECTION OF DESIRED OGARATE) AT LEAST.  THE 
044182,000155:                                                                                                  #        RE-EVALUATION OF DESIRED OGARATE IN SUBSEQUENT ROLL DAP PASSES (1/2
044183,000156:                                                                                                  #        SECOND INTERVALS) WILL BENEFIT FROM THE CONVERGENT NATURE OF THE
044184,000157:                                                                                                  #        APPROXIMATION.
044185,000158: 
044186,000159:                                                                                                  #  FOR LARGE OGAERROR THE TANGENT INTERSECTS +-MINLIM SWITCH BOUNDRY BEFORE
044187,000160:                                                                                                  #        INTERSECTING THE STRAIGHT LINE SWITCH.  HOWEVER THE MINLIM IS
044188,000161:                                                                                                  #        IGNORED IN COMPUTING THE FIRING TIME, SO THAT THE EXTENSION (INTO
044189,000162:                                                                                                  #        THE COAST REGION) OF THE STRAIGHT LINE SWITCH IS WHAT IS FIRED TO.
044190,000163:                                                                                                  #        IF THE ROLL DAP FINDS ITSELF IN THE COAST REGION BEFORE REACHING
044191,000164:                                                                                                  #        THE DESIRED INTERSECTION (IE, IN THE REGION BETWEEN THE MINLIM
044192,000165:                                                                                                  #        AND THE STRAIGHT LINE SWITCH) IT WILL EXIBIT NORMAL COAST-REGION
044193,000166:                                                                                                  #        BEHAVIOR AND TURN OFF THE JETS.  THE PURPOSE OF THIS FIRING POLICY
044194,000167:                                                                                                  #        IS TO MAINTAIN STATIC ROLL STABILITY IN THE EVENT OF A JET
044195,000168:                                                                                                  #        FAILED-ON.
044196,000169: 
044197,000170:                                                                                                  #  WHEN THE OPERATING POINT IS IN REGION 1 THE SAME APPROXIMATION IS
044198,000171:                                                                                                  #        MADE, BUT AT AN ARTIFICIALLY-CREATED OR DUMMY OPERATING POINT,
044199,000172:                                                                                                  #        DEFINED BY.. OGAERROR = INTERSECTION OF CONTROL PARABOLA AND
044200,000173:                                                                                                  #        OGAERROR AXIS, OGARATE = +-LMCRATE WHERE SIGN IS OPPOSITE THAT OF
044201,000174:                                                                                                  #        REAL OPERATING POINT RATE.  WHEN THE OPERATING POINT HAS PASSED
044202,000175:                                                                                                  #        FROM REGION 1 TO REGION 6', THE DUMMY POINT IS NO LONGER REQUIRED,
044203,000176:                                                                                                  #        AND THE SOLUTION REVERTS TO THAT OF A REGULAR REGION 6' POINT.
044204,000177: 
044205,000178:                                                                                                  #  EQUATION FOR SWITCHING PARABOLA (SEE FIGURE ABOVE)....
044206,000179:                                                                                                  #                                    2
044207,000180:                                                                                                  #        SOGAERROR = (DB - (SOGARATE) (1/CONACC)/2) SGN(SOGARATE)
044208,000181: 
044209,000182:                                                                                                  #  EQUATION FOR SWITCHING STRAIGHT LINE SEGMENT....
044210,000183: 
044211,000184:                                                                                                  #        SOGARATE = -(-SLOPE)(SOGAERROR) - SGN(SOGARATE) INTERCEP
044212,000185: 
044213,000186:                                                                                                  #                WHERE  INTERCEP = DB(-SLOPE) - LMCRATE

044217,000190: 
044218,000191:                                                                                                  #        DOGADOT = NUM/DEN, WHERE
044219,000192:                                                                                                  #                                       2
