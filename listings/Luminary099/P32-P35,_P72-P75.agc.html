Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

026638,000043:                                                                                                  #  PURPOSE
026639,000044: 
026640,000045:                                                                                                  #        (1)     TO CALCULATE PARAMETERS ASSOCIATED WITH THE FOLLOWING
026641,000046:                                                                                                  #                CONCENTRIC FLIGHT PLAN MANEUVERS - THE CO-ELLIPTIC SEQUENCE
026642,000047:                                                                                                  #                INITIATION (CSI) MANEUVER AND THE CONSTANT DELTA ALTITUDE
026643,000048:                                                                                                  #                (CDH) MANEUVER.
026644,000049: 
026645,000050:                                                                                                  #        (2)     TO CALCULATE THESE PARAMETERS BASED UPON MANEUVER DATA
026646,000051:                                                                                                  #                APPROVED AND KEYED INTO THE DSKY BY THE ASTRONAUT.
026647,000052: 
026648,000053:                                                                                                  #        (3)     TO DISPLAY TO THE ASTRONAUT AND THE GROUND DEPENDENT VARIABLES
026649,000054:                                                                                                  #                ASSOCIATED WITH THE CONCENTRIC FLIGNT PLAN MANEUVERS FOR
026650,000055:                                                                                                  #                APPROVAL BY THE ASTRONAUT/GROUND.
026651,000056: 
026652,000057:                                                                                                  #        (4)     TO STORE THE CSI TARGET PARAMETERS FOR USE BY THE DESIRED
026653,000058:                                                                                                  #                THRUSTING PROGRAM.
026654,000059: 
026655,000060:                                                                                                  #  ASSUMPTIONS
026656,000061: 
026657,000062:                                                                                                  #        (1)     AT A SELECTED TPI TIME THE LINE OF SIGHT BETWEEN THE ACTIVE
026658,000063:                                                                                                  #                AND PASSIVE VEHICLES IS SELECTED TO BE A PRESCRIBED ANGLE (E)
026659,000064:                                                                                                  #                FROM THE HORIZONTAL PLANE DEFINED BY THE ACTIVE VEHICLE
026660,000065:                                                                                                  #                POSITION.
026661,000066: 
026662,000067:                                                                                                  #        (2)     THE TIME BETWEEN CSI IGNITION AND CDH IGNITION MUST BE
026663,000068:                                                                                                  #                COMPUTED TO BE GREATER THAN 10 MINUTES FOR SUCCESSFUL
026664,000069:                                                                                                  #                COMPLETION OF THE PROGRAM.
026665,000070: 
026666,000071:                                                                                                  #        (3)     THE TIME BETWEEN CDH IGNITION AND TPI IGNITION MUST BE
026667,000072:                                                                                                  #                COMPUTED TO BE GREATER THAN 10 MINUTES FOR SUCCESSFUL
026668,000073:                                                                                                  #                COMPLETION OF THE PROGRAM.
026669,000074: 
026670,000075:                                                                                                  #        (4)     CDH DELTA V IS SELECTED TO MINIMIZE THE VARIATION OF THE
026671,000076:                                                                                                  #                ALTITUDE DIFFERENCE BETWEEN THE ORBITS.
026672,000077: 
026673,000078:                                                                                                  #        (5)     CSI BURN IS DEFINED SUCH THAT THE IMPULSIVE DELTA V IS IN THE
026674,000079:                                                                                                  #                HORIZONTAL PLANE DEFINED BY THE ACTIVE VEHICLE POSITION AT CSI
026675,000080:                                                                                                  #                IGNITION.
026676,000081: 
026677,000082:                                                                                                  #        (6)     THE PERICENTER ALTITUDE OF THE ORBIT FOLLOWING CSI AND CDH
026678,000083:                                                                                                  #                MUST BE GREATER THAN 35,000 FT (LUNAR ORBIT) OR 85 NM (EARTH
026679,000084:                                                                                                  #                ORBIT) FOR SUCCESSFUL COMPLETION OF THIS PROGRAM.
026680,000085: 
026681,000086:                                                                                                  #        (7)     THE CSI AND CDH MANEUVERS ARE ORIGINALLY ASSUMED TO BE
026682,000087:                                                                                                  #                PARALLEL TO THE PLANE OF THE CSM ORBIT.  HOWEVER CREW

026696,000101:                                                                                                  #                OR CSM STATE VECTORS FOR THIS PROGRAM.  IF RADAR USE IS
026697,000102:                                                                                                  #                DESIRED THE RADAR WAS TURNED ON AND LOCKED ON THE CSM BY
026698,000103:                                                                                                  #                PREVIOUS SELECTION OF P20.  RADAR SIGHTING MARKS WILL BE MADE

026701,000106:                                                                                                  #                MARK COUNTER IS ZEROED BY THE SELECTION OF P20 AND AFTER EACH
026702,000107:                                                                                                  #                THRUSTING MANEUVER.
026703,000108: 
026704,000109:                                                                                                  #        (11)    THE ISS NEED NOT BE ON TO COMPLETE THIS PROGRAM.
026705,000110: 
026706,000111:                                                                                                  #        (12)    THE OPERATION OF THE PROGRAM UTILIZES THE FOLLOWING FLAGS -
026707,000112: 
026708,000113:                                                                                                  #                        ACTIVE VEHICLE FLAG - DESIGNATES THE VEHICLE WHICH IS
026709,000114:                                                                                                  #                        DOING RENDEZVOUS THRUSTING MANEUVERS TO THE PROGRAM WHICH
026710,000115:                                                                                                  #                        CALCULATES THE MANEUVER PARAMETERS.  SET AT THE START OF
026711,000116:                                                                                                  #                        EACH RENDEZVOUS PRE-THRUSTING PROGRAM.
026712,000117: 
026713,000118:                                                                                                  #                        FINAL FLAG - SELECTS FINAL PROGRAM DISPLAYS AFTER CREW HAS
026714,000119:                                                                                                  #                        COMPLETED THE FINAL MANEUVER COMPUTATION AND DISPLAY
026715,000120:                                                                                                  #                        CYCLE.

026718,000123:                                                                                                  #                        STEERING REQUIRED FOR EXECUTION OF THIS MANEUVER BY THE
026719,000124:                                                                                                  #                        THRUSTING PROGRAM SELECTED AFTER COMPLETION OF THIS
026720,000125:                                                                                                  #                        PROGRAM.
026721,000126:

026723,000128:                                                                                                  #                A THRUSTING MANEUVER.
026724,000129: 
026725,000130:                                                                                                  #        (14)    THIS PROGRAM IS SELECTED BY THE ASTRONAUT BY DSKY ENTRY -
