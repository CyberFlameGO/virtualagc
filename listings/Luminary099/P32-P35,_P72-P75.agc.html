Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	16:27 JULY 14,1969
Note that the date is the date of the printout, not the date of the program revision.

026648,000045:                                                                                                  #  PURPOSE
026649,000046: 
026650,000047:                                                                                                  #        (1)     TO CALCULATE PARAMETERS ASSOCIATED WITH THE FOLLOWING
026651,000048:                                                                                                  #                CONCENTRIC FLIGHT PLAN MANEUVERS - THE CO-ELLIPTIC SEQUENCE
026652,000049:                                                                                                  #                INITIATION (CSI) MANEUVER AND THE CONSTANT DELTA ALTITUDE
026653,000050:                                                                                                  #                (CDH) MANEUVER.
026654,000051: 
026655,000052:                                                                                                  #        (2)     TO CALCULATE THESE PARAMETERS BASED UPON MANEUVER DATA
026656,000053:                                                                                                  #                APPROVED AND KEYED INTO THE DSKY BY THE ASTRONAUT.
026657,000054: 
026658,000055:                                                                                                  #        (3)     TO DISPLAY TO THE ASTRONAUT AND THE GROUND DEPENDENT VARIABLES
026659,000056:                                                                                                  #                ASSOCIATED WITH THE CONCENTRIC FLIGNT PLAN MANEUVERS FOR
026660,000057:                                                                                                  #                APPROVAL BY THE ASTRONAUT/GROUND.
026661,000058: 
026662,000059:                                                                                                  #        (4)     TO STORE THE CSI TARGET PARAMETERS FOR USE BY THE DESIRED
026663,000060:                                                                                                  #                THRUSTING PROGRAM.
026664,000061: 
026665,000062:                                                                                                  #  ASSUMPTIONS
026666,000063: 
026667,000064:                                                                                                  #        (1)     AT A SELECTED TPI TIME THE LINE OF SIGHT BETWEEN THE ACTIVE
026668,000065:                                                                                                  #                AND PASSIVE VEHICLES IS SELECTED TO BE A PRESCRIBED ANGLE (E)
026669,000066:                                                                                                  #                FROM THE HORIZONTAL PLANE DEFINED BY THE ACTIVE VEHICLE
026670,000067:                                                                                                  #                POSITION.
026671,000068: 
026672,000069:                                                                                                  #        (2)     THE TIME BETWEEN CSI IGNITION AND CDH IGNITION MUST BE
026673,000070:                                                                                                  #                COMPUTED TO BE GREATER THAN 10 MINUTES FOR SUCCESSFUL
026674,000071:                                                                                                  #                COMPLETION OF THE PROGRAM.
026675,000072: 
026676,000073:                                                                                                  #        (3)     THE TIME BETWEEN CDH IGNITION AND TPI IGNITION MUST BE
026677,000074:                                                                                                  #                COMPUTED TO BE GREATER THAN 10 MINUTES FOR SUCCESSFUL
026678,000075:                                                                                                  #                COMPLETION OF THE PROGRAM.
026679,000076: 
026680,000077:                                                                                                  #        (4)     CDH DELTA V IS SELECTED TO MINIMIZE THE VARIATION OF THE
026681,000078:                                                                                                  #                ALTITUDE DIFFERENCE BETWEEN THE ORBITS.
026682,000079: 
026683,000080:                                                                                                  #        (5)     CSI BURN IS DEFINED SUCH THAT THE IMPULSIVE DELTA V IS IN THE
026684,000081:                                                                                                  #                HORIZONTAL PLANE DEFINED BY THE ACTIVE VEHICLE POSITION AT CSI
026685,000082:                                                                                                  #                IGNITION.
026686,000083: 
026687,000084:                                                                                                  #        (6)     THE PERICENTER ALTITUDE OF THE ORBIT FOLLOWING CSI AND CDH
026688,000085:                                                                                                  #                MUST BE GREATER THAN 35,000 FT (LUNAR ORBIT) OR 85 NM (EARTH
026689,000086:                                                                                                  #                ORBIT) FOR SUCCESSFUL COMPLETION OF THIS PROGRAM.
026690,000087: 
026691,000088:                                                                                                  #        (7)     THE CSI AND CDH MANEUVERS ARE ORIGINALLY ASSUMED TO BE
026692,000089:                                                                                                  #                PARALLEL TO THE PLANE OF THE CSM ORBIT.  HOWEVER CREW

026711,000108:                                                                                                  #                MARK COUNTER IS ZEROED BY THE SELECTION OF P20 AND AFTER EACH
026712,000109:                                                                                                  #                THRUSTING MANEUVER.

026717,000114:

026721,000118:                                                                                                  #                        EACH RENDEZVOUS PRE-THRUSTING PROGRAM.

026723,000120:                                                                                                  #                        FINAL FLAG - SELECTS FINAL PROGRAM DISPLAYS AFTER CREW HAS
026724,000121:                                                                                                  #                        COMPLETED THE FINAL MANEUVER COMPUTATION AND DISPLAY
026725,000122:                                                                                                  #                        CYCLE.

026728,000125:                                                                                                  #                        STEERING REQUIRED FOR EXECUTION OF THIS MANEUVER BY THE
026729,000126:                                                                                                  #                        THRUSTING PROGRAM SELECTED AFTER COMPLETION OF THIS
026730,000127:                                                                                                  #                        PROGRAM.

026733,000130:                                                                                                  #                A THRUSTING MANEUVER.

026735,000132:                                                                                                  #        (14)    THIS PROGRAM IS SELECTED BY THE ASTRONAUT BY DSKY ENTRY -
