The scanned page images are available at the Virtual AGC Project website, as well as higher-quality (but much larger) images at the Virtual AGC Project's collection in the Internet Archive. 010971,000023:                                                                                                  #  RESTART TABLES
010972,000024: 
010973,000025:                                                                                                  #  ------------------
010974,000026: 
010975,000027:                                                                                                  #  THERE ARE TWO FORMS OF RESTART TABLES FOR EACH GROUP.  THEY ARE KNOWN AS THE EVEN RESTART TABLES AND THE ODD
010976,000028:                                                                                                  #  RESTART TABLES.  THE ODD TABLES HAVE ONLY ONE ENTRY OF THREE LOCATIONS WHILE THE EVEN TABLES HAVE TWO ENTRIES
010977,000029:                                                                                                  #  EACH USING THREE LOCATIONS. THE INFORMATION AS TO WHETHER IT IS A JOB, WAITLIST, OR A LONGCALL IS GIVEN BY THE
010978,000030:                                                                                                  #  WAY THINGS ARE PUT INTO THE TABLES.
010979,000031:                                                                                                  #       A JOB HAS ITS PRIORITY STORED IN PRDTTAB OF THE CORRECT PHASE SPOT - A POSITIVE PRIORITY INDICATES A
010980,000032:                                                                                                  #  FINDVAC JOB, A NEGATIVE PRIORITY A NOVAC.  THE 2CADR OF THE JOB IS STORED IN THE CADRTAB.
010981,000033:                                                                                                  #  FOR EXAMPLE,
010982,000034: 
010983,000035:                                                                                                  #                                          5.7SPOT  OCT    23000
010984,000036:                                                                                                  #                                                   2CADR  SOMEJOB
010985,000037: 
010986,000038:                                                                                                  #  A RESTART OF GROUP 5 WITH PHASE SEVEN WOULD THEN CAUSE SOMEJOB TO BE RESTARTED AS A FINDVAC WITH PRIORITY 23.
010987,000039: 
010988,000040:                                                                                                  #                                          5.5SPOT  OCT    -23000
010989,000041:                                                                                                  #                                                   2CADR  ANYJOB
010990,000042: 
010991,000043:                                                                                                  #  HERE A RESTART OF GROUP 5 WITH PHASE 7 WOULD CAUSE ANYJOB TO BE RESTARTED AS A NOVAC WITH PRIORITY 23.
010992,000044:                                                                                                  #  A LONGCALL HAS ITS GENADR OF ITS 2CADR STORED NEGATIVELY AND ITS BBCON STORED POSITIVELY.  IN ITS PRDTTAB IS
010993,000045:                                                                                                  #  PLACED THE LOCATION OF A DP REGISTER THAT CONTAINS THE DELTA TIME THAT LONGCALL HAD BEEN ORIGINALLY STARTED
010994,000046:                                                                                                  #  WITH.  EXAMPLE,
010995,000047: 
010996,000048:                                                                                                  #                                          3.6SPOT  GENADR DELTAT
010997,000049:                                                                                                  #                                                  -GENADR LONGTASK
010998,000050:                                                                                                  #                                                   BBCON  LONGTASK
010999,000051: 
011000,000052:                                                                                                  #                                                   OCT    31000
011001,000053:                                                                                                  #                                                   2CADR  JOBAGAIN
011002,000054: 
011003,000055:                                                                                                  #  THIS WOULD START UP LONGTASK AT THE APPROPRIATE TIME, OR IMMEDIATELY IF THE TIME HAD ALREADY PASSED. IT SHOULD
011004,000056:                                                                                                  #  BE NOTED THAT IF DELTAT IS IN A SWITCHED E BANK, THIS INFORMATOIN SHOULD BE IN THE BBCON OFTHE 2CADR OF THE
011005,000057:                                                                                                  #  TASK.  FROM ABOVE, WE SEE THAT THE SECOND PART OF THIS PHASE WOULD BE STARTED AS A JOB WITH A PRIORITY OF 31.
011006,000058: 
011007,000059:                                                                                                  #  WAITLIST CALLS ARE IDENTIFIED BY THE FACT THAT THEIR 2CADR IS STORED NEGATIVELY. IF PRDTTAB OF THE PHASE SPOT
011008,000060:                                                                                                  #  IS POSITIVE, THEN IT CONTAINS THE DELTA TIME, IF PRDTTAB IS NEGATIVE THEN IT IS THE -GENADR OF AN ERASABLE
011009,000061:                                                                                                  #  LOCATION CONTAINING THE DELTA TIME, THAT IS, THE TIME IS STORED INDIRECTLY.  IT SHOULD BE NOTED AS ABOVE, THAT
011010,000062:                                                                                                  #  IF THE TIME IS STORED INDIRECTLY, THE BBCON MUST CONTAIN THE NECESSARY E BANK INFORMATION IF APPLICABLE.  WITH
011011,000063:                                                                                                  #  WAITLIST WE HAVE ONE FURTHER OPTION, IF -0 IS STORED IN PRDTTAB, IT WILL CAUSE AN IMMEDIATE RESTART OF THE
011012,000064:                                                                                                  #  TASK.  EXAMPLES,
011013,000065: 
011014,000066:                                                                                                  #                                                   OCT    77777           THIS WILL CAUSE AN IMMEDIATE RESTART
011015,000067:                                                                                                  #                                                  -2CADR  ATASK           OF THE TASK :ATASK:
011016,000068:                                                                                                  #     
011017,000069:                                                                                                  #                                                   DEC    200             IF THE TIME OF THE 2 SECONDS SINCE DUMMY
011018,000070:                                                                                                  #                                                  -2CADR  DUMMY           WAS PUT ON WAITLIST IS UP, IT WILL BEGIN
011019,000071:                                                                                                  #                                                                          IN 10 MS, OTHERWISE IT WILL BEGIN WHEN
011020,000072:                                                                                                  #                                                                          IT NORMALLY WOULD HAVE BEGUN.
