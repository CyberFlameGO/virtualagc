Source Code

These source-code files were transcribed from a printout in Don Eyles's personal collection, scanned by archive.org, and financially sponsored by Linden Sims. A team of volunteers performed the transcription and proof-reading. The scanned page images are available at the Virtual AGC Project website, as well as higher-quality (but much larger) images at the Virtual AGC Project's collection in the Internet Archive. Report any problems by creating "issues" at the Virtual AGC Project's GitHub Repository.

Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 56 OF AGC PROGRAM ZERLINA BY ZOROASTER
	9:12 OCT. 21,1970
Note that the date is the date of the printout, not the date of the program revision.

031134,000032:                                                                                                  #  1. IF AVERAGE G IS OFF:
031135,000033:                                                                                                  #           FLASH DISPLAY V04N06. R2 INDICATES WHICH SHIP'S STATE VECTOR IS
031136,000034:                                                                                                  #            TO BE UPDATED. INITIAL CHOICE IS THIS SHIP (R2=1). ASTRONAUT
031137,000035:                                                                                                  #            CAN CHANGE TO OTHER SHIP BY V22EXE, WHERE X NOT EQ 1.
031138,000036:                                                                                                  #           SELECTED STATE VECTOR UPDATED BY THISPREC (OTHPREC).
031139,000037:                                                                                                  #           CALLS SR30.1 (WHICH CALLS TFFCONMU + TFFRP/RA) TO CALCULATE
031140,000038:                                                                                                  #            RPER (PERIGEE RADIUS), RAPO (APOGEE RADIUS), HPER (PERIGEE
031141,000039:                                                                                                  #            HEIGHT ABOVE LAUNCH PAD OR LUNAR LANDING SITE), HAPO (APOGEE
031142,000040:                                                                                                  #            HEIGHT AS ABOVE), TPER (TIME TO PERIGEE), TFF (TIME TO
031143,000041:                                                                                                  #            INTERSECT 300 KFT ABOVE PAD OR 35KFT ABOVE LANDING SITE).
031144,000042:                                                                                                  #           FLASH MONITOR V16N44 (HAPO, HPER, TFF).TFF IS -59M59S IF IT WAS
031145,000043:                                                                                                  #            NOT COMPUTABLE, OTHERWISE IT INCREMENTS ONCE PER SECOND.
031146,000044:                                                                                                  #            ASTRONAUT HAS OPTION TO MONITOR TPER BY KEYING IN N 32 E.
031147,000045:                                                                                                  #            DISPLAY IS IN HMS, IS NEGATIVE (AS WAS TFF), AND INCREMENTS
031148,000046:                                                                                                  #            ONCE PER SECOND ONLY IF TFF DISPLAY WAS -59M59S.
031149,000047: 
031150,000048:                                                                                                  #  2. IF AVERAGE G IS ON:
031151,000049:                                                                                                  #           CALLS SR30.1 APPROX EVERY TWO SECS.  STATE VECTOR IS ALWAYS
031152,000050:                                                                                                  #            FOR THIS VEHICLE. V82 DOES NOT DISTURB STATE VECTOR.  RESULTS
031153,000051:                                                                                                  #            OF SR30.1 ARE RAPO, RPER, HAPO, HPER, TPER, TFF.
031154,000052:                                                                                                  #           FLASH MONITOR V16N44 (HAPO, HPER, TFF).
031155,000053:                                                                                                  #  ADDENDUM: HAPO AND HPER SHOULD BE CHANGED TO READ HAPOX AND HPERX IN THE
031156,000054:                                                                                                  #            ABOVE REMARKS.
