Source Code

These source-code files are part of a reconstructed copy of Artemis 071, the first release of the Apollo 15 through 17 Command Module (CM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Artemis 072 previously transcribed from a digitized copy of that program. The code was then updated by undoing changes between the two versions. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021154-. Note that page numbers in the reconstructed code match those on the Artemis 072 printout, although the changed code would likely have changed page numbers for a real Artemis 071 listing.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

013925,000026: 
013926,000027:                                                                                                  #        THE KEYBOARD AND DISPLAY SYSTEM PROGRAM OPERATES UNDER EXECUTIVE
013927,000028:                                                                                                  #        CONTROL AND PROCESSES INFORMATION EXCHANGED BETWEEN THE AGC AND THE
013928,000029:                                                                                                  #        COMPUTER OPERATOR.  THE INPUTS TO THE PROGRAM ARE FROM THE KEYBOARD,

013930,000031: 
013931,000032:                                                                                                  #        THE LANGUAGE OF COMMUNICATION WITH THE PROGRAM IS A PAIR OF WORDS
013932,000033:                                                                                                  #        KNOWN AS VERB AND NOUN.  EACH OF THESE IS REPRESENTED BY A 2 CHARACTER
013933,000034:                                                                                                  #        DECIMAL NUMBER.  THE VERB CODE INDICATES WHAT ACTION IS TO BE TAKEN, THE
013934,000035:                                                                                                  #        NOUN CODE INDICATES TO WHAT THIS ACTION IS APPLIED.  NOUNS USUALLY
013935,000036:                                                                                                  #        REFER TO A GROUP OF ERASABLE REGISTERS.
013936,000037: 
013937,000038:                                                                                                  #        VERBS ARE GROUPED INTO DISPLAYS, LOADS, MONITORS (DISPLAYS THAT ARE
013938,000039:                                                                                                  #        UPDATED ONCE PER SECOND), SPECIAL FUNCTIONS, AND EXTENDED VERBS (THESE
013939,000040:                                                                                                  #        ARE OUTSIDE OF THE DOMAIN OF PINBALL AND CAN BE FOUND UNDER LOG SECTION
013940,000041:                                                                                                  #        :EXTENDED VERBS:).

013942,000043:                                                                                                  #        A LIST OF VERBS AND NOUNS IS GIVEN IN LOG SECTION :ASSEMBLY AND

013949,000050:                                                                                                  #        INTERRUPT KEYRUPT1 (KEYRUPT2) AND PLACES THE 5 BIT KEY CODE INTO
013950,000051:                                                                                                  #        CHANNEL 15 (CHANNEL 16).  KEYRUPT1 (KEYRUPT2) PLACES THE KEY
013951,000052:                                                                                                  #        CODE INTO MPAC, ENTERS AN EXECUTIVE REQUEST FOR THE KEYBOARD AND DISPLAY
013952,000053:                                                                                                  #        PROGRAM (AT :CHARIN:), AND EXECUTES A RESUME.
013953,000054: 
013954,000055:                                                                                                  #        UPLINK:
013955,000056:                                                                                                  #        EACH WORD RECEIVED BY THE UPLINK ACTIVATES INTERRUPT UPRUPT WHICH
013956,000057:                                                                                                  #        PLACES THE 5 BIT KEY CODE INTO MPAC, ENTERS AN EXECUTIVE REQUEST FOR THE
013957,000058:                                                                                                  #        KEYBOARD AND DISPLAY PROGRAM (AT :CHARIN:) AND EXECUTES A RESUME.

013963,000064:                                                                                                  #        SYMBOL TABLE FOR PAGE NUMBERS).
