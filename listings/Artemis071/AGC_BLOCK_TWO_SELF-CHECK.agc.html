Source Code

These source-code files are part of a reconstructed copy of Artemis 071, the first release of the Apollo 15 through 17 Command Module (CM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Artemis 072 previously transcribed from a digitized copy of that program. The code was then updated by undoing changes between the two versions. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021154-. Note that page numbers in the reconstructed code match those on the Artemis 072 printout, although the changed code would likely have changed page numbers for a real Artemis 071 listing.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

059549,000026:                                                                                                  #  FUNCTIONAL DESCRIPTION
059550,000027: 
059551,000028:                                                                                                  #  PROGRAM HAS TWO MAIN PARTS. THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
059552,000029:                                                                                                  #  PART OF THE BACK-UP IDLE LOOP. THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
059553,000030:                                                                                                  #  STARTING VERB.
059554,000031:                                                                                                  #  THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
059555,000032:                                                                                                  #  THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK, ONE AT A TIME.
059556,000033:                                                                                                  #  IN ALL THERE ARE  7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK. MORE DETAIL DESCRIPTION MAY BE
059557,000034:                                                                                                  #  FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
059558,000035:                                                                                                  #  THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27). BELOW IS
059559,000036:                                                                                                  #  A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
059560,000037:                                                                                                  #  OCTAL, TO LOAD INTO SMODE.
059561,000038:                                                                                                  #        +-4             ERASABLE MEMORY
059562,000039:                                                                                                  #        +-5             FIXED MEMORY
059563,000040:                                                                                                  #        +-1,2,3,6,7,10  EVERYTHING IN OPTIONS 4 AND 5.
059564,000041:                                                                                                  #        -0              SAME AS +-10 UNTIL AN ERROR IS DETECTED.
059565,000042:                                                                                                  #        +0              NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
059566,000043: 
059567,000044:                                                                                                  #  WARNINGS
059568,000045: 
059569,000046:                                                                                                  #        USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
059570,000047:                                                                                                  #        SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
059571,000048: 
059572,000049:                                                                                                  #  CALLING SEQUENCE
059573,000050: 
059574,000051:                                                                                                  #        TO CALL SELF-CHECK KEY IN
059575,000052:                                                                                                  #                V 21 N 27 E     OPTION NUMBER E
059576,000053:                                                                                                  #        TO CALL SHOW-BANKSUM KEY IN
059577,000054:                                                                                                  #                V 91 E          DISPLAYS FIRST BANK
059578,000055:                                                                                                  #                V 33 E          PROCEED, DISPLAYS NEXT BANK
059579,000056: 
059580,000057: 
059581,000058:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
059582,000059: 
059583,000060:                                                                                                  #  SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED. IF SO + OPTION NUMBERS PUT
059584,000061:                                                                                                  #  COMPUTER INTO BACKUP IDLE LOOP, - OPTION NUMBERS RESTART THE OPTION.
059585,000062:                                                                                                  #  THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
059586,000063:                                                                                                  #  SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E). THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP
059587,000064: 
059588,000065: 
059589,000066:                                                                                                  #  OUTPUT

059591,000068:                                                                                                  #  SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
059592,000069:                                                                                                  #  TURNS ON THE ALARM LIGHT. THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E. FOR FURTHER
059593,000070:                                                                                                  #  INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
059594,000071:                                                                                                  #  IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
059595,000072:                                                                                                  #  INITIATED FRESH START (SLAP1).
059596,000073:                                                                                                  #  SHOW-BANKSUM STARTING WITH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
059597,000074:                                                                                                  #  THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
059598,000075: 
059599,000076: 
059600,000077:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
059601,000078: 
059602,000079:                                                                                                  #        ACCOMPLISHED BY FRESH START
059603,000080:                                                                                                  #                SMODE SET TO +0
059604,000081: 
059605,000082:                                                                                                  #  DEBRIS
059606,000083: 
059607,000084:                                                                                                  #        ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
059608,000085:                                                                                                  #        EXCEPTION IS A RESTART. RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
059609,000086:                                                                                                  #        E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
