Source Code

These source-code files are part of a reconstructed copy of Artemis 071, the first release of the Apollo 15 through 17 Command Module (CM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Artemis 072 previously transcribed from a digitized copy of that program. The code was then updated by undoing changes between the two versions. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021154-. Note that page numbers in the reconstructed code match those on the Artemis 072 printout, although the changed code would likely have changed page numbers for a real Artemis 071 listing.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

063386,000039:                                                                                                  #  UPFLAG AND DOWNFLAG ARE ENTIRELY GENERAL FLAG SETTING AND CLEARING SUBROUTINES.  USING THEM, WHETHER OR
063387,000040:                                                                                                  #  NOT IN INTERRUPT, ONE MAY SET OR CLEAR ANY SINGLE, NAMED BIT IN ANY ERASABLE REGISTER, SUBJECT OF COURSE TO
063388,000041:                                                                                                  #  EBANK SETTING.  A "NAMED" BIT, AS THE WORD IS USED HERE, IS ANY BIT WITH A NAME FORMALLY ASSIGNED BY THE YUL
063389,000042:                                                                                                  #  ASSEMBLER.
063390,000043: 
063391,000044:                                                                                                  #  AT PRESENT THE ONLY NAMED BITS ARE THOSE IN THE FLAGWORDS.  ASSEMBLER CHANGES WILL MAKE IT POSSIBLE TO
063392,000045:                                                                                                  #  NAME ANY BIT IN ERASABLE MEMORY.
063393,000046: 
063394,000047:                                                                                                  #  CALLING SEQUENCES ARE AS FOLLOWS:-
063395,000048:                                                                                                  #                TC      UPFLAG                  TC      DOWNFLAG
063396,000049:                                                                                                  #                ADRES   NAME OF FLAG            ADRES   NAME OF FLAG
063397,000050: 
063398,000051:                                                                                                  #  RETURN IS TO THE LOCATION FOLLOWING THE "ADRES" ABOUT .58 MS AFTER THE "TC".
063399,000052:                                                                                                  #  UPON RETURN A CONTAINS THE CURRENT FLAGWRD SETTING.

063502,000155: 
063503,000156:                                                                                                  #  WRITTEN BY D. EYLES
063504,000157:                                                                                                  #  MOD 1 BY KERNAN                                               UTILITYM REV 17 11/18/67
063505,000158: 
063506,000159:                                                                                                  #  MOD 2 BY SCHULENBERG  (REMOVE RELINT)   SKIPPER REV 4 2/28/68
063507,000160: 
063508,000161:                                                                                                  #  THIS ROUTINE IS USEFULL FOR TRANSFERING N CONSECUTIVE ERASABLE OR FIXED QUANTITIES TO SOME OTHER N
063509,000162:                                                                                                  #  CONSECUTIVE ERASABLE LOCATIONS.  IF BOTH BLOCKS OF DATA ARE IN SWITCHABLE EBANKS, THEY MUST BE IN THE SAME ONE.
063510,000163: 
063511,000164:                                                                                                  #  GENTRAN IS CALLABLE IN A JOB AS WELL AS A RUPT.  THE CALLING SEQUENCE IS:
063512,000165: 
063513,000166:                                                                                                  #        I       CA      N-1             # OF QUANTITIES MINUS ONE.
063514,000167:                                                                                                  #        I +1    TC      GENTRAN         IN FIXED-FIXED.
063515,000168:                                                                                                  #        I +2    ADRES   L               STARTING ADRES OF DATA TO BE MOVED.
063516,000169:                                                                                                  #        I +3    ADRES   M               STARTING ADRES OF DUPLICATION BLOCK.
063517,000170:                                                                                                  #        I +4                            RETURNS HERE.
063518,000171: 
063519,000172:                                                                                                  #  GENTRAN TAKES 25 MCT'S (300 MICROSECONDS) PER ITEM + 5 MCT'S (60 MICS) FOR ENTERING AND EXITING.
