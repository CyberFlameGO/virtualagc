Source Code

These source-code files are part of a reconstructed copy of Artemis 071, the first release of the Apollo 15 through 17 Command Module (CM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Artemis 072 previously transcribed from a digitized copy of that program. The code was then updated by undoing changes between the two versions. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021154-. Note that page numbers in the reconstructed code match those on the Artemis 072 printout, although the changed code would likely have changed page numbers for a real Artemis 071 listing.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

042475,000021:                                                                                                  #  LOG SECTION...TVCDAPS                         SUBROUTINE...DAPCSM
042476,000022:                                                                                                  #  MODIFIED BY SCHLUNDT                          21 OCTOBER 1968
042477,000023:                                                                                                  #  FUNCTIONAL DESCRIPTION....
042478,000024: 
042479,000025:                                                                                                  #        SELF-PERPETUATING T5 TASKS WHICH GENERATE THE COMMAND SIGNALS
042480,000026:                                                                                                  #        FOR THE PITCH AND YAW SPS GIMBAL ACTUATORS DURING TVC (SPS) BURNS,
042481,000027:                                                                                                  #        IN RESPONSE TO BODY-AXIS RATE COMMANDS FROM CROSS-PRODUCT STEERING
042482,000028:                                                                                                  #        (S40.8).  IF NO STEERING (IMPULSIVE BURNS) MAINTAINS ATTITUDE-HOLD
042483,000029:                                                                                                  #        ABOUT THE REFERENCE (INITIAL) DIRECTIONS (ZERO RATE COMMANDS).
042484,000030: 
042485,000031:                                                                                                  #        THE PITCH AND YAW LOOPS ARE SEPARATE, BUT STRUCTURED IDENTICALLY.
042486,000032:                                                                                                  #        EACH ATTITUDE-RATE LOOP INCLUDES GIMBAL ANGLE RATE DERIVATION,
042487,000033:                                                                                                  #        GIMBAL/BODY AXIS TRANSFORMATION, BODY-AXIS ATTITUDE ERROR
042488,000034:                                                                                                  #        INTEGRATION WITH ERROR LIMITING, THE GENERALIZED 6TH-ORDER FILTER
042489,000035:                                                                                                  #        FOR CSM OR CSM/LM OPERATION, A FILTER OUTPUT LIMITER,
042490,000036:                                                                                                  #        CG-OFFSET TRACKER FILTER, AND THE CG-TRACKER MINOR LOOP.
042491,000037: 
042492,000038:                                                                                                  #        THE DAPS ARE CYCLIC, CALLING EACH OTHER AT 1/2 THE DAP SAMPLE
042493,000039:                                                                                                  #        TIME, AS DETERMINED BY T5TVCDT. THE ACTUATOR COMMANDS ARE
042494,000040:                                                                                                  #        REGENERATED AS ANALOG VOLTAGES BY THE OPTICS ERROR COUNTERS, WHICH
042495,000041:                                                                                                  #        TRANSMIT THE SIGNAL TO THE ACTUATOR SERVOS WHEN THERE IS PROPER CDU
042496,000042:                                                                                                  #        MODING.
042497,000043: 
042498,000044:                                                                                                  #  CALLING SEQUENCE.... (TYPICALLY)
042499,000045: 
042500,000046:                                                                                                  #        T5 CALL OF TVCDAPON (TVCINITIALIZE) BY DOTVCON (P40)
042501,000047:                                                                                                  #        T5 CALL OF DAPINIT (TVCDAPS) BY TVCINIT4 (TVCINITIALIZE)
042502,000048:                                                                                                  #        T5 CALL OF PITCHDAP BY DAPINIT
042503,000049:                                                                                                  #        T5 CALL OF YAWDAP BY PITCHDAP
042504,000050:                                                                                                  #        T5 CALL OF PITCHDAP BY YAWDAP
042505,000051:                                                                                                  #                ETC.
042506,000052:                                                                                                  #        (AUTOMATIC SEQUENCING FROM TVCDAPON)
042507,000053: 
042508,000054:                                                                                                  #  NORMAL EXIT MODE....RESUME
042509,000055: 
042510,000056:                                                                                                  #  ALARM OR ABORT EXIT MODES....NONE
042511,000057: 
042512,000058:                                                                                                  #  SUBROUTINES CALLED....
042513,000059: 
042514,000060:                                                                                                  #        HACK FOR STROKE TEST (V68) WAVEFORM GENERATION

042516,000062:                                                                                                  #        DAPINIT FOR INITIAL CDUS FOR RATE MEASUREMENTS
042517,000063:                                                                                                  #        ERRORLIM, ACTLIM FOR INPUT (ATTITUDE-ERROR INTEGRATION) AND
042518,000064:                                                                                                  #                OUTPUT (ACTUATOR COMMAND) LIMITING, COMMON TO PITCH AND
042519,000065:                                                                                                  #                YAW DAPS
