Source Code

These source-code files are part of a reconstructed copy of Luminary 178, the final release of the Apollo 14 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Zerlina 56, Luminary 210, and Luminary 131, previously transcribed from digitized copies of those programs. The code was created by combining the three, using Zerlina 56 as a base, since it was based off of Luminary 183. The differences between the three were carefully merged using the Luminary memos as a guide, with all changes made after Luminary 178 being backed out. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021152N. Note that page numbers in the reconstructed code match those for the baseline log section mentioned in each file's changelog; the page numbers for a real Luminary 178 listing would be different.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

059666,000024:                                                                                                  #  MODIFICATION BY: LOWELL G HULL (A.C.ELECTRONICS)
059667,000025:                                                                                                  #  THESE PROGRAMS ENABLE THE LM DAP TO CONTROL THE THRUST TIMES OF THE REACTION CONTROL SYSTEM JETS BY USING TIME6.
059668,000026:                                                                                                  #  SINCE THE LM DAP MAINTAINS EXCLUSIVE CONTROL OVER TIME6 AND ITS INTERRUPTS, THE FOLLOWING CONVENTIONS HAVE BEEN
059669,000027:                                                                                                  #  ESTABLISHED AND MUST NOT BE TAMPERED WITH:
059670,000028:                                                                                                  #           1. NO NUMBER IS EVER PLACED INTO TIME6 EXCEPT BY LM DAP.
059671,000029:                                                                                                  #           2. NO PROGRAM OTHER THAN LM DAP ENABLES THE TIME6 COUNTER.
059672,000030:                                                                                                  #           3. TO USE TIME6, THE FOLLOWING SEQUENCE IS ALWAYS EMPLOYED:
059673,000031:                                                                                                  #                  A. A POSITIVE (NON-ZERO) NUMBER IS STORED IN TIME6.
059674,000032:                                                                                                  #                  B. THE TIME6 CLOCK IS ENABLED.
059675,000033:                                                                                                  #                  C. TIME6 IS INTERROGATED AND IS:
059676,000034:                                                                                                  #                         I. NEVER FOUND NEGATIVE(NON-ZERO) OR +0.
059677,000035:                                                                                                  #                        II. SOMETIMES FOUND POSITIVE(BETWEEN 1 AND 240D) INDICATING THAT IT IS ACTIVE.
059678,000036:                                                                                                  #                       III. SOMETIMES FOUND POSMAX INDICATING THAT IT IS INACTIVE AND NOT ENABLED.
059679,000037:                                                                                                  #                        IV. SOMETIMES FOUND NEGATIVE ZERO INDICATING THAT:
059680,000038:                                                                                                  #                                A. A T6RUPT IS ABOUT TO OCCUR AT THE NEXT DINC, OR
059681,000039:                                                                                                  #                                B. A T6RUPT IS WAITING IN THE PRIORITY CHAIN, OR
059682,000040:                                                                                                  #                                C. A T6RUPT IS IN PROCESS NOW.
059683,000041:                                                                                                  #           4) ALL PROGRAMS WHICH OPERATE IN EITHER INTERRUPT MODE OR WITH INTERRUPT INHIBITED MUST CALL T6JOBCHK
059684,000042:                                                                                                  #              EVERY 5 MILLISECONDS TO PROCESS A POSSIBLE WAITING T6RUPT BEFORE IT CAN BE HONORED BY THE HARDWARE.
059685,000043: 
059686,000044:                                                                                                  #          (5. PROGRAM JTLST, IN Q,R-AXES, HANDLES THE INPUT LIST.)
