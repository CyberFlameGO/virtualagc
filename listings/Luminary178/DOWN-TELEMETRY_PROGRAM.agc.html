Source Code

These source-code files are part of a reconstructed copy of Luminary 178, the final release of the Apollo 14 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Zerlina 56, Luminary 210, and Luminary 131, previously transcribed from digitized copies of those programs. The code was created by combining the three, using Zerlina 56 as a base, since it was based off of Luminary 183. The differences between the three were carefully merged using the Luminary memos as a guide, with all changes made after Luminary 178 being backed out. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021152N. Note that page numbers in the reconstructed code match those for the baseline log section mentioned in each file's changelog; the page numbers for a real Luminary 178 listing would be different.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

042570,000032:                                                                                                  #  LOG SECTION- DOWN-TELEMETRY PROGRAM
042571,000033:                                                                                                  #  FUNCTIONAL DESCRIPTION- THIS ROUTINE IS INITIATED BY TELEMETRY END
042572,000034:                                                                                                  #           PULSE FROM THE DOWNLINK TELEMETRY CONVERTER. THIS PULSE OCCURS
042573,000035:                                                                                                  #           AT 50 TIMES PER SEC(EVERY 20 MS) THEREFORE DODOWNTM IS
042574,000036:                                                                                                  #           EXECUTED AT THESE RATES. THIS ROUTINE SELECTS THE APPROPRIATE
042575,000037:                                                                                                  #           AGC DATA TO BE TRANSMITTED DOWNLINK AND LOADS IT INTO OUTPUT
042576,000038:                                                                                                  #           CHANNELS 34 AND 35. THE INFORMATION IS THEN GATED OUT FROM THE
042577,000039:                                                                                                  #           LGC IN SERIAL FASHION.
042578,000040:                                                                                                  #           THIS PROGRAM IS CODED FOR A 2 SECOND DOWNLIST. SINCE DOWNRUPTS
042579,000041:                                                                                                  #           OCCUR EVERY 20MS AND 2 AGC COMPUTER WORDS CAN BE PLACED IN
042580,000042:                                                                                                  #           CHANNELS 34 AND 35 DURING EACH DOWNRUPT THE PROGRAM IS CAPABLE
042581,000043:                                                                                                  #           OF SENDING 200 AGC WORDS EVERY 2 SECONDS.
042582,000044:                                                                                                  #  CALLING SEQUENCE- NONE
042583,000045:                                                                                                  #           PROGRAM IS ENTERED VIA TCF DODOWNTM WHICH IS EXECUTED AS A
042584,000046:                                                                                                  #           RESULT OF A DOWNRUPT. CONTROL IS RETURNED VIA TCF RESUME WHICH
042585,000047:                                                                                                  #           IN EFFECT IS A RESUME.
042586,000048:                                                                                                  #  SUBROUTINES CALLED- NONE
042587,000049:                                                                                                  #  NORMAL EXIT MODE- TCF RESUME
042588,000050:                                                                                                  #  ALARM OR ABORT EXIT MODE- NONE
042589,000051:                                                                                                  #  RESTART PROTECTION:
042590,000052:                                                                                                  #           ON A FRESH START AND RESTART THE 'STARTSUB' SUBROUTINE WILL INITIALIZE THE DOWNLIST POINTER(ACTUALLY
042591,000053:                                                                                                  #           DNTMGOTO) TO THE BEGINNING OF THE CURRENT DOWNLIST(I.E. CURRENT CONTENTS OF DNLSTADR). THIS HAS THE
042592,000054:                                                                                                  #           EFFECT OF IGNORING THE REMAINDER OF THE DOWNLIST WHICH THE DOWN-TELEMETRY PROGRAM WAS WORKING ON WHEN
042593,000055:                                                                                                  #           THE RESTART(OR FRESH START) OCCURRED AND RESUME DOWN TELEMETRY FROM THE BEGINNING OF THE CURRENT
042594,000056:                                                                                                  #           DOWNLIST.
042595,000057:                                                                                                  #           ALSO OF INTEREST IS THE FACT THAT ON A RESTART THE AGC WILL ZERO DOWNLINK CHANNELS 13, 34 AND 35.
042596,000058:                                                                                                  #  DOWNLINK LIST SELECTION:
042597,000059:                                                                                                  #        THE APPROPRIATE DOWNLINK LISTS ARE SELECTED BY THE FOLLOWING:
042598,000060:                                                                                                  #        1. FRESH START
042599,000061:                                                                                                  #        2. V37EXXE WHERE XX = THE MAJOR MODE BEING SELECTED.
042600,000062:                                                                                                  #        3. UPDATE PROGRAM(P27)
042601,000063:                                                                                                  #        4.     NON-V37 SELECTABLE TYPE PROGRAMS(E.G. AGS INITIALIZATION(SUNDANCE,LUMINARY) AND P61-P62
042602,000064:                                                                                                  #               TRANSITION(COLOSSUS) ETC.).
042603,000065:                                                                                                  #  DOWNLINK LIST RULES AND LIMITATIONS:
042604,000066:                                                                                                  #           READ SECTION(S) WHICH FOLLOW 'DEBRIS' WRITEUP.
042605,000067:                                                                                                  #  OUTPUT-  EVERY 2 SECONDS 100 DOUBLE PRECISION WORDS(I.E. 200 LGC
042606,000068:                                                                                                  #           COMPUTER WORDS) ARE TRANSMITTED VIA DOWNLINK.
042607,000069:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED- NONE
042608,000070:                                                                                                  #           'DNTMGOTO' AND 'DNLSTADR' ARE INITIALIZED BY THE FRESH START PROGRAM.

042619,000081:                                                                                                  #  NOTES REGARDING DOWNLINK LISTS ASSOCIATED WITH THIS PROGRAM:
042620,000082:                                                                                                  #  1. DOWNLISTS. - DOWNLISTS MUST BE COMPILED IN THE SAME BANK AS THE
042621,000083:                                                                                                  #     DOWN-TELEMETRY PROGRAM. THIS IS DONE FOR EASE OF CODING, FASTER
042622,000084:                                                                                                  #     EXECUTION.
042623,000085:                                                                                                  #  2. EACH DOWNLINK LIST CONSISTS OF A CONTROL LIST AND A NUMBER OF
042624,000086:                                                                                                  #     SUBLISTS.
042625,000087:                                                                                                  #  3. A SUBLIST REFERS TO A SNAPSHOT OR DATA COMMON TO THE SAME OR OTHER
042626,000088:                                                                                                  #     DOWNLINK LISTS.  ANY SUBLIST CONTAINING COMMON DATA NEEDS TO BE
042627,000089:                                                                                                  #     CODED ONLY ONCE FOR THE APPLICABLE DOWNLINK LISTS.
042628,000090:                                                                                                  #  4. SNAPSHOT SUBLISTS REFER SPECIFICALLY TO HOMOGENOUS DATA WHICH MUST BE
042629,000091:                                                                                                  #     SAVED IN A BUFFER DURING ONE DOWNRUPT.
042630,000092:                                                                                                  #  5. THE 1DNADR FOR THE 1ST WORD OF SNAPSHOT DATA IS FOUND AT THE END
042631,000093:                                                                                                  #     OF EACH SNAPSHOT SUBLIST, SINCE THE PROGRAM CODING SENDS THIS DP WORD
042632,000094:                                                                                                  #     IMMEDIATELY AFTER STORING THE OTHERS IN THE SNAPSHOT BUFFER.
042633,000095:                                                                                                  #  6. ALL LISTS ARE COMBINATIONS OF CODED ERASABLE ADDRESS CONSTANTS
042634,000096:                                                                                                  #     CREATED FOR THE DOWNLIST PROGRAM.
042635,000097:                                                                                                  #     A. 1DNADR              1-WORD DOWNLIST ADDRESS.
042636,000098:                                                                                                  #         SAME AS ECADR, BUT USED WHEN THE WORD ADDRESSED IS THE LEFT
042637,000099:                                                                                                  #         HALF OF A DOUBLE-PRECISION WORD FOR DOWN TELEMETRY.
042638,000100:                                                                                                  #     B. 2DNADR - 6DNADR     N-WORD DOWNLIST ADDRESS, N = 2 - 6.
042639,000101:                                                                                                  #         SAME AS 1DNADR, BUT WITH THE 4 UNUSED BITS OF THE ECADR FORMAT
042640,000102:                                                                                                  #         FILLED IN WITH 0001-0101.  USED TO POINT TO A LIST OF N DOUBLE-
042641,000103:                                                                                                  #         PRECISION WORDS, STORED CONSECUTIVELY, FOR DOWN TELEMETRY.
042642,000104:                                                                                                  #     C. DNCHAN              DOWNLIST CHANNEL ADDRESS.
042643,000105:                                                                                                  #         SAME AS 1DNADR, BUT WITH PREFIX BITS 0111.  USED TO POINT TO
042644,000106:                                                                                                  #         A PAIR OF CHANNELS FOR DOWN TELEMETRY.
042645,000107:                                                                                                  #     D. DNPTR               DOWN TELEMETRY SUBLIST POINTER.
042646,000108:                                                                                                  #      SAME AS CAF BUT TAGGED AS A CONSTANT. USED IN CONTROL LIST TO POINT TO A SUBLIST.
042647,000109:                                                                                                  #      CAUTION--- A DNPTR CANNOT BE USED IN A SUBLIST.
042648,000110:                                                                                                  #  7. THE WORD ORDER CODE IS SET TO ZERO AT THE BEGINNING OF EACH DOWNLIST (I.E. CONTROL LIST) AND WHEN
042649,000111:                                                                                                  #     A '1DNADR TIME2' IS DETECTED IN THE CONTROL LIST(ONLY).
042650,000112:                                                                                                  #  8. IN THE SNAPSHOT SUBLIST ONLY, THE DNADR'S CANNOT POINT TO THE FIRST WORD OF ANY EBANK.

042881,000343:                                                                                                  #  SUBROUTINE NAME- DNDUMP
042882,000344:                                                                                                  #  FUNCTIONAL DESCRIPTION - TO SEND (DUMP) ALL 8 BANKS OF ERASABLE STORAGE TWICE.  BANKS ARE SENT ONE AT A TIME
042883,000345:                                                                                                  #           EACH BANK IS PRECEEDED BY AN ID WORD, SYNCH BITS, ECADR AND TIME1 FOLLOWED BY THE 256D WORDS OF EACH
042884,000346:                                                                                                  #           EBANK. EBANKS ARE DUMPED IN ORDER(I.E. EBANK 0 FIRST, THEN EBANK1 ETC.)
042885,000347:                                                                                                  #  CALLING SEQUENCE- THE GROUND OR ASTRONAUT BY KEYING V74E CAN INITIALIZE THE DUMP.
042886,000348:                                                                                                  #           AFTER KEYING IN V74E THE CURRENT DOWNLIST WILL BE IMMEDIATELY TERMINATED AND THE DOWNLINK ERASABLE DUMP
042887,000349:                                                                                                  #           WILL BEGIN.
042888,000350:                                                                                                  #           ONCE INITIATED THE DOWNLINK ERASABLE DUMP CAN BE TERMINATED (AND INTERRUPTED DOWNLIST REINSTATED) ONLY
042889,000351:                                                                                                  #           BY THE FOLLOWING:
042890,000352:                                                                                                  #            1. A FRESH START
042891,000353:                                                                                                  #            2. COMPLETION OF BOTH COMPLETE DUMPS
042892,000354:                                                                                                  #            3. AND INVOLUNTARILY BY A RESTART.
042893,000355:                                                                                                  #  NORMAL EXIT MODE- TCF DNPHASE1
042894,000356:                                                                                                  #  ALARM OR ABORT MODE- NONE
042895,000357:                                                                                                  #  *SUBROUTINES CALLED- NONE.
042896,000358:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED-
042897,000359:                                                                                                  #           NONE
