Source Code

These source-code files are part of a reconstructed copy of Luminary 178, the final release of the Apollo 14 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Zerlina 56, Luminary 210, and Luminary 131, previously transcribed from digitized copies of those programs. The code was created by combining the three, using Zerlina 56 as a base, since it was based off of Luminary 183. The differences between the three were carefully merged using the Luminary memos as a guide, with all changes made after Luminary 178 being backed out. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021152N. Note that page numbers in the reconstructed code match those for the baseline log section mentioned in each file's changelog; the page numbers for a real Luminary 178 listing would be different.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

060329,000346:                                                                                                  #  PROGRAM DESCRIPTION:    D. KEENE   5/24/67

060332,000349: 
060333,000350:                                                                                                  #  MOD 2 BY CRAIG WORK, 6 APRIL 68 CONVERTS ATTITUDE ERROR DISPLAY SCALING FROM 16 7/8 DEG. TO 42 3/16 DEGREES.
060334,000351: 
060335,000352:                                                                                                  #      THIS SUBROUTINE IS USED TO DISPLAY ATTITUDE ERRORS ON THE FDAI VIA THE DIGITAL TO ANALOG CONVERTERS (DACS)
060336,000353:                                                                                                  #  IN THE CDUS.  CARE IS TAKEN TO METER OUT THE APPROPRIATE NUMBER OF PULSES TO THE IMU ERROR COUNTERS AND PREVENT
060337,000354:                                                                                                  #  OVERFLOW, TO CONTROL THE RELAY SEQUENCING, AND TO AVOID INTERFERENCE WITH THE COARSE ALIGN LOOP WHICH ALSO USES
060338,000355:                                                                                                  #  THE DACS.
060339,000356: 
060340,000357: 
060341,000358:                                                                                                  #  CALLING SEQUENCE:
060342,000359: 
060343,000360:                                                                                                  #      DURING THE INITIALIZATION SECTION OF THE USER'S PROGRAM, BIT3 OF RCSFLAGS SHOULD BE SET TO INITIATE THE
060344,000361:                                                                                                  #  TURN-ON SEQUENCE WITHIN THE NEEDLES PROGRAM:
060345,000362: 
060346,000363:                                                                                                  #           CS     RCSFLAGS        IN EBANK6
060347,000364:                                                                                                  #           MASK   BIT3
060348,000365:                                                                                                  #           ADS    RCSFLAGS
060349,000366: 
060350,000367:                                                                                                  #  THEREAFTER, THE ATTITUDE ERRORS GENERATED BY THE USER SHOULD BE TRANSFERED TO THE FOLLOWING LOCATIONS IN EBANK6:
060351,000368: 
060352,000369:                                                                                                  #           AK     SCALED 180 DEGREES  NOTE: THESE LOCATIONS ARE SUBJECT
060353,000370:                                                                                                  #           AK1    SCALED 180 DEGREES        TO CHANGE
060354,000371:                                                                                                  #           AK2    SCALED 180 DEGREES
060355,000372: 
060356,000373:                                                                                                  #  FULL SCALED DEFLECTION OF THE NEEDLES CORRESPONDS TO 5 1/16 DEGREES, WHILE 384 BITS IN THE IMU ERROR COUNTER
060357,000374:                                                                                                  #  CORRESPONDS TO 42 3/16 DEGREES.  (DAC MAXIMUM CAPACITY IS 384 BITS.) 46 BITS EFFECTIVELY PIN THE NEEDLES.
060358,000375: 
060359,000376:                                                                                                  #  FOR RATE DISPLAY, FULL SCALE NEEDLE DEFLECTION IS 1 17/64 DEG/SEC, AND A FULL COUNTER IS ABOUT 10.5 DEG/SEC.
060360,000377:                                                                                                  #  A CALL TO NEEDLER WILL THEN UPDATE THE DISPLAY:
060361,000378: 
060362,000379:                                                                                                  #           INHINT
060363,000380:                                                                                                  #           TC     IBNKCALL        NOTE: EBANK SHOULD BE SET TO E6

060365,000382:                                                                                                  #           RELINT

060367,000384:                                                                                                  #      THIS PROCESS SHOULD BE REPEATED EACH TIME THE ERRORS ARE UPDATED.  AT LEAST 3 PASSES THRU THE PROGRAM ARE
060368,000385:                                                                                                  #  REQUIRED BEFORE ANYTHING IS ACTUALLY DISPLAYED ON THE ERROR METERS.
