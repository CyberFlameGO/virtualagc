Source Code

These source-code files are part of a reconstructed copy of Luminary 163, the first (unflown) release of the Apollo 14 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction began with reconstructed source code for Luminary 173. Changes between revision 163 and 173 were backed out, as described by Luminary memos 157 and 158. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021152N. Note that page numbers in the reconstructed code match those for the baseline log section mentioned in each file's changelog; the page numbers for a real Luminary 163 listing would be different.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

042418,000030:                                                                                                  #  LOG SECTION- DOWN-TELEMETRY PROGRAM
042419,000031:                                                                                                  #  FUNCTIONAL DESCRIPTION- THIS ROUTINE IS INITIATED BY TELEMETRY END
042420,000032:                                                                                                  #           PULSE FROM THE DOWNLINK TELEMETRY CONVERTER. THIS PULSE OCCURS
042421,000033:                                                                                                  #           AT 50 TIMES PER SEC(EVERY 20 MS) THEREFORE DODOWNTM IS
042422,000034:                                                                                                  #           EXECUTED AT THESE RATES. THIS ROUTINE SELECTS THE APPROPRIATE
042423,000035:                                                                                                  #           AGC DATA TO BE TRANSMITTED DOWNLINK AND LOADS IT INTO OUTPUT
042424,000036:                                                                                                  #           CHANNELS 34 AND 35. THE INFORMATION IS THEN GATED OUT FROM THE
042425,000037:                                                                                                  #           LGC IN SERIAL FASHION.
042426,000038:                                                                                                  #           THIS PROGRAM IS CODED FOR A 2 SECOND DOWNLIST. SINCE DOWNRUPTS
042427,000039:                                                                                                  #           OCCUR EVERY 20MS AND 2 AGC COMPUTER WORDS CAN BE PLACED IN
042428,000040:                                                                                                  #           CHANNELS 34 AND 35 DURING EACH DOWNRUPT THE PROGRAM IS CAPABLE
042429,000041:                                                                                                  #           OF SENDING 200 AGC WORDS EVERY 2 SECONDS.
042430,000042:                                                                                                  #  CALLING SEQUENCE- NONE
042431,000043:                                                                                                  #           PROGRAM IS ENTERED VIA TCF DODOWNTM WHICH IS EXECUTED AS A
042432,000044:                                                                                                  #           RESULT OF A DOWNRUPT. CONTROL IS RETURNED VIA TCF RESUME WHICH
042433,000045:                                                                                                  #           IN EFFECT IS A RESUME.
042434,000046:                                                                                                  #  SUBROUTINES CALLED- NONE
042435,000047:                                                                                                  #  NORMAL EXIT MODE- TCF RESUME
042436,000048:                                                                                                  #  ALARM OR ABORT EXIT MODE- NONE
042437,000049:                                                                                                  #  RESTART PROTECTION:
042438,000050:                                                                                                  #           ON A FRESH START AND RESTART THE 'STARTSUB' SUBROUTINE WILL INITIALIZE THE DOWNLIST POINTER(ACTUALLY
042439,000051:                                                                                                  #           DNTMGOTO) TO THE BEGINNING OF THE CURRENT DOWNLIST(I.E. CURRENT CONTENTS OF DNLSTADR). THIS HAS THE
042440,000052:                                                                                                  #           EFFECT OF IGNORING THE REMAINDER OF THE DOWNLIST WHICH THE DOWN-TELEMETRY PROGRAM WAS WORKING ON WHEN
042441,000053:                                                                                                  #           THE RESTART(OR FRESH START) OCCURRED AND RESUME DOWN TELEMETRY FROM THE BEGINNING OF THE CURRENT
042442,000054:                                                                                                  #           DOWNLIST.
042443,000055:                                                                                                  #           ALSO OF INTEREST IS THE FACT THAT ON A RESTART THE AGC WILL ZERO DOWNLINK CHANNELS 13, 34 AND 35.
042444,000056:                                                                                                  #  DOWNLINK LIST SELECTION:
042445,000057:                                                                                                  #        THE APPROPRIATE DOWNLINK LISTS ARE SELECTED BY THE FOLLOWING:
042446,000058:                                                                                                  #        1. FRESH START
042447,000059:                                                                                                  #        2. V37EXXE WHERE XX = THE MAJOR MODE BEING SELECTED.
042448,000060:                                                                                                  #        3. UPDATE PROGRAM(P27)
042449,000061:                                                                                                  #        4.     NON-V37 SELECTABLE TYPE PROGRAMS(E.G. AGS INITIALIZATION(SUNDANCE,LUMINARY) AND P61-P62
042450,000062:                                                                                                  #               TRANSITION(COLOSSUS) ETC.).
042451,000063:                                                                                                  #  DOWNLINK LIST RULES AND LIMITATIONS:
042452,000064:                                                                                                  #           READ SECTION(S) WHICH FOLLOW 'DEBRIS' WRITEUP.
042453,000065:                                                                                                  #  OUTPUT-  EVERY 2 SECONDS 100 DOUBLE PRECISION WORDS(I.E. 200 LGC
042454,000066:                                                                                                  #           COMPUTER WORDS) ARE TRANSMITTED VIA DOWNLINK.
042455,000067:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED- NONE
042456,000068:                                                                                                  #           'DNTMGOTO' AND 'DNLSTADR' ARE INITIALIZED BY THE FRESH START PROGRAM.
042457,000069:                                                                                                  #  DEBRIS(ERASABLE LOCATIONS DESTROYED BY THIS PROGRAM)-
042458,000070:                                                                                                  #           LDATALST,DNTMBUFF TO DNTMBUFF +21D,TMINDEX,DNQ.

042467,000079:                                                                                                  #  NOTES REGARDING DOWNLINK LISTS ASSOCIATED WITH THIS PROGRAM:
042468,000080:                                                                                                  #  1. DOWNLISTS. - DOWNLISTS MUST BE COMPILED IN THE SAME BANK AS THE
042469,000081:                                                                                                  #     DOWN-TELEMETRY PROGRAM. THIS IS DONE FOR EASE OF CODING, FASTER
042470,000082:                                                                                                  #     EXECUTION.
042471,000083:                                                                                                  #  2. EACH DOWNLINK LIST CONSISTS OF A CONTROL LIST AND A NUMBER OF
042472,000084:                                                                                                  #     SUBLISTS.
042473,000085:                                                                                                  #  3. A SUBLIST REFERS TO A SNAPSHOT OR DATA COMMON TO THE SAME OR OTHER
042474,000086:                                                                                                  #     DOWNLINK LISTS.  ANY SUBLIST CONTAINING COMMON DATA NEEDS TO BE
042475,000087:                                                                                                  #     CODED ONLY ONCE FOR THE APPLICABLE DOWNLINK LISTS.
042476,000088:                                                                                                  #  4. SNAPSHOT SUBLISTS REFER SPECIFICALLY TO HOMOGENOUS DATA WHICH MUST BE
042477,000089:                                                                                                  #     SAVED IN A BUFFER DURING ONE DOWNRUPT.
042478,000090:                                                                                                  #  5. THE 1DNADR FOR THE 1ST WORD OF SNAPSHOT DATA IS FOUND AT THE END
042479,000091:                                                                                                  #     OF EACH SNAPSHOT SUBLIST, SINCE THE PROGRAM CODING SENDS THIS DP WORD
042480,000092:                                                                                                  #     IMMEDIATELY AFTER STORING THE OTHERS IN THE SNAPSHOT BUFFER.
042481,000093:                                                                                                  #  6. ALL LISTS ARE COMBINATIONS OF CODED ERASABLE ADDRESS CONSTANTS
042482,000094:                                                                                                  #     CREATED FOR THE DOWNLIST PROGRAM.
042483,000095:                                                                                                  #     A. 1DNADR              1-WORD DOWNLIST ADDRESS.
042484,000096:                                                                                                  #         SAME AS ECADR, BUT USED WHEN THE WORD ADDRESSED IS THE LEFT
042485,000097:                                                                                                  #         HALF OF A DOUBLE-PRECISION WORD FOR DOWN TELEMETRY.
042486,000098:                                                                                                  #     B. 2DNADR - 6DNADR     N-WORD DOWNLIST ADDRESS, N = 2 - 6.
042487,000099:                                                                                                  #         SAME AS 1DNADR, BUT WITH THE 4 UNUSED BITS OF THE ECADR FORMAT
042488,000100:                                                                                                  #         FILLED IN WITH 0001-0101.  USED TO POINT TO A LIST OF N DOUBLE-
042489,000101:                                                                                                  #         PRECISION WORDS, STORED CONSECUTIVELY, FOR DOWN TELEMETRY.
042490,000102:                                                                                                  #     C. DNCHAN              DOWNLIST CHANNEL ADDRESS.
042491,000103:                                                                                                  #         SAME AS 1DNADR, BUT WITH PREFIX BITS 0111.  USED TO POINT TO
042492,000104:                                                                                                  #         A PAIR OF CHANNELS FOR DOWN TELEMETRY.
042493,000105:                                                                                                  #     D. DNPTR               DOWN TELEMETRY SUBLIST POINTER.
042494,000106:                                                                                                  #      SAME AS CAF BUT TAGGED AS A CONSTANT. USED IN CONTROL LIST TO POINT TO A SUBLIST.
042495,000107:                                                                                                  #      CAUTION--- A DNPTR CANNOT BE USED IN A SUBLIST.
042496,000108:                                                                                                  #  7. THE WORD ORDER CODE IS SET TO ZERO AT THE BEGINNING OF EACH DOWNLIST (I.E. CONTROL LIST) AND WHEN
042497,000109:                                                                                                  #     A '1DNADR TIME2' IS DETECTED IN THE CONTROL LIST(ONLY).
042498,000110:                                                                                                  #  8. IN THE SNAPSHOT SUBLIST ONLY, THE DNADR'S CANNOT POINT TO THE FIRST WORD OF ANY EBANK.

042646,000258:                                                                                                  #  THE FOLLOWING CODING (FROM SNAPLOOP TO SNAPEND)IS FOR THE PURPOSE OF TAKING A SNAPSHOT OF 12 DP REGISTERS.
042647,000259:                                                                                                  #  THIS IS DONE BY SAVING 11 DP REGISTERS IN DNTMBUFF AND SENDING THE FIRST DP WORD IMMEDIATELY.
042648,000260:                                                                                                  #  THE SNAPSHOT PROCESSING IS THE MOST TIME CONSUMING AND THEREFORE THE CODING AND LIST STRUCTURE WERE DESIGNED
042649,000261:                                                                                                  #  TO MINIMIZE TIME. THE TIME OPTIMIZATION RESULTS IN RULES UNIQUE TO THE SNAPSHOT PORTION OF THE DOWNLIST.

042729,000341:                                                                                                  #  SUBROUTINE NAME- DNDUMP
042730,000342:                                                                                                  #  FUNCTIONAL DESCRIPTION - TO SEND (DUMP) ALL 8 BANKS OF ERASABLE STORAGE TWICE.  BANKS ARE SENT ONE AT A TIME
042731,000343:                                                                                                  #           EACH BANK IS PRECEEDED BY AN ID WORD, SYNCH BITS, ECADR AND TIME1 FOLLOWED BY THE 256D WORDS OF EACH
042732,000344:                                                                                                  #           EBANK. EBANKS ARE DUMPED IN ORDER(I.E. EBANK 0 FIRST, THEN EBANK1 ETC.)
042733,000345:                                                                                                  #  CALLING SEQUENCE- THE GROUND OR ASTRONAUT BY KEYING V74E CAN INITIALIZE THE DUMP.
042734,000346:                                                                                                  #           AFTER KEYING IN V74E THE CURRENT DOWNLIST WILL BE IMMEDIATELY TERMINATED AND THE DOWNLINK ERASABLE DUMP
042735,000347:                                                                                                  #           WILL BEGIN.
042736,000348:                                                                                                  #           ONCE INITIATED THE DOWNLINK ERASABLE DUMP CAN BE TERMINATED (AND INTERRUPTED DOWNLIST REINSTATED) ONLY
042737,000349:                                                                                                  #           BY THE FOLLOWING:
042738,000350:                                                                                                  #            1. A FRESH START
042739,000351:                                                                                                  #            2. COMPLETION OF BOTH COMPLETE DUMPS
