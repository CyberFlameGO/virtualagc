Source Code

These source-code files are part of a reconstructed copy of Luminary 163, the first (unflown) release of the Apollo 14 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction began with reconstructed source code for Luminary 173. Changes between revision 163 and 173 were backed out, as described by Luminary memos 157 and 158. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021152N. Note that page numbers in the reconstructed code match those for the baseline log section mentioned in each file's changelog; the page numbers for a real Luminary 163 listing would be different.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

004106,000025:                                                                                                  #        EQUALS  IS USED IN TWO WAYS.  IT IS OFTEN USED TO CHAIN A GROUP
004107,000026:                                                                                                  #                OF ASSIGNMENTS SO THAT THE GROUP MAY BE MOVED WITH THE
004108,000027:                                                                                                  #                CHANGING OF ONLY ONE CARD.  EXAMPLE.
004109,000028: 
004110,000029:                                                                                                  #                                        X       EQUALS  START
004111,000030:                                                                                                  #                                        Y       EQUALS  X       +SIZE.X
004112,000031:                                                                                                  #                                        Z       EQUALS  Y       +SIZE.Y
004113,000032: 
004114,000033:                                                                                                  #                (X, Y, AND Z ARE CONSECUTIVE AND BEGIN AT START.        )
004115,000034:                                                                                                  #                (SIZE.X AND SIZE.Y ARE THE RESPECTIVE SIZES OF X AND Y,
004116,000035:                                                                                                  #                USUALLY NUMERIC, IE. 1, 2, 6, 18D ETC.                  )
004117,000036: 
004118,000037:                                                                                                  #        EQUALS  OFTEN IMPLIES THE SHARING OF REGISTERS (DIFFERENT NAMES
004119,000038:                                                                                                  #                AND DIFFERENT DATA).  EXAMPLE.
004120,000039: 
004121,000040:                                                                                                  #                        X       EQUALS  Y
004122,000041: 
004123,000042:                                                                                                  #        =       MEANS THAT MULTIPLE NAMES HAVE BEEN GIVEN TO THE SAME DATA.
004124,000043:                                                                                                  #                (THIS IS LOGICAL EQUIVALENCE, NOT SHARING)  EXAMPLE.
004125,000044: 
004126,000045:                                                                                                  #                        X       =       Y
004127,000046: 
004128,000047:                                                                                                  #        THE SIZE AND UTILIZATION OF AN ERASABLE ARE OFTEN INCLUDED IN
004129,000048:                                                                                                  #        THE COMMENTS IN THE FOLLOWING FORM.  M(SIZE)N.
004130,000049: 
004131,000050:                                                                                                  #                M       REFERS TO THE MOBILITY OF THE ASSIGNMENT.
004132,000051:                                                                                                  #                        B       MEANS THAT THE SYMBOL IS REFERENCED BY BASIC
004133,000052:                                                                                                  #                                INSTRUCTIONS AND THUS IS E-BANK SENSITIVE.
004134,000053:                                                                                                  #                        I       MEANS THAT THE SYMBOL IS REFERENCED ONLY BY
004135,000054:                                                                                                  #                                INTERPRETIVE INSTRUCTIONS, AND IS THUS E-BANK
004136,000055:                                                                                                  #                                INSENSITIVE AND MAY APPEAR IN ANY E-BANK.
004137,000056: 
004138,000057:                                                                                                  #                SIZE    IS THE NUMBER OF REGISTERS INCLUDED BY THE SYMBOL.
004139,000058: 
004140,000059:                                                                                                  #                N       INDICATES THE NATURE OR PERMANENCE OF THE CONTENTS.
004141,000060:                                                                                                  #                        PL      MEANS THAT THE CONTENTS ARE PAD LOADED.
004142,000061:                                                                                                  #                        DSP     MEANS THAT THE REGISTER IS USED FOR A DISPLAY.
004143,000062:                                                                                                  #                        PRM     MEANS THAT THE REGISTER IS PERMANENT, IE. IT
004144,000063:                                                                                                  #                                IS USED DURING THE ENTIRE MISSION FOR ONE
004145,000064:                                                                                                  #                                PURPOSE AND CANNOT BE SHARED.
004146,000065:                                                                                                  #                        TMP     MEANS THAT THE REGISTER IS USED TEMPORARILY OR
004147,000066:                                                                                                  #                                IS A SCRATCH REGISTER FOR THE ROUTINE TO WHICH
004148,000067:                                                                                                  #                                IT IS ASSIGNED.  THAT IS, IT NEED NOT BE SET
004149,000068:                                                                                                  #                                PRIOR TO INVOCATION OF THE ROUTINE NOR DOES IT
004150,000069:                                                                                                  #                                CONTAIN USEFUL OUTPUT TO ANOTHER ROUTINE.  THUS
