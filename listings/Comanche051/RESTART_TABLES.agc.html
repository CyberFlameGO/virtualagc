Source Code

These source-code files are part of a reconstructed copy of Comanche 51, the initial release (though not the final mission release) of the Apollo 11 Command Module (CM) Apollo Guidance Computer (AGC) software. The flown, final release was Comanche 55. The reconstruction is believed to be exactly accurate, but in the absence of an actual Apollo-era listing of Comanche 51, some explanation of the reconstruction process is needed to insure confidence in that accuracy.

The starting point was the source code of Comanche 55, whose original source code is available. Comanche 55 source code was then modified by undoing changes known to have been made between revisions 51 and 55. The most-significant steps involved reverting changes related to the R-2 lunar potential model, the details of which were known from the previous, separate, independent reconstruction of the Apollo 10 Lunar Module software Luminary 69/2. The now-reconstructed Comanche 51 source code was verified by assembling it and checking that the assembled code had the expected known-correct memory-bank checksums. Note that page numbers in the reconstructed code match those on the Comanche 55 or Luminary 69 printouts, although the source code would likely have different page numbers in a contemporary Comanche 51 listing.

Annotations that were not present in the contemporary source code have been added to the reconstructed code to justify each change relative to Comanche 55. Here's a guide to some of the Apollo documentation relevant to those annotations:
  • MIT/IL engineering drawing 2021153D, sheet 3. Listing of the memory-bank checksums for Comanche 51.
  • Comanche 55 (Apollo 11 CM final release) assembly listing.
  • Reconstructed Luminary69/2 (Apollo 10 LM final release) assembly listing.
  • Luminary 69 (Apollo 10 LM initial release) assembly listing.
  • Colossus 249 (Apollo 9 CM final release) assembly listing.
  • Programmed Guidance Equations for Colossus 2, Based on Comanche 55. A pseudocode representation of Comanche 55, showing change bars for differences from Comanche 45/2.
There is also a short writeup about the reconstruction process in our software repository that may be helpful.

Comments expected to have been present in the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC Project's GitHub repository.

008795,000023:                                                                                                  #  THERE ARE TWO FORMS OF RESTART TABLES FOR EACH GROUP.  THEY ARE KNOWN AS THE EVEN RESTART TABLES AND THE ODD
008796,000024:                                                                                                  #  RESTART TABLES.  THE ODD TABLES HAVE ONLY ONE ENTRY OF THREE LOCATIONS WHILE THE EVEN TABLES HAVE TWO ENTRIES
008797,000025:                                                                                                  #  EACH USING THREE LOCATIONS.  THE INFORMATION AS TO WHETHER IT IS A JOB, WAITLIST, OR A LONGCALL IS GIVEN BY THE
008798,000026:                                                                                                  #  WAY THINGS ARE PUT IN TO THE TABLES.
008799,000027: 
008800,000028:                                                                                                  #  A JOB HAS ITS PRIORITY STORED IN PRDTTAB OF THE CORRECT PHASE SPOT - A POSITIVE PRIORITY INDICATES A
008801,000029:                                                                                                  #  FINDVAC JOB, A NEGATIVE PRIORITY A NOVAC.  THE 2CADR OF THE JOB IS STORED IN THE CADRTAB.
008802,000030:                                                                                                  #  FOR EXAMPLE,
008803,000031: 
008804,000032:                                                                                                  #                5.7SPOT         OCT     23000
008805,000033:                                                                                                  #                                2CADR   SOMEJOB
008806,000034: 
008807,000035:                                                                                                  #  A RESTART OF GROUP 5 WITH PHASE SEVEN WOULD THEN CAUSE SOMEJOB TO BE RESTARTED AS A FINDVAC WITH PRIORITY 23.
008808,000036:

008810,000038:                                                                                                  #                                2CADR   ANYJOB
008811,000039: 
008812,000040:                                                                                                  #  HERE A RESTART OF GROUP 5 WITH PHASE 7 WOULD CAUSE ANYJOB TO BE RESTARTED AS A NOVAC WITH PRIORITY 23.

008814,000042:                                                                                                  #  PLACED THE LOCATION OF A DP REGISTER THAT CONTAINS THE DELTA TIME THAT LONGCALL HAD BEEN ORIGINALLY STARTED
008815,000043:                                                                                                  #  WITH.  EXAMPLE,
008816,000044: 
008817,000045:                                                                                                  #                3.6SPOT         GENADR  DELTAT
008818,000046:                                                                                                  #                                -GENADR LONGTASK
008819,000047:                                                                                                  #                                BBCON   LONGTASK
008820,000048: 
008821,000049:                                                                                                  #                                OCT     31000
008822,000050:                                                                                                  #                                2CADR   JOBAGAIN
008823,000051: 
008824,000052:                                                                                                  #  THIS WOULD START UP LONGTASK AT THE APPROPRIATE TIME, OR IMMEDIATELY IF THE TIME HAD ALREADY PASSED.  IT SHOULD
008825,000053:                                                                                                  #  BE NOTED THAT IF DELTAT IS IN A SWITCHED E BANK, THIS INFORMATOIN SHOULD BE IN THE BBCON OF THE 2CADR OF THE
008826,000054:                                                                                                  #  TASK.  FROM ABOVE, WE SEE THAT THE SECOND PART OF THIS PHASE WOULD BE STARTED AS A JOB WITH A PRIORITY OF 31.
008827,000055: 
008828,000056:                                                                                                  #  WAITLIST CALLS ARE IDENTIFIED BY THE FACT THAT THEIR 2CADR IS STORED NEGATIVELY.  IF PRDTTAB OF THE PHASE SPOT
008829,000057:                                                                                                  #  IS POSITIVE, THEN IT CONTAINS THE DELTA TIME, IF PRDTTAB IS NEGATIVE THEN IT IS THE -GENADR OF AN ERASABLE
008830,000058:                                                                                                  #  LOCATION CONTAINING THE DELTA TIME, THAT IS, THE TIME IS STORED INDIRECTLY. IT SHOULD BE NOTED AS ABOVE, THAT
008831,000059:                                                                                                  #  IF THE TIME IS STORED INDIRECTLY, THE BBCON MUST CONTAIN THE NECESSARY E BANK INFORMATION IF APPLICABLE.  WITH
008832,000060:                                                                                                  #  WAITLIST WE HAVE ONE FURTHER OPTION, IF -0 IS STORED IN PRDTTAB, IT WILL CAUSE AN IMMEDIATE RESTART OF THE
008833,000061:                                                                                                  #  TASK.  EXAMPLES,
008834,000062: 
008835,000063:                                                                                                  #                                OCT     77777           THIS WILL CAUSE AN IMMEDIATE RESTART
008836,000064:                                                                                                  #                                -2CADR  ATASK           OF THE TASK :ATASK:

008839,000067:                                                                                                  #                                -2CADR  DUMMY           WAS PUT ON WAITLIST IS UP, IT WILL BEGIN

008841,000069:                                                                                                  #                                                        IT NORMALLY WOULD HAVE BEGUN.
