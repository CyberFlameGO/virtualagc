Source Code

These source-code files were transcribed from scans made from Don Eyles's personal copy of Luminary 069. They were scanned at archive.org's Boston facility, and the scanning was sponsored by Onno Hommes. The code was transcribed from these scans by a team of volunteers who are referenced in the program comments. Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". In some cases, where similar code blocks exist in previously-transcribed AGC programs (primarily Luminary 99, from Apollo 11) those code blocks were used as a starting point and then corrected to agree with the Luminary 69 scans. The full scans are available at the Virtual AGC project's collection at archive.org, while more-convenient reduced-size (but reduced-quality) images are available at the main Virtual AGC website. Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 069 OF AGC PROGRAM LUMINARY BY NASA 2021112-011
	19:02 NOV. 25,1968
Note that the date is the date of the printout, not the date of the program revision.

055361,000031:                                                                                                  #  FUNCTIONAL DESCRIPTION
055362,000032:                                                                                                  #        PROGRAM HAS TWO MAIN PARTS. THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
055363,000033:                                                                                                  #  PART OF THE BACK-UP IDLE LOOP. THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
055364,000034:                                                                                                  #  STARTING VERB.
055365,000035:                                                                                                  #        THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
055366,000036:                                                                                                  #        THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK, ONE AT A TIME.
055367,000037:                                                                                                  #        IN ALL THERE ARE 7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK. MORE DETAIL DESCRIPTION MAY BE
055368,000038:                                                                                                  #  FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
055369,000039: 
055370,000040:                                                                                                  #        THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27). BELOW IS
055371,000041:                                                                                                  #   A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
055372,000042:                                                                                                  #  OCTAL, TO LOAD INTO SMODE.
055373,000043:                                                                                                  #  +-4   ERASABLE MEMORY
055374,000044:                                                                                                  #  +-5   FIXED MEMORY
055375,000045:                                                                                                  #  +-1,2,3,6,7,10   EVERYTHING IN OPTIONS 4 AND 5.
055376,000046:                                                                                                  #  -0    SAME AS +-10 UNTIL AN ERROR IS DETECTED.
055377,000047:                                                                                                  #  +0    NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
055378,000048: 
055379,000049:                                                                                                  #  WARNINGS
055380,000050:                                                                                                  #        USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
055381,000051:                                                                                                  #        SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
055382,000052: 
055383,000053:                                                                                                  #  CALLING SEQUENCE
055384,000054:                                                                                                  #        TO CALL SELF-CHECK KEY IN
055385,000055:                                                                                                  #                V 21 N 27 E     OPTION NUMBER E
055386,000056:                                                                                                  #        TO CALL SHOW-BANKSUM KEY IN
055387,000057:                                                                                                  #                V 91 E          DISPLAYS FIRST BANK
055388,000058:                                                                                                  #                V 33 E          PROCEED, DISPLAYS NEXT BANK
055389,000059: 
055390,000060:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
055391,000061:                                                                                                  #        SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED. IF SO + OPTION NUMBERS PUT
055392,000062:                                                                                                  #  COMPUTER INTO BACKUP IDLE LOOP, - OPTION NUMBERS RESTART THE OPTION.
055393,000063:                                                                                                  #        THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
055394,000064:                                                                                                  #        SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E).  THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP
055395,000065: 
055396,000066:                                                                                                  #  OUTPUT

055398,000068:                                                                                                  #        SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
055399,000069:                                                                                                  #  TURNS ON THE ALARM LIGHT. THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E. FOR FURTHER
055400,000070:                                                                                                  #  INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
055401,000071:                                                                                                  #  IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
055402,000072:                                                                                                  #  INITIATED FRESH START (SLAP1).
055403,000073: 
055404,000074:                                                                                                  #        SHOW-BANKSUM STARTING WITH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
055405,000075:                                                                                                  #  THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
055406,000076: 
055407,000077:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
055408,000078:                                                                                                  #        ACCOMPLISHED BY FRESH START
055409,000079:                                                                                                  #                SMODE SET TO +0
055410,000080: 
055411,000081:                                                                                                  #  DEBRIS
055412,000082:                                                                                                  #        ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
055413,000083:                                                                                                  #  EXCEPTION IS A RESTART. RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
055414,000084:                                                                                                  #  E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
