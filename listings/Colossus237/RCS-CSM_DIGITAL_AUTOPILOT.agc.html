Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

042513,000258:                                                                                                  #  WHEN AUTOMATIC MANEUVERS ARE BEING PERFORMED, THE FOLLOWING ANGLE ADDITION MUST BE MADE TO PROVIDE A SMOOTH
042514,000259:                                                                                                  #  SEQUENCE OF ANGULAR COMMANDS TO THE AUTOPILOT--
042515,000260: 
042516,000261:                                                                                                  #           CDUXD = CDUXD + DELCDUX    (DOUBLE PRECISION)
042517,000262:                                                                                                  #           CDUYD = CDUYD + DELCDUY    (DOUBLE PRECISION)
042518,000263:                                                                                                  #           CDUZD = CDUZD + DELCDUZ    (DOUBLE PRECISION)
042519,000264: 
042520,000265:                                                                                                  #  THE STEERING PROGRAMS-
042521,000266:                                                                                                  #           1) ATTITUDE MANEUVER ROUTINE
042522,000267:                                                                                                  #           2) LEM TRACKING
042523,000268: 
042524,000269:                                                                                                  #  SHOULD GENERATE THE DESIRED ANGLES (CDUXD, CDUYD, CDUZD) AS WELL AS THE INCREMENTAL ANGLES (DELCDUX, DELCDUY,
042525,000270:                                                                                                  #  DELCDUZ) SO THAT THE GIMBAL ANGLE COMMANDS CAN BE INTERPOLATED BETWEEN UPDATES.
042526,000271: 
042527,000272:                                                                                                  #  HOLDFLAG CODING-
042528,000273: 
042529,000274:                                                                                                  #      +  = GRAB PRESENT CDU ANGLES AND STORE IN THETADX, THETADY, THETADZ
042530,000275:                                                                                                  #           AND PERFORM ATTITUDE HOLD ABOUT THESE ANGLES
042531,000276:                                                                                                  #           ALSO IGNORE AUTOMATIC STEERING
042532,000277:                                                                                                  #           SET = + BY
042533,000278:                                                                                                  #                  1) INITIALIZATION PHASE OF AUTOPILOT
042534,000279:                                                                                                  #                  2) OCCURANCE OF RHC COMMANDS
042535,000280:                                                                                                  #                  3) FREE MODE
042536,000281:                                                                                                  #                  4) SWITCH OVER TO ATTITUDE HOLD FROM AUTO
042537,000282:                                                                                                  #                     WHILE DOING AUTOMATIC STEERING (IN THIS CASE
042538,000283:                                                                                                  #                     HOLDFLAG IS NOT ACTUALLY SET TO +, BUT THE LOGIC
042539,000284:                                                                                                  #                     FUNCTIONS AS IF IT WERE.)
042540,000285:                                                                                                  #                  5) S/C CONTROL SWITCH IN SCS
042541,000286:                                                                                                  #                  6) IMU POWER OFF
