Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

050514,000032:                                                                                                  #  FUNCTIONAL DESCRIPTION-
050515,000033:                                                                                                  #          PART OF A SECTION OF PROGRAMS,-WAITLIST, TASKOVER, T3RUPT, USED  TO CALL A PROGRAM, (CALLED A TASK),
050516,000034:                                                                                                  #     WHICH IS TO BEGIN IN C(A) CENTISECONDS.  WAITLIST UPDATES TIME3, LST1 AND LST2. THE MEANING OF THESE LISTS
050517,000035:                                                                                                  #     FOLLOW.
050518,000036: 
050519,000037:                                                                                                  #                  C(TIME3)  = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
050520,000038: 
050521,000039: 
050522,000040:                                                                                                  #                               C(LST1)    = -(T2-T1)+1
050523,000041:                                                                                                  #                               C(LST1 +1) = -(T3-T2)+1
050524,000042:                                                                                                  #                               C(LST1 +2) = -(T4-T3)+1
050525,000043:                                                                                                  #                                          .
050526,000044:                                                                                                  #                                          .
050527,000045:                                                                                                  #                               C(LST1 +6) = -(T8-T7)+1
050528,000046:                                                                                                  #                               C(LST1 +7) = -(T9-T8)+1
050529,000047: 
050530,000048:                                                                                                  #                               C(LST2)    = 2CADR OF TASK1
050531,000049:                                                                                                  #                               C(LST2 +2) = 2CADR OF TASK2
050532,000050:                                                                                                  #                                          .
050533,000051:                                                                                                  #                                          .
050534,000052:                                                                                                  #                               C(LST2 +14)= 2CADR OF TASK8
050535,000053:                                                                                                  #                               C(LST2 +16)= 2CADR OF TASK9
050536,000054: 
050537,000055:                                                                                                  #  WARNINGS-
050538,000056:                                                                                                  #  --------
050539,000057:                                                                                                  #               1) 1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
050540,000058:                                                                                                  #               2) 9 TASKS MAXIMUM
050541,000059:                                                                                                  #               3) TASKS CALLED UNDER INTERRUPT INHIBITED
050542,000060:                                                                                                  #               4) TASKS END BY TC TASKOVER
050543,000061: 
050544,000062:                                                                                                  #  CALLING SEQUENCE-
050545,000063: 
050546,000064:                                                                                                  #               L-1    CA     DELTAT   (TIME IN CENTISECONDS TO TASK START)
050547,000065:                                                                                                  #               L      TC     WAITLIST
050548,000066:                                                                                                  #               L+1    2CADR  DESIRED TASK
050549,000067:                                                                                                  #               L+2    (MINOR OF 2CADR)
050550,000068:                                                                                                  #               L+3    RELINT          (RETURNS HERE)
050551,000069: 
050552,000070:                                                                                                  #  TWIDDLE-
050553,000071:                                                                                                  #  -------
050554,000072:                                                                                                  #               TWIDDLE IS FOR USE WHEN THE TASK BEING SET UP IS IN THE SAME EBANK AND FBANK AS THE USER.   IN
050555,000073:                                                                                                  #               SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,

050557,000075:                                                                                                  #               SAVING A WORD.   TWIDDLE IS LIKE WAITLIST IN EVERY RESPECT EXCEPT CALLING SEQUENCE, TO WIT-

050560,000078:                                                                                                  #               L      TC     TWIDDLE
050561,000079:                                                                                                  #               L+1    ADRES  DESIRED TASK
050562,000080:                                                                                                  #               L+2    RELINT               (RETURNS HERE)

050565,000083:

050567,000085:

050575,000093:                                                                                                  #               ACCOMPLISHED BY FRESH START,--LST2,..., LST2 +16 =ENDTASK
050576,000094:                                                                                                  #                                             LST1,..., LST1 +7  =NEG1/2
050577,000095: 
050578,000096:                                                                                                  #  OUTPUT--
050579,000097: 
050580,000098:                                                                                                  #               LST1 AND LST2 UPDATED WITH NEW TASK AND ASSOCIATED TIME.
050581,000099:                                                                                                  #  DEBRIS-
050582,000100: 
050583,000101:                                                                                                  #               CENTRALS- A,Q,L
050584,000102:                                                                                                  #               OTHER   - WAITEXIT, WAITADR, WAITTEMP, WAITBANK
050585,000103:                                                                                                  #  DETAILED ANALYSIS OF TIMING-
050586,000104:                                                                                                  #          CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR)   IN EXACTLY DELTA T CENTISECONDS.
050587,000105:                                                                                                  #     THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
050588,000106: 
050589,000107:                                                                                                  #               LET TO = THE TIME OF THE TC WAITLIST
050590,000108:                                                                                                  #               LET TS = TO +147U + COUNTER INCREMENTS (SET UP TIME)
050591,000109:                                                                                                  #               LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
050592,000110:                                                                                                  #               LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
050593,000111:                                                                                                  #               LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
050594,000112:                                                                                                  #     (Z=0, USUALLY)
050595,000113:                                                                                                  #               LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
050596,000114:                                                                                                  #            THEN   DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
050597,000115:                                                                                                  #     *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
050598,000116: 
050599,000117: 
050600,000118:                                                                                                  #          IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
050601,000119:                                                                                                  #     INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
050602,000120:                                                                                                  #     OF OTHER TASKS INHIBITING THE INTERRUPT.
