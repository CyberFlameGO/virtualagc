Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

041178,000025: 
041179,000026:                                                                                                  #      *AN ADAPTATION OF THE LEM P-AXIS CONTROLLER
041180,000027:                                                                                                  #      *MAINTAIN OGA WITHIN 5 DEG DEADBND OF OGAD, WHERE OGAD = OGA AS SEEN
041181,000028:                                                                                                  #       BY IGNOVER  (IGNITION)

041184,000031:                                                                                                  #      *USES T6 CLOCK TO TIME JET FIRINGS

041186,000033:                                                                                                  #      *MINIMUM JET FIRING TIME = 15 MS
041187,000034:                                                                                                  #      *JET PAIRS FIRE ALTERNATELY
041188,000035:                                                                                                  #      *AT LEAST 1/2 SECOND DELAY BEFORE A NEW JET PAIR IS FIRED
041189,000036:                                                                                                  #      *JET FIRINGS MAY NOT BE EXTENDED, ONLY SHORTENED, WHEN RE-EVALUATION
041190,000037:                                                                                                  #       OF A JET FIRING TIME IS MADE ON A LATER PASS
041191,000038: 
041192,000039:                                                                                                  #  CALLING SEQUENCE....
041193,000040: 
041194,000041:                                                                                                  #      *ROLLDAP CALL VIA WAITLIST, IN PARTICULAR BY TVCEXEC (EVERY 1/2 SEC)
041195,000042:                                                                                                  #       WITH A 3CS DELAY TO ALLOW FREE TIME FOR OTHER RUPTS (DWNRPT, ETC.)
041196,000043:

041198,000045:

041273,000120:                                                                                                  #  GIVEN THE OPERATING POINT NOT IN THE COAST REGION, THE DESIRED OGARATE
041274,000121:                                                                                                  #       IS AT THE POINT OF PENETRATION OF THE COAST REGION BY THE CONTROL
041275,000122:                                                                                                  #       PARABOLA WHICH PASSES THROUGH THE OPERATING POINT. FOR REGION 3
041276,000123:                                                                                                  #       DESIRED OGARATE IS SIMPLY +-MAXLIM. FOR REGIONS 1 OR 6 THE SOLUTION
041277,000124:                                                                                                  #       TO A QUADRATIC IS REQUIRED (THE PENETRATION IS ALONG THE STRAIGHT
041278,000125:                                                                                                  #       LINE OR MINLIM BOUNDRY SWITCH LINES). AN APPROXIMATION IS MADE
041279,000126:                                                                                                  #       INSTEAD. CONSIDER AN OPERATING POINT IN REGION 6. PASS A TANGENT TO
041280,000127:                                                                                                  #       THE CONTROL PARABOLA THROUGH THE OPERATING POINT, AND FIND ITS
041281,000128:                                                                                                  #       INTERSECTION WITH THE STRAIGHT LINE SECTION OF THE SWITCH CURVE...
041282,000129:                                                                                                  #       THE INTERSECTION DEFINES DESIRED OGARATE. IF THE OPERATING POINT IS
041283,000130:                                                                                                  #       CLOSE TO THE SWITCH LINE, THE APPROXIMATION IS QUITE GOOD (INDEED
041284,000131:                                                                                                  #       THE APPROXIMATE AND QUADRATIC SOLUTIONS CONVERGE IN THE LIMIT AS
041285,000132:                                                                                                  #       THE SWITCH LINE IS APPROACHED). IF THE OPERATING POINT IS NOT CLOSE
041286,000133:                                                                                                  #       TO THE SWITCH LINE, THE APPROXIMATE SOLUTION GIVES VALID TREND
041287,000134:                                                                                                  #       INFORMATION (DIRECTION OF DESIRED OGARATE) AT LEAST. THE 
041288,000135:                                                                                                  #       RE-EVALUATION OF DESIRED OGARATE IN SUBSEQUENT ROLL DAP PASSES (1/2
041289,000136:                                                                                                  #       SECOND INTERVALS) WILL BENEFIT FROM THE CONVERGENT NATURE OF THE
041290,000137:                                                                                                  #       APPROXIMATION.
041291,000138: 
041292,000139:                                                                                                  #  FOR LARGE OGAERROR THE TANGENT INTERSECTS +-MINLIM SWITCH BOUNDRY BEFORE
041293,000140:                                                                                                  #       INTERSECTING THE STRAIGHT LINE SWITCH. HOWEVER THE MINLIM IS
041294,000141:                                                                                                  #       IGNORED IN COMPUTING THE FIRING TIME, SO THAT THE EXTENSION (INTO
041295,000142:                                                                                                  #       THE COAST REGION) OF THE STRAIGHT LINE SWITCH IS WHAT IS FIRED TO.
041296,000143:                                                                                                  #       IF THE ROLL DAP FINDS ITSELF IN THE COAST REGION BEFORE REACHING
041297,000144:                                                                                                  #       THE DESIRED INTERSECTION (IE, IN THE REGION BETWEEN THE MINLIM
041298,000145:                                                                                                  #       AND THE STRAIGHT LINE SWITCH) IT WILL EXIBIT NORMAL COAST-REGION
041299,000146:                                                                                                  #       BEHAVIOR AND TURN OFF THE JETS. THE PURPOSE OF THIS FIRING POLICY
041300,000147:                                                                                                  #       IS TO MAINTAIN STATIC ROLL STABILITY IN THE EVENT OF A JET
041301,000148:                                                                                                  #       FAILED-ON.
041302,000149: 
041303,000150:                                                                                                  #  WHEN THE OPERATING POINT IS IN REGION 1 THE SAME APPROXIMATION IS
041304,000151:                                                                                                  #       MADE, BUT AT AN ARTIFICIALLY-CREATED OR DUMMY OPERATING POINT,
041305,000152:                                                                                                  #       DEFINED BY.. OGAERROR = INTERSECTION OF CONTROL PARABOLA AND
041306,000153:                                                                                                  #       OGAERROR AXIS, OGARATE = +-LMCRATE WHERE SIGN IS OPPOSITE THAT OF
041307,000154:                                                                                                  #       REAL OPERATING POINT RATE.  WHEN THE OPERATING POINT HAS PASSED
041308,000155:                                                                                                  #       FROM REGION 1 TO REGION 6, THE DUMMY POINT IS NO LONGER REQUIRED,
041309,000156:                                                                                                  #       AND THE SOLUTION REVERTS TO THAT OF A REGULAR REGION 6 POINT.
041310,000157: 
041311,000158: 
041312,000159:                                                                                                  #  EQUATION FOR SWITCHING PARABOLA (SEE FIGURE ABOVE)....
041313,000160:                                                                                                  #                                  2
041314,000161:                                                                                                  #      SOGAERROR = (DB - (SOGARATE) (1/CONACC)/2) SGN(SOGARATE)
041315,000162: 
041316,000163: 
041317,000164:                                                                                                  #  EQUATION FOR SWITCHING STRAIGHT LINE SEGMENT....
041318,000165: 
041319,000166:                                                                                                  #       SOGARATE = -(-SLOPE)(SOGAERROR) - SGN(SOGARATE) INTERCEP
041320,000167: 
041321,000168:                                                                                                  #                WHERE  INTERCEP = DB(-SLOPE) - LMCRATE

041325,000172: 
041326,000173:                                                                                                  #       DOGADOT = NUM/DEN, WHERE
041327,000174:                                                                                                  #                                  2
