Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

012454,000303:                                                                                                  #  SUBROUTINE NAME:       EARTH ROTATOR     (EARROT1 OR EARROT2)           DATE:           15 FEB 67
012455,000304:                                                                                                  #  MOD NO:  N +1                                                           LOG SECTION:    POWERED FLIGHT SUBROS
012456,000305:                                                                                                  #  MOD BY:  ENTRY GROUP  (BAIRNSFATHER)
012457,000306:                                                                                                  #  FUNCTIONAL DESCRIPTION:      THIS ROUTINE PROJECTS THE INITIAL EARTH TARGET VECTOR   RTINIT   AHEAD THROUGH
012458,000307:                                                                                                  #           THE ESTIMATED TIME OF FLIGHT. INITIAL CALL RESOLVES THE INITIAL TARGET VECTOR   RTINIT   INTO EASTERLY
012459,000308:                                                                                                  #           AND NORMAL COMPONENTS   RTEAST   AND   RTNORM   . INITIAL AND  SUBSEQUENT CALLS ROTATE THIS VECTOR
012460,000309:                                                                                                  #           ABOUT THE (FULL) UNIT POLAR AXIS   UNITW   THROUGH THE ANGLE    WIE DTEAROT   TO OBTAIN THE ROTATED
012461,000310:                                                                                                  #           TARGET VECTOR   RT   . ALL VECTORS EXCEPT   UNITW   ARE HALF UNIT.
012462,000311:                                                                                                  #           THE EQUATIONS ARE
012463,000312:                                                                                                  #                  -    -        -                     -
012464,000313:                                                                                                  #                  RT = RTINIT + RTNORM (COS(WT) -1) + RTEAST SIN(WT)
012465,000314:                                                                                                  #           WHERE  WT = WIE DTEAROT
012466,000315:                                                                                                  #                  RTINIT = INITIAL TARGET VECTOR
012467,000316:                                                                                                  #                  -        -     -
012468,000317:                                                                                                  #                  RTEAST = UNITW*RTINIT
012469,000318:                                                                                                  #                  -        -      -
012470,000319:                                                                                                  #                  RTNORM = RTEAST*UNITW
012471,000320: 
012472,000321:                                                                                                  #           FOR CONTINUOUS UPDATING, ONLY ONE ENTRY TO EARROT1 IS REQUIRED, WITH SUBSEQUENT ENTRIES AT EARROT2.
012473,000322:                                                                                                  #  CALLING SEQUENCE:  FIRST CALL                    SUBSEQUENT CALL
