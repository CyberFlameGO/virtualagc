Source Code

These source-code files are part of a reconstructed copy of Luminary 173, the second (unflown) release of the Apollo 14 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction began with reconstructed source code for Luminary 178. Changes between revision 173 and 178 were backed out, as described by Luminary memo 167 (revision 1). The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021152N. Note that page numbers in the reconstructed code match those for the baseline log section mentioned in each file's changelog; the page numbers for a real Luminary 173 listing would be different.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

059562,000021:                                                                                                  #  PROGRAM NAMES:  (1) T6JOBCHK      MOD. NO. 5    OCTOBER 2, 1967
059563,000022:                                                                                                  #                 (2) DOT6RUPT
059564,000023: 
059565,000024:                                                                                                  #  MODIFICATION BY: LOWELL G HULL (A.C.ELECTRONICS)
059566,000025:                                                                                                  #  THESE PROGRAMS ENABLE THE LM DAP TO CONTROL THE THRUST TIMES OF THE REACTION CONTROL SYSTEM JETS BY USING TIME6.
059567,000026:                                                                                                  #  SINCE THE LM DAP MAINTAINS EXCLUSIVE CONTROL OVER TIME6 AND ITS INTERRUPTS, THE FOLLOWING CONVENTIONS HAVE BEEN
059568,000027:                                                                                                  #  ESTABLISHED AND MUST NOT BE TAMPERED WITH:
059569,000028:                                                                                                  #           1. NO NUMBER IS EVER PLACED INTO TIME6 EXCEPT BY LM DAP.
059570,000029:                                                                                                  #           2. NO PROGRAM OTHER THAN LM DAP ENABLES THE TIME6 COUNTER.
059571,000030:                                                                                                  #           3. TO USE TIME6, THE FOLLOWING SEQUENCE IS ALWAYS EMPLOYED:
059572,000031:                                                                                                  #                  A. A POSITIVE (NON-ZERO) NUMBER IS STORED IN TIME6.
059573,000032:                                                                                                  #                  B. THE TIME6 CLOCK IS ENABLED.
059574,000033:                                                                                                  #                  C. TIME6 IS INTERROGATED AND IS:
059575,000034:                                                                                                  #                         I. NEVER FOUND NEGATIVE(NON-ZERO) OR +0.
059576,000035:                                                                                                  #                        II. SOMETIMES FOUND POSITIVE(BETWEEN 1 AND 240D) INDICATING THAT IT IS ACTIVE.
059577,000036:                                                                                                  #                       III. SOMETIMES FOUND POSMAX INDICATING THAT IT IS INACTIVE AND NOT ENABLED.
059578,000037:                                                                                                  #                        IV. SOMETIMES FOUND NEGATIVE ZERO INDICATING THAT:
059579,000038:                                                                                                  #                                A. A T6RUPT IS ABOUT TO OCCUR AT THE NEXT DINC, OR
059580,000039:                                                                                                  #                                B. A T6RUPT IS WAITING IN THE PRIORITY CHAIN, OR
059581,000040:                                                                                                  #                                C. A T6RUPT IS IN PROCESS NOW.
059582,000041:                                                                                                  #           4) ALL PROGRAMS WHICH OPERATE IN EITHER INTERRUPT MODE OR WITH INTERRUPT INHIBITED MUST CALL T6JOBCHK
059583,000042:                                                                                                  #              EVERY 5 MILLISECONDS TO PROCESS A POSSIBLE WAITING T6RUPT BEFORE IT CAN BE HONORED BY THE HARDWARE.
059584,000043: 
059585,000044:                                                                                                  #          (5. PROGRAM JTLST, IN Q,R-AXES, HANDLES THE INPUT LIST.)
059586,000045:                                                                                                  #  T6JOBCHK CALLING SEQUENCE:
059587,000046: 
059588,000047:                                                                                                  #                                          L        TC     T6JOBCHK
059589,000048:                                                                                                  #                                          L +1    (RETURN)
059590,000049: 
059591,000050:                                                                                                  #  DOT6RUPT CALLING SEQUENCE:
059592,000051: 
059593,000052:                                                                                                  #                                                   DXCH   ARUPT           T6RUPT LEAD IN AT LOCATION 4004.
059594,000053:                                                                                                  #                                                   EXTEND
059595,000054:                                                                                                  #                                                   DCA    T6ADR
059596,000055:                                                                                                  #                                                   DTCB
059597,000056: 
059598,000057:                                                                                                  #  SUBROUTINES CALLED: DOT6RUPT CALLS T6JOBCHK.
059599,000058: 
059600,000059:                                                                                                  #  NORMAL EXIT MODES: T6JOBCHK RETURNS TO L +1.
059601,000060:                                                                                                  #                     DOT6RUPT TRANSFERS CONTROL TO RESUME.
