Source Code

These source-code files are part of a reconstructed copy of Luminary 173, the second (unflown) release of the Apollo 14 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction began with reconstructed source code for Luminary 178. Changes between revision 173 and 178 were backed out, as described by Luminary memo 167 (revision 1). The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021152N. Note that page numbers in the reconstructed code match those for the baseline log section mentioned in each file's changelog; the page numbers for a real Luminary 173 listing would be different.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

039225,000021:                                                                                                  #  PROGRAM NAME:   FINDCDUW
039226,000022: 
039227,000023:                                                                                                  #  MOD NUMBER:     1         68 07 15
039228,000024: 
039229,000025:                                                                                                  #  MOD AUTHOR:     KLUMPP
039230,000026: 
039231,000027:                                                                                                  #  OBJECTS OF MOD: 1.        TO SUPPLY COMMANDED GIMBAL ANGLES FOR NOUN 22.
039232,000028:                                                                                                  #                  2.        TO MAINTAIN CORRECT AND CURRENT THRUST
039233,000029:                                                                                                  #                            DIRECTION DATA IN ALL MODES.  THIS IS DONE BY
039234,000030:                                                                                                  #                            FETCHING FOR THE THRUST DIRECTION FILTER THE
039235,000031:                                                                                                  #                            CDUD'S IN PNGCS-AUTO, THE CDU'S IN ALL OTHER
039236,000032:                                                                                                  #                            MODES.
039237,000033:                                                                                                  #                  3.        TO SUBSTITUTE A STOPRATE FOR THE NORMAL
039238,000034:                                                                                                  #                            AUTOPILOT COMMANDS WHENEVER
039239,000035:                                                                                                  #                            1) NOT IN PNGCS-AUTO, OR
039240,000036:                                                                                                  #                            2) ENGINE IS OFF.
039241,000037: 
039242,000038:                                                                                                  #  FUNCTIONAL DESCRIPTION:
039243,000039: 
039244,000040:                                                                                                  #  FINDCDUW PROVIDES THE INTERFACES BETWEEN THE VARIOUS POWERED FLITE GUIDANCE PROGRAMS
039245,000041:                                                                                                  #  AND THE DIGITAL AUTOPILOT.  THE INPUTS TO FINDCDUW ARE THE THRUST COMMAND VECTOR
039246,000042:                                                                                                  #  AND THE WINDOW COMMAND VECTOR, AND THE OUTPUTS ARE THE GIMBAL ANGLE
039247,000043:                                                                                                  #  INCREMENTS, THE COMMANDED ATTITUDE ANGLE RATES, AND THE COMMANDED
039248,000044:                                                                                                  #  ATTITUDE LAG ANGLES (WHICH ACCOUNT FOR THE ANGLES BY WHICH THE BODY WILL
039249,000045:                                                                                                  #  LAG BEHIND A RAMP COMMAND IN ATTITUDE ANGLE DUE TO THE FINITE ANGULAR
039250,000046:                                                                                                  #  ACCELERATIONS AVAILABLE).
039251,000047: 
039252,000048:                                                                                                  #  FINDCDUW ALINES THE ESTIMATED THRUST VECTOR FROM THE THRUST DIRECTION
039253,000049:                                                                                                  #  FILTER WITH THE THRUST COMMAND VECTOR, AND, WHEN  XOVINHIB  SET,
039254,000050:                                                                                                  #  ALINES THE +Z HALF OF THE LM ZX PLANE WITH THE WINDOW COMMAND VECTOR.

039257,000053:                                                                                                  #  SPECIFICATIONS:
039258,000054: 
039259,000055:                                                                                                  #  INITIALIZATION: A SINGLE INTERPRETIVE CALL TO  INITCDUW  IS REQUIRED
039260,000056:                                                                                                  #                  BEFORE EACH GUIDED MANEUVER USING FINDCDUW.
039261,000057: 
039262,000058:                                                                                                  #  CALL:           INTERPRETIVE CALL TO FINDCDUW WITH THE THRUST COMMAND
039263,000059:                                                                                                  #                  VECTOR IN MPAC.  INTERPRETIVE CALL TO FINDCDUW -2 WITH
039264,000060:                                                                                                  #                  THE THRUST COMMAND VECTOR IN UNFC/2 AND NOT IN MPAC.
039265,000061: 
039266,000062:                                                                                                  #  RETURNS:        NORMAL INTERPRETIVE IN ALL CASES
039267,000063:

039269,000065: 
039270,000066:                                                                                                  #                  2.       IF NOT PNGCS AUTO, DO STOPRATE AND RETURN
039271,000067:                                                                                                  #                           WITHOUT ISSUING AUTOPILOT CMDS.
039272,000068: 
039273,000069:                                                                                                  #                  3.       IF ENGINE OFF, DO STOPRATE AND RETURN WITHOUT
039274,000070:                                                                                                  #                           ISSUING AUTOPILOT CMDS.
039275,000071: 
039276,000072:                                                                                                  #  ALARMS:         00401 IF INPUTS DETERMINE AN ATTITUDE IN GIMBAL LOCK.
039277,000073:                                                                                                  #                           FINDCDUW DRIVES CDUXD AND CDUYD TO THE RQD VALUES,
039278,000074:                                                                                                  #                           BUT DRIVES CDUZD ONLY TO THE GIMBAL LOCK CONE.
039279,000075:                                                                                                  #                  00402 IF UNFC/2 OR UNWC/2 PRODUCE OVERFLOW WHEN
039280,000076:                                                                                                  #                           UNITIZED USING NORMUNIT.  FINDCDUW ISSUES
039281,000077:                                                                                                  #                           STOPRATE AS ONLY INPUT TO AUTOPILOT.
039282,000078: 
039283,000079:                                                                                                  #  INPUTS:         UNFC/2   THRUST COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
039284,000080:                                                                                                  #                  UNWC/2   WINDOW COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
039285,000081:                                                                                                  #                  OGABIAS  POSSIBLE BIAS FOR OUTER GIMBAL ANGLE (ZEROED IN INITCDUW), UNITS OF PI.
039286,000082:                                                                                                  #                  XOVINHIB FLAG DENOTING X AXIS OVERRIDE INHIBITED.
039287,000083:                                                                                                  #                  CSMDOCKD FLAG DENOTING CSM DOCKED.
039288,000084:                                                                                                  #                  STEERSW  FLAG DENOTING INSUFF THRUST FOR THRUST DIR FLTR.
039289,000085: 
039290,000086:                                                                                                  #  OUTPUTS:        DELCDUX,Y,Z
039291,000087:                                                                                                  #                  OMEGAPD,+1,+2
039292,000088:                                                                                                  #                  DELPEROR,+1,+2
039293,000089:                                                                                                  #                  CPHI,+1,+2 FOR NOUN22

039296,000092:                                                                                                  #                  WRITING INTO THESE LOCATIONS THE SINES AND COSINES
039297,000093:                                                                                                  #                  OF THE CDUD'S IN PNGCS-AUTO, OF THE CDU'S OTHERWISE.
