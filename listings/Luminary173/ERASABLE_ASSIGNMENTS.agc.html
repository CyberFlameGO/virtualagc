Source Code

These source-code files are part of a reconstructed copy of Luminary 173, the second (unflown) release of the Apollo 14 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction began with reconstructed source code for Luminary 178. Changes between revision 173 and 178 were backed out, as described by Luminary memo 167 (revision 1). The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021152N. Note that page numbers in the reconstructed code match those for the baseline log section mentioned in each file's changelog; the page numbers for a real Luminary 173 listing would be different.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

004113,000027:                                                                                                  #        EQUALS  IS USED IN TWO WAYS.  IT IS OFTEN USED TO CHAIN A GROUP
004114,000028:                                                                                                  #                OF ASSIGNMENTS SO THAT THE GROUP MAY BE MOVED WITH THE
004115,000029:                                                                                                  #                CHANGING OF ONLY ONE CARD.  EXAMPLE.
004116,000030: 
004117,000031:                                                                                                  #                                        X       EQUALS  START
004118,000032:                                                                                                  #                                        Y       EQUALS  X       +SIZE.X
004119,000033:                                                                                                  #                                        Z       EQUALS  Y       +SIZE.Y
004120,000034: 
004121,000035:                                                                                                  #                (X, Y, AND Z ARE CONSECUTIVE AND BEGIN AT START.        )
004122,000036:                                                                                                  #                (SIZE.X AND SIZE.Y ARE THE RESPECTIVE SIZES OF X AND Y,
004123,000037:                                                                                                  #                USUALLY NUMERIC, IE. 1, 2, 6, 18D ETC.                  )
004124,000038: 
004125,000039:                                                                                                  #        EQUALS  OFTEN IMPLIES THE SHARING OF REGISTERS (DIFFERENT NAMES
004126,000040:                                                                                                  #                AND DIFFERENT DATA).  EXAMPLE.
004127,000041: 
004128,000042:                                                                                                  #                        X       EQUALS  Y
004129,000043: 
004130,000044:                                                                                                  #        =       MEANS THAT MULTIPLE NAMES HAVE BEEN GIVEN TO THE SAME DATA.

004132,000046: 
004133,000047:                                                                                                  #                        X       =       Y

004135,000049:                                                                                                  #        THE SIZE AND UTILIZATION OF AN ERASABLE ARE OFTEN INCLUDED IN
004136,000050:                                                                                                  #        THE COMMENTS IN THE FOLLOWING FORM.  M(SIZE)N.
004137,000051: 
004138,000052:                                                                                                  #                M       REFERS TO THE MOBILITY OF THE ASSIGNMENT.
004139,000053:                                                                                                  #                        B       MEANS THAT THE SYMBOL IS REFERENCED BY BASIC
004140,000054:                                                                                                  #                                INSTRUCTIONS AND THUS IS E-BANK SENSITIVE.
004141,000055:                                                                                                  #                        I       MEANS THAT THE SYMBOL IS REFERENCED ONLY BY
004142,000056:                                                                                                  #                                INTERPRETIVE INSTRUCTIONS, AND IS THUS E-BANK
004143,000057:                                                                                                  #                                INSENSITIVE AND MAY APPEAR IN ANY E-BANK.
004144,000058: 
004145,000059:                                                                                                  #                SIZE    IS THE NUMBER OF REGISTERS INCLUDED BY THE SYMBOL.
004146,000060: 
004147,000061:                                                                                                  #                N       INDICATES THE NATURE OR PERMANENCE OF THE CONTENTS.
004148,000062:                                                                                                  #                        PL      MEANS THAT THE CONTENTS ARE PAD LOADED.
004149,000063:                                                                                                  #                        DSP     MEANS THAT THE REGISTER IS USED FOR A DISPLAY.
004150,000064:                                                                                                  #                        PRM     MEANS THAT THE REGISTER IS PERMANENT, IE. IT
004151,000065:                                                                                                  #                                IS USED DURING THE ENTIRE MISSION FOR ONE
004152,000066:                                                                                                  #                                PURPOSE AND CANNOT BE SHARED.
004153,000067:                                                                                                  #                        TMP     MEANS THAT THE REGISTER IS USED TEMPORARILY OR
004154,000068:                                                                                                  #                                IS A SCRATCH REGISTER FOR THE ROUTINE TO WHICH
004155,000069:                                                                                                  #                                IT IS ASSIGNED.  THAT IS, IT NEED NOT BE SET
004156,000070:                                                                                                  #                                PRIOR TO INVOCATION OF THE ROUTINE NOR DOES IT
004157,000071:                                                                                                  #                                CONTAIN USEFUL OUTPUT TO ANOTHER ROUTINE.  THUS
