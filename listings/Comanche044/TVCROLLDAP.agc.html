Source Code

These source-code files are part of a reconstructed copy of Comanche 044, the original release, though not the final mission release, of the Apollo 10 Command Module (CM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Comanche 051 previously reconstructed from a digitized copy of Comanche 055. The code was then updated by undoing changes made between revisions 044 and 051. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021153D. While there is thus a high degree of confidence in the reconstructed code, there are nevertheless a few changes that remain open to question, particularly in log section P20-P25, though even if these eventually prove incorrect they should not adversely affect code execution. Note also that page numbers in the reconstructed code match those on the Comanche 055 printout, although the added code would likely have changed page numbers for a real Comanche 044 listing.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

043024,000122:                                                                                                  #  GIVEN THE OPERATING POINT NOT IN THE COAST REGION, THE DESIRED OGARATE
043025,000123:                                                                                                  #        IS AT THE POINT OF PENETRATION OF THE COAST REGION BY THE CONTROL
043026,000124:                                                                                                  #        PARABOLA WHICH PASSES THROUGH THE OPERATING POINT.  FOR REGION 3
043027,000125:                                                                                                  #        DESIRED OGARATE IS SIMPLY +-MAXLIM.  FOR REGIONS 1 OR 6 THE SOLUTION
043028,000126:                                                                                                  #        TO A QUADRATIC IS REQUIRED (THE PENETRATION IS ALONG THE STRAIGHT
043029,000127:                                                                                                  #        LINE OR MINLIM BOUNDRY SWITCH LINES).  AN APPROXIMATION IS MADE
043030,000128:                                                                                                  #        INSTEAD. TAKE AN OPERATING POINT IN REGION 6'.  PASS A TANGENT TO
043031,000129:                                                                                                  #        THE CONTROL PARABOLA THROUGH THE OPERATING POINT, AND FIND ITS
043032,000130:                                                                                                  #        INTERSECTION WITH THE STRAIGHT LINE SECTION OF THE SWITCH CURVE...
043033,000131:                                                                                                  #        THE INTERSECTION DEFINES DESIRED OGARATE.  IF THE OPERATING POINT IS
043034,000132:                                                                                                  #        CLOSE TO THE SWITCH LINE, THE APPROXIMATION IS QUITE GOOD (INDEED
043035,000133:                                                                                                  #        THE APPROXIMATE AND QUADRATIC SOLUTIONS CONVERGE IN THE LIMIT AS
043036,000134:                                                                                                  #        THE SWITCH LINE IS APPROACHED).  IF THE OPERATING POINT IS NOT CLOSE
043037,000135:                                                                                                  #        TO THE SWITCH LINE, THE APPROXIMATE SOLUTION GIVES VALID TREND
043038,000136:                                                                                                  #        INFORMATION (DIRECTION OF DESIRED OGARATE) AT LEAST.  THE 
043039,000137:                                                                                                  #        RE-EVALUATION OF DESIRED OGARATE IN SUBSEQUENT ROLL DAP PASSES (1/2
043040,000138:                                                                                                  #        SECOND INTERVALS) WILL BENEFIT FROM THE CONVERGENT NATURE OF THE
043041,000139:                                                                                                  #        APPROXIMATION.
043042,000140: 
043043,000141:                                                                                                  #  FOR LARGE OGAERROR THE TANGENT INTERSECTS +-MINLIM SWITCH BOUNDRY BEFORE
043044,000142:                                                                                                  #        INTERSECTING THE STRAIGHT LINE SWITCH.  HOWEVER THE MINLIM IS
043045,000143:                                                                                                  #        IGNORED IN COMPUTING THE FIRING TIME, SO THAT THE EXTENSION (INTO
043046,000144:                                                                                                  #        THE COAST REGION) OF THE STRAIGHT LINE SWITCH IS WHAT IS FIRED TO.
043047,000145:                                                                                                  #        IF THE ROLL DAP FINDS ITSELF IN THE COAST REGION BEFORE REACHING
043048,000146:                                                                                                  #        THE DESIRED INTERSECTION (IE, IN THE REGION BETWEEN THE MINLIM
043049,000147:                                                                                                  #        AND THE STRAIGHT LINE SWITCH) IT WILL EXIBIT NORMAL COAST-REGION
043050,000148:                                                                                                  #        BEHAVIOR AND TURN OFF THE JETS.  THE PURPOSE OF THIS FIRING POLICY
043051,000149:                                                                                                  #        IS TO MAINTAIN STATIC ROLL STABILITY IN THE EVENT OF A JET
043052,000150:                                                                                                  #        FAILED-ON.
043053,000151: 
043054,000152:                                                                                                  #  WHEN THE OPERATING POINT IS IN REGION 1 THE SAME APPROXIMATION IS
043055,000153:                                                                                                  #        MADE, BUT AT AN ARTIFICIALLY-CREATED OR DUMMY OPERATING POINT,
043056,000154:                                                                                                  #        DEFINED BY.. OGAERROR = INTERSECTION OF CONTROL PARABOLA AND
043057,000155:                                                                                                  #        OGAERROR AXIS, OGARATE = +-LMCRATE WHERE SIGN IS OPPOSITE THAT OF
043058,000156:                                                                                                  #        REAL OPERATING POINT RATE.  WHEN THE OPERATING POINT HAS PASSED
043059,000157:                                                                                                  #        FROM REGION 1 TO REGION 6', THE DUMMY POINT IS NO LONGER REQUIRED,
043060,000158:                                                                                                  #        AND THE SOLUTION REVERTS TO THAT OF A REGULAR REGION 6' POINT.
