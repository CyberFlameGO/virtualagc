Source Code

These source-code files are part of a reconstructed copy of Comanche 044, the original release, though not the final mission release, of the Apollo 10 Command Module (CM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Comanche 051 previously reconstructed from a digitized copy of Comanche 055. The code was then updated by undoing changes made between revisions 044 and 051. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021153D. While there is thus a high degree of confidence in the reconstructed code, there are nevertheless a few changes that remain open to question, particularly in log section P20-P25, though even if these eventually prove incorrect they should not adversely affect code execution. Note also that page numbers in the reconstructed code match those on the Comanche 055 printout, although the added code would likely have changed page numbers for a real Comanche 044 listing.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

016774,000143:                                                                                                  #  PROGRAM DESCRIPTION - VECPOINT
016775,000144: 
016776,000145:                                                                                                  #  THIS INTERPRETIVE SUBROUTINE MAY BE USED TO POINT A SPACECRAFT AXIS IN A DESIRED DIRECTION.  THE AXIS
016777,000146:                                                                                                  #  TO BE POINTED MUST APPEAR AS A HALF UNIT DOUBLE PRECISION VECTOR IN SUCCESSIVE LOCATIONS OF ERASABLE MEMORY
016778,000147:                                                                                                  #  BEGINNING WITH THE LOCATION CALLED SCAXIS.  THE COMPONENTS OF THIS VECTOR ARE GIVEN IN SPACECRAFT COORDINATES.
016779,000148:                                                                                                  #  THE DIRECTION IN WHICH THIS AXIS IS TO BE POINTED MUST APPEAR AS A HALF UNIT DOUBLE PRECISION VECTOR IN
016780,000149:                                                                                                  #  SUCCESSIVE LOCATIONS OF ERASABLE MEMORY BEGINNING WITH THE ADDRESS CALLED POINTVSM.  THE COMPONENTS OF THIS
016781,000150:                                                                                                  #  VECTOR ARE GIVEN IN STABLE MEMBER COORDINATES.  WITH THIS INFORMATION VECPOINT COMPUTES A SET OF THREE GIMBAL
016782,000151:                                                                                                  #  ANGLES (2S COMPLEMENT) CORRESPONDING TO THE CROSS-PRODUCT ROTATION BETWEEN SCAXIS AND POINTVSM AND STORES THEM
016783,000152:                                                                                                  #  IN T(MPAC) BEFORE RETURNING TO THE CALLER.
016784,000153: 
016785,000154:                                                                                                  #  THIS ROTATION, HOWEVER, MAY BRING THE S/C INTO GIMBAL LOCK.  WHEN POINTING A VECTOR IN THE Y-Z PLANE,
016786,000155:                                                                                                  #  THE TRANSPONDER AXIS, OR THE AOT FOR THE LEM, THE PROGRAM WILL CORRECT THIS PROBLEM BY ROTATING THE CROSS-
016787,000156:                                                                                                  #  PRODUCT ATTITUDE ABOUT POINTVSM BY A FIXED AMOUNT SUFFICIENT TO ROTATE THE DESIRED S/C ATTITUDE OUT OF GIMBAL
016788,000157:                                                                                                  #  LOCK.  IF THE AXIS TO BE POINTED IS MORE THAN 40.6 DEGREES BUT LESS THAN 60.5 DEG FROM THE +X (OR -X) AXIS,
016789,000158:                                                                                                  #  THE ADDITIONAL ROTATION TO AVOID GIMAL LOCK IS 35 DEGREES.  IF THE AXIS IS MORE THAN 60.5 DEGEES FROM +X (OR -X)
016790,000159:                                                                                                  #  THE ADDITIONAL ROTATION IS 35 DEGREES.  THE GIMBAL ANGLES CORRESPONDING TO THIS ATTITUDE ARE THEN COMPUTED AND
016791,000160:                                                                                                  #  STORED AS 2S COMPLIMENT ANGLES IN T(MPAC) BEFORE RETURNING TO THE CALLER.
016792,000161: 
016793,000162:                                                                                                  #  WHEN POINTING THE X-AXIS, OR THE THRUST VECTOR, OR ANY VECTOR WITHIN 40.6 DEG OF THE X-AXIS, VECPOINT
016794,000163:                                                                                                  #  CANNOT CORRECT FOR A CROSS-PRODUCT ROTATION INTO GIMBAL LOCK.  IN THIS CASE A PLATFORM REALIGNMENT WOULD BE
016795,000164:                                                                                                  #  REQUIRED TO POINT THE VECTOR IN THE DESIRED DIRECTION.  AT PRESENT NO INDICATION IS GIVEN FOR THIS SITUATION
016796,000165:                                                                                                  #  EXCEPT THAT THE FINAL MIDDLE GIMBAL ANGLE IN MPAC +2 IS GREATER THAN 59 DEGREES.
016797,000166: 
016798,000167:                                                                                                  #  CALLING SEQUENCE -
016799,000168: 
016800,000169:                                                                                                  #        1)      LOAD SCAXIS, POINTVSM
016801,000170:                                                                                                  #        2)      CALL
016802,000171:                                                                                                  #                        VECPOINT
016803,000172: 
016804,000173:                                                                                                  #  RETURNS WITH
016805,000174: 
016806,000175:                                                                                                  #        1)      DESIRED OUTER GIMBAL ANGLE IN MPAC
016807,000176:                                                                                                  #        2)      DESIRED INNER GIMBAL ANGLE IN MPAC +1
016808,000177:                                                                                                  #        3)      DESIRED MIDDLE GIMBAL ANGLE IN MPAC +2
