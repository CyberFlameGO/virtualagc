Source Code

These source-code files are part of a reconstructed copy of Comanche 044, the original release, though not the final mission release, of the Apollo 10 Command Module (CM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Comanche 051 previously reconstructed from a digitized copy of Comanche 055. The code was then updated by undoing changes made between revisions 044 and 051. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021153D. While there is thus a high degree of confidence in the reconstructed code, there are nevertheless a few changes that remain open to question, particularly in log section P20-P25, though even if these eventually prove incorrect they should not adversely affect code execution. Note also that page numbers in the reconstructed code match those on the Comanche 055 printout, although the added code would likely have changed page numbers for a real Comanche 044 listing.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

042671,000019:                                                                                                  #  NAME          STROKE TEST PACKAGE             (INCLUDING INITIALIZATION PACKAGE)
042672,000020:                                                                                                  #  LOG SECTION...TVCSTROKETEST                   SUBROUTINE...DAPCSM
042673,000021:                                                                                                  #  MODIFIED BY SCHLUNDT                          21 OCTOBER 1968
042674,000022: 
042675,000023:                                                                                                  #  FUNCTIONAL DESCRIPTION....
042676,000024:                                                                                                  #        STROKE TEST PACKAGE GENERATES A WAVEFORM DESIGNED TO EXCITE BENDING
042677,000025:                                                                                                  #        STRKTSTI (STROKE TEST INITIALIZATION) IS CALLED AS A JOB BY VB68.
042678,000026:                                                                                                  #                IT INITIALIZES ALL ERASABLES REQD FOR A STROKE TEST, AND
042679,000027:                                                                                                  #                THEN TESTS FOR CSM/LM (BIT 13 OF DAPDATR1). IF CSM/LM,
042680,000028:                                                                                                  #                IN EITHER HIGH OR LOW-BANDWIDTH MODE, THE TEST IS STARTED
042681,000029:                                                                                                  #                IMMEDIATELY.  IF NOT CSM/LM, PROGRAM EXITS WITH NO ACTION.
042682,000030:                                                                                                  #        HACK (STROKE TEST) GENERATES THE WAVEFORM BY DUMPING PULSE BURSTS
042683,000031:                                                                                                  #                OF PROPER SIGN AND IN PROPER SEQUENCE DIRECTLY INTO
042684,000032:                                                                                                  #                TVCPITCH, WORKING IN CONJUNCITON WITH BOTH PITCH AND YAW
042685,000033:                                                                                                  #                TVC DAPS, WITH INTERMEDIATE WAITLIST CALLS.  NOTE, HOWEVER
042686,000034:                                                                                                  #                THAT THE STROKE TEST IS PERFORMED ONLY IN THE PITCH AXIS.
042687,000035:                                                                                                  #                AN EXAMPLE WAVEFORM IS GIVEN BELOW, TO DEMONSTRATE STROKE-
042688,000036:                                                                                                  #                TEST PARAMETER SELECTION
042689,000037:                                                                                                  #        RESTARTS CAUSE TEST TO BE TERMINATED.  ANOTHER V68 REQD IF TEST
042690,000038:                                                                                                  #                IS TO BE RE-RUN.
042691,000039:                                                                                                  #        PULSE BURST SIZE IS PAD-LOADED (ESTROKER) SO THAT AMPLITUDE OF
042692,000040:                                                                                                  #                WAVEFORM CAN BE CHANGED.  THERE ARE TEN PULSE BURSTS IN
042693,000041:                                                                                                  #                THE HALF-AMPLITUDE OF THE FIRST FREQUENCY SET IN THE
042694,000042:                                                                                                  #                STANDARD WAVEFORM.  AMPLITUDE IS 10(ESTROKER)(1/42.15),
042695,000043:                                                                                                  #                NOMINALLY 50/42.15 = 1.185 DEG
042696,000044: 
042697,000045:                                                                                                  #  CALLING SEQUENCE....
042698,000046:                                                                                                  #        EXTENDED VERB 68 SETS UP STRKTSTI JOB
042699,000047:                                                                                                  #        PITCH AND YAW TVCDAPS, FINDING STROKER NON-ZERO, DO A ..TC HACK..
042700,000048:                                                                                                  #        AN INTERNALLY-GENERATED WAITLIST CALL ENTERS AT ..HACKWLST..
042701,000049: 
042702,000050:                                                                                                  #  NORMAL EXIT MODES....
042703,000051:                                                                                                  #        TC BUNKER (..Q.. IF ENTRY FROM DAP, ..TCTSKOVR.. IF FROM WAITLIST) LIST
042704,000052: 
042705,000053:                                                                                                  #  SUBROUTINES CALLED....
042706,000054:                                                                                                  #        WAITLIST
042707,000055: 
042708,000056:                                                                                                  #  ALARM OR ABORT EXIT MODES....
042709,000057:                                                                                                  #        NONE
042710,000058: 
042711,000059:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED....
042712,000060:                                                                                                  #        ESTROKER (PAD-LOAD)
042713,000061:                                                                                                  #        STROKER, CADDY, REVS, CARD, N
042714,000062: 
042715,000063:                                                                                                  #  OUTPUT....
042716,000064:                                                                                                  #        STRKTSTI...INITIALIZATION FOR STROKE TEST
042717,000065:                                                                                                  #        HACK, HACKWLST...PULSE BURSTS INTO TVCPITCH VIA ..ADS..
042718,000066:                                                                                                  #                          RESETS STROKER = +0 WHEN TEST COMPLETED
042719,000067: 
042720,000068:                                                                                                  #  DEBRIS....
042721,000069:                                                                                                  #        N = CADDY = +0, CARD = -0, REVS = -1
042722,000070:                                                                                                  #        BUNKER
