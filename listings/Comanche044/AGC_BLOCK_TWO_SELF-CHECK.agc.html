Source Code

These source-code files are part of a reconstructed copy of Comanche 044, the original release, though not the final mission release, of the Apollo 10 Command Module (CM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Comanche 051 previously reconstructed from a digitized copy of Comanche 055. The code was then updated by undoing changes made between revisions 044 and 051. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021153D. While there is thus a high degree of confidence in the reconstructed code, there are nevertheless a few changes that remain open to question, particularly in log section P20-P25, though even if these eventually prove incorrect they should not adversely affect code execution. Note also that page numbers in the reconstructed code match those on the Comanche 055 printout, although the added code would likely have changed page numbers for a real Comanche 044 listing.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

059073,000024:                                                                                                  #  FUNCTIONAL DESCRIPTION
059074,000025:                                                                                                  #        PROGRAM HAS TWO MAIN PARTS.  THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
059075,000026:                                                                                                  #        PART OF THE BACK-UP IDLE LOOP.  THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
059076,000027:                                                                                                  #        STARTING VERB.
059077,000028: 
059078,000029:                                                                                                  #        THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
059079,000030: 
059080,000031:                                                                                                  #        THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK, ONE AT A TIME.
059081,000032: 
059082,000033:                                                                                                  #        IN ALL THERE ARE 7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK.  MORE DETAIL DESCRIPTION MAY BE
059083,000034:                                                                                                  #        FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
059084,000035: 
059085,000036:                                                                                                  #        THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27).  BELOW IS
059086,000037:                                                                                                  #        A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
059087,000038:                                                                                                  #        OCTAL, TO LOAD INTO SMODE.
059088,000039:                                                                                                  #                +-4             ERASABLE MEMORY
059089,000040:                                                                                                  #                +-5             FIXED MEMORY
059090,000041:                                                                                                  #                +-1,2,3,6,7,10  EVERYTHING IN OPTIONS 4 AND 5.
059091,000042:                                                                                                  #                -0              SAME AS +-10 UNTIL AN ERROR IS DETECTED.
059092,000043:                                                                                                  #                +0              NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
059093,000044: 
059094,000045:                                                                                                  #  WARNINGS
059095,000046:                                                                                                  #        USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
059096,000047:                                                                                                  #        SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
059097,000048: 
059098,000049:                                                                                                  #  CALLING SEQUENCE
059099,000050:                                                                                                  #        TO CALL SELF-CHECK KEY IN
059100,000051:                                                                                                  #                V 21 N 27 E     OPTION NUMBER E
059101,000052:                                                                                                  #        TO CALL SHOW-BANKSUM KEY IN
059102,000053:                                                                                                  #                V 91 E          DISPLAYS FIRST BANK
059103,000054:                                                                                                  #                V 33 E          PROCEED, DISPLAYS NEXT BANK
059104,000055: 
059105,000056:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
059106,000057:                                                                                                  #        SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED.  IF SO + OPTION NUMBERS PUT
059107,000058:                                                                                                  #        COMPUTER INTO BACKUP IDLE LOOP, - OPTION NUMBERS RESTART THE OPTION.
059108,000059: 
059109,000060:                                                                                                  #        THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
059110,000061: 
059111,000062:                                                                                                  #        SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E).  THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP
059112,000063:

059115,000066:                                                                                                  #        SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
059116,000067:                                                                                                  #        TURNS ON THE ALARM LIGHT.  THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E.  FOR FURTHER
059117,000068:                                                                                                  #        INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
059118,000069:                                                                                                  #        IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
059119,000070:                                                                                                  #        INITIATED FRESH START (SLAP1).
059120,000071: 
059121,000072:                                                                                                  #        SHOW-BANKSUM STARTING WITH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
059122,000073:                                                                                                  #        THE BANK NUMBER, AND IN R3 THE BUGGER WORD.

059124,000075:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
